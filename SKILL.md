# 故事工坊2.0 - 网文创作全能助手

> **从灵感到爆更，一站式网文创作解决方案**

**核心能力**：灵感激活 → 设定构建 → 大纲规划 → 正文创作 → 情节优化 → 卡文急救 → 节奏诊断

## 任务目标

- **本Skill用于**：网络小说创作全流程辅助，从书名生成、文案撰写、人设设计，到大纲规划、开篇创作、章节写作、节奏优化、百万字长篇规划
- **能力包含**：书名生成、平台文案适配、人设卡构建、大纲生成、黄金开篇、爽点生成、单章写作、对话场景、冲突制造、反转生成、节奏诊断优化、百万字长篇规划、卡文急救
- **触发条件**：用户说"写小说"、"卡文"、"书名"、"大纲"、"人设"、"爽文"、"逆袭"等表达创作意图时

## 创作模式选择

本Skill提供两种创作模式，请根据您的需求选择：

### 🚀 模式A：AI全自动模式（推荐新手）
**适用场景**：没有具体创作想法，想要快速开始；第一次使用，希望快速上手

**使用流程**：
1. 输入一句话需求（如"写一个修仙爽文"或"都市逆袭爽文"）
2. AI自动生成完整的五阶段设定框架
3. 用户确认或微调后，直接进入内容生成

### 📝 模式B：渐进询问模式（适合有想法）
**适用场景**：有具体的创作想法；想要精细控制每个设定

**使用流程**：
1. 逐一回答12个问题（分五阶段）
2. 每个问题后可选择「由AI自动生成剩余所有设定」
3. 随时可以切换到全自动模式

## 核心原理速览

### 黄金五章法则
```
第1章：穿越/重生 → 世界观 → 身份处境（第3章前必须出现威胁或机会）
第2章：金手指暗示 → 冲突引入 → 读者期待建立
第3章：首战或挫折 → 能力展示 → 爽点初现
第4章：势力扩张 → 矛盾升级 → 期待强化
第5章：小高潮 → 收尾 → 新悬念
```

### 三翻四震结构
```
第一翻：反派嚣张 → 读者压抑
第一震：主角处境更惨 → 情绪加剧
第二翻：主角隐忍 → 读者期待
第二震：反派得寸进尺 → 愤怒积压
第三翻：主角反击 → 情绪引爆
第三震+第四震：连番打脸 → 情绪宣泄 → 满足达成
```

### 爽文核心公式
```
爽 = 压抑的积累 + 爆发的释放 + 预期的达成 + 超越预期的惊喜
```

## 使用示例

### 示例1：生成爆款书名
**用户需求**：都市重生题材，情绪是爽文逆袭

**执行流程**：
1. 读取 `references/guides/title-guide.md`
2. 应用至少10种命名技巧
3. 生成30个不同风格的书名

**部分输出**：
- 《重生1990：从摆摊首富开始》（数字式+时代标签）
- 《被豪门赶出家门，我转身身家百亿》（反差式+爽点）
- 《重生那年，我先拿下了千亿女友》（悬念式+冲突）

### 示例2：卡文急救
**场景**：用户写到第50章，主角刚打败反派，不知道下一步剧情

**诊断**：剧情断崖型卡文

**急救方案**：
1. 新势力介入（建议3个具体方案）
2. 内部危机（主角阵营出现叛徒/资源危机）
3. 意外发现（发现新线索/新地图/新秘密）

### 示例3：爽点设计
**场景**：设计一个"扮猪吃虎"的高光时刻

**执行流程**：
1. 铺垫：反派嚣张、轻视主角
2. 压制：主角故意示弱
3. 爆发：关键时刻亮出底牌
4. 打脸：反派震惊、众人改观
5. 余味：情绪延续、期待下一轮

## 注意事项

1. **智能体主导**：所有创作内容由智能体基于用户需求和参考文档生成
2. **质量优先**：生成的内容应具备实际可用性，而非简单堆砌
3. **平台适配**：文案生成时必须严格按照目标平台的用户画像和风格规范调整
4. **创作自由度**：在参考文档基础上，鼓励创新和差异化
5. **迭代优化**：用户反馈后可快速调整，提供多个版本供选择
6. **写作质量**：避免模板化词汇，用动作/场景代替解释性叙述
