## 核心写作理念

> **七字真言：简单、粗暴、爽！**

网文的本质是情绪消费。读者要的只有三件事：**爽、爽、还是爽！**

### 黄金五章结构

```
第一章：重生/穿越 + 身份设定 + 金手指觉醒
第二章：展现实力 + 第一次小爽点
第三章：第一次打脸 + 建立主线目标
第四章：持续成长 + 埋下伏笔
第五章：大爽点爆发 + 建立长期期待
```

### 三翻四震写法

- **三翻**：反复压制（压→更压→极压）
- **四震**：四次情绪爆发点
  - 震惊：主角展现出意想不到的能力
  - 震动：配角群体的反应
  - 震撼：更大的场面或更强的敌人
  - 震怒：反派被彻底碾压

### 套路循环公式

```
压抑 → 期待 → 释放 → 满足 → 新压抑 → 新期待...
```

**输出格式**：

### 2.2 金手指设计

**金手指类型库**（详见 `references/gold-finger-types.md`）：

| 类型 | 核心优势 | 限制设计 |
|-----|---------|---------|
| 系统流 | 任务奖励、属性面板 | 任务惩罚、冷却时间 |
| 重生流 | 先知优势、信息差 | 重生代价、前世遗憾 |
| 祖传宝物 | 传承老爷爷、空间存储 | 升级条件、使用限制 |
| 血脉天赋 | 天生强大、特殊体质 | 觉醒条件、血脉代价 |
| 技能异能 | 独特能力、直接碾压 | 消耗代价、使用次数 |

**金手指设计要点**：
1. 升级路径清晰（可预期）
2. 使用有限制（制造紧张感）
3. 有副作用或代价（增加戏剧性）

### 2.3 世界观设定

**核心要素**：
- 阶层体系（明确爽点跃升空间）
- 势力分布（图谱化）
- 规则体系（规则冲突=爽点来源）
- 关键场景（3-5个核心场景）

**世界观搭建公式**：
```
底层（困境）→ 中层（挑战）→ 高层（巅峰）
     ↑              ↑              ↓
     ←────── 跃升路径 ──────→
```

---

### 2.4 冲突与反派设计

**冲突升级四阶梯**（详见 `references/conflict-design.md`）：

| 等级 | 形式 | 情绪 | 结果 |
|-----|------|------|------|
| 一级 | 言语冲突、小摩擦 | 轻微不快 | 主角吃亏或小胜 |
| 二级 | 正面交锋、实力压制 | 愤怒、压抑 | 主角暂时失利但成长 |
| 三级 | 公开对决、生死较量 | 仇恨、复仇心 | 主角展现实力，局部胜利 |
| 四级 | 最后决战、巅峰对决 | 极致紧张→释放 | 主角彻底胜利 |

**反派设计三要素**：
1. **合理性**：动机必须可信
2. **威胁性**：对主角构成真正威胁
3. **可打脸性**：最终能被主角碾压

## 模块三：大纲规划

### 3.1 市场定位型大纲

**适用**：追求商业成绩、读者喜欢套路明确的

**结构**：
```
开篇钩子 → 困境设定 → 金手指觉醒 → 第一次打脸 →
成长积累 → 遇到更强敌人 → 中期高潮 → 最终决战 → 圆满结局
```

**爽点密度**：每5-8章一个中爽点，每20-30章一个大爽点

---

### 3.2 人物经历型大纲

**适用**：注重人物弧光、有深度的故事

**结构**：
```
人物现状 → 内心渴望 → 外部阻碍 → 挣扎与选择 →
转变与成长 → 新的平衡 → 升华或回归
```

**核心问题**：
- 主角内心真正想要的是什么？
- 什么阻止他得到它？
- 他愿意付出什么代价？

---

### 3.3 分卷大纲生成

**输入**：题材 + 字数目标 + 分卷数量

**分卷结构模板**：
```
【第X卷：卷名】（第X-Y章，约X万字）
【核心冲突】本卷主要矛盾
【主要爽点】开局爽点/中段爽点/结尾爽点
【关键事件】3-5个关键情节点
【人物出场】主角/配角/反派
【本卷目标】主角达成的成就
```

---

### 3.4 百万字长篇规划

**四阶段结构**：
1. **开篇阶段**（第1-30万字）：建立世界观 + 核心冲突 + 第一个大爽点
2. **发展阶段**（第30-100万字）：势力扩张 + 实力提升 + 多个中爽点
3. **高潮阶段**（第100-200万字）：终极敌人 + 最终决战 + 核心爽点爆发
4. **结局阶段**（第200万字+）：收束所有伏笔 + 圆满结局

**进度管理建议**：
- 每卷设一个核心目标
- 每10万字回顾调整
- 保持人物成长线清晰
- 定期检查世界观一致性

## 模块四：正文创作

### 4.1 黄金开篇

**开篇三要素**：
1. 交代穿越/重生（第一句必须说明）
2. 交代世界观（简洁）
3. 交代身份处境

**开篇七技巧**：
- 冲突前置式
- 悬念式
- 反差式
- 金手指式
- 时间点式
- 对话式
- 动作式

**黄金三章检查清单**：
- [ ] 第一章有核心冲突？
- [ ] 前三章有爽点释放？
- [ ] 建立核心期待？

---

### 4.2 单章写作

**四段式结构**：
```
引入(10-20%) → 发展(40-50%) → 高潮(30-40%) → 结尾(10%)
```

**章节类型**：
- **战斗章**：对峙 → 试探 → 决战 → 结果
- **情感章**：铺垫 → 变化 → 爆发 → 收束
- **日常章**：场景 → 互动 → 小冲突 → 期待

---

### 4.3 对话场景

**对话写作原则**：
- 口语化，带停顿
- 每句对话独立成段
- 用动作/反应代替"说"
- 潜台词藏在话里

**错误示例**：
```
"我不会放过你的。"林风愤怒地说。
```

**正确示例**：
```
林风看着赵峰，没有说话。
"我不会，放过你的。"
```

---

### 4.4 打斗场景

**五层结构**：
1. **对峙**：双方出场，气势交锋
2. **试探**：互相试探，了解对手
3. **僵持**：实力碰撞，形势紧张
4. **突破**：主角使用金手指/底牌
5. **决胜**：一击定胜负，反派崩溃

**打斗描写要点**：
- 过程详细（不要一招秒杀）
- 层次分明（层层递进）
- 金手指运用（能力独特性）
- 心理博弈（不只是实力对抗）

---

### 4.5 爽点设计

**爽点类型库**（详见 `references/cool-point-guide.md`）：

| 类型 | 子类型 | 示例 |
|-----|-------|------|
| 身份爽 | 身份揭秘/逆袭/对比 | 废柴→天才 |
| 实力爽 | 碾压/突破/反差 | 一招秒杀 |

## 模块五：情节优化

### 5.1 冲突制造

**冲突类型**：
- 人际冲突（主角 vs 反派/配角）
- 内心冲突（主角 vs 自己）
- 环境冲突（主角 vs 规则/命运）
- 目标冲突（利益/理念/情感）

**冲突设计检查**：
- [ ] 冲突是否足够激烈？
- [ ] 冲突是否合理可信？
- [ ] 冲突是否推动剧情？
- [ ] 冲突是否有层次递进？

---

### 5.2 反转设计

**反转类型**：
- 预期违背（以为是A，实际是B）
- 因果倒置（以为是原因，实际是结果）
- 细节伏笔（前面埋的线索，后面揭示真相）

**反转设计要点**：
1. 铺垫要自然（不要突然反转）
2. 揭示要震撼（反转要让人惊呼）
3. 逻辑要自洽（反转要能说服读者）

---

### 5.3 节奏诊断

**节奏问题识别**：
- 节奏过快：情节跳跃，读者跟不上
- 节奏过慢：大段铺垫，读者流失
- 节奏混乱：爽点分布不均

**节奏优化策略**：
- 快节奏：冲突密集、事件连发
- 慢节奏：细节描写、情感铺垫
- 变化节奏：张弛有度，有起有伏

### 5.4 情感渲染引擎

#### 5.4.1 情感坐标系模型

为每个角色定义三维情感状态，实现情感精准把控：

| 维度 | 说明 | 取值范围 |
|------|------|----------|
| **信念强度** | 对某价值观的坚持程度 | 0~1 |
| **情绪极性** | 当前主导情绪倾向 | -1（负）~ +1（正） |
| **行动意愿** | 采取主动行为的可能性 | 0~1 |

**情感状态标注模板**：
```

- 信念强度：X.X → [变化原因]
- 情绪极性：X.X → [具体情绪，如：激动、恐惧、平静]
- 行动意愿：X.X → [行动倾向]
```

**三维影响**：
- 信念强度 → 影响对话中的措辞坚定性
- 情绪极性 → 决定比喻与节奏（高正值多用明亮意象）

| **呼吸** | 紧张/放松/激动 | 急促、屏息、深长、起伏 |
| **心跳** | 心动/恐惧/紧张 | 漏拍、加速、剧烈、漏了一拍 |

**复合生理描写示例**：
```
她张了张嘴，喉咙却像被什么堵住了。
手不自觉地攥紧了裙角，指节泛白。
眼眶发热，视线开始模糊。
她拼命眨眼，把那股潮意压了回去。
深吸一口气，声音却还是抖的：「……我没事。」
```

#### 5.4.6 动机图谱与关系网络

3. **内在冲突**：两类动机之间的张力

**角色动机模板**：

```
- 内在冲突：[如：正义理想 vs 复仇执念]
```

#### 男频爽文写作风格指南

**核心原则**：简单、粗暴、爽！

**节奏控制**：
- 句子要短，一句话不超过15个字
- 段落要短，每段不超过3行
- 章节要快，一章一个爽点或冲突
- 升级要清晰，属性面板/等级跃升可视化

**情绪调动**：
```

```

**禁止写法**：
| ❌ 禁止 | ✅ 替代 |
|--------|--------|
| 内心独白过长 | 用动作和对话展示 |
| 环境描写过多 | 一句话带过 |
| 解释性文字 | 用剧情展示 |
| 温柔/委婉表达 | 直接/粗暴/有力 |

**黄金五章公式**：

```
第二章：展现实力 + 第一次小爽点
第三章：第一次打脸 + 建立主线目标

```

```
【世界观】宏大背景设定
【主角】独特设定+成长路径

```

**言情/情感类公式**：
```
【人物】身份+性格+命运
【情感】相遇→相知→相守
【核心】甜/虐/治愈

示例：
《重生后，渣总追妻火葬场》

```

## 高级写作技巧

### 避免模板化词汇

| 禁止词 | 替代方案 |
|-------|---------|
| 充满了、坚定、不甘 | 用动作代替 |
| 冷冷地说、愤怒地说 | 用场景/反应代替 |
| 紧紧地、狠狠地、猛地 | 用具体动作代替 |

**示例对比**：
```
❌ 错误：林风眼中充满了愤怒，紧紧握住拳头，不甘地看着赵峰。

✅ 正确：林风盯着赵峰，拳头慢慢收紧，指节发白。他呼吸变得急促，但没有说话。
```

### Show, Don't Tell

- 用动作展示情绪，而不是直接告诉读者
- 用场景营造氛围，而不是解释说明
- 用细节暗示状态，而不是直接描述

### 平静的狠劲

```
❌ 错误：林风愤怒地说："我要杀了你！"他狠狠地握紧拳头。

✅ 正确：林风看着赵峰，没有说话。他慢慢走到赵峰面前，蹲下。"赵峰。""你...你不敢...""死吧。"
```

### 各平台发表指南

#### 都市/现实类平台

**特点**：
- 用户基数大，流量可观
- 推荐算法决定流量
- 前8万字是生死线

**投稿要求**：
- 正文2万字以上可申请签约
- 书名+简介决定点击率
- 标签选择影响推荐

**提升技巧**：
- 前3章必须冲突前置
- 每天更新4000字以上
- 利用平台热点活动
- 引导读者追更

#### 玄幻/奇幻类平台

**特点**：
- 付费阅读，精品化
- 订阅制，忠实读者
- 推荐机制复杂

**投稿要求**：
- 签约门槛较高
- 创新设定加分
- 文笔要求相对较高

**提升技巧**：
- 世界观要有深度
- 人物成长要清晰
- 更新稳定是关键
- 参与平台活动

#### 言情/情感类平台

**特点**：
- 言情/甜宠为主
- 感情线是核心
- 版权价值高

**投稿要求**：
- 签约门槛适中
- 感情描写要细腻
- 人设要讨喜

**提升技巧**：
- 感情递进要自然
- 配角要出彩
- 文笔要有质感
- 避免敏感题材

#### 脑洞/快节奏类平台

**特点**：
- 同人/穿越/系统流
- 更新速度要求极高
- 天榜曝光量大

**投稿要求**：
- 日更10000字起步
- 脑洞要足够大
- 追热点能力强

**提升技巧**：
- 前10章定生死
- 节奏要极快
- 爽点要密集
- 更新绝不能断

### 常见问题解答

#### Q1：新手应该选哪个平台？
**A**：
- 想快速验证题材 → 都市/现实类平台
- 想写精品长篇 → 玄幻/奇幻类平台
- 想写感情线 → 言情/情感类平台
- 同人脑洞文 → 脑洞/快节奏类平台

#### Q2：日更4000字很难坚持怎么办？
**A**：
- 先存稿再发书（建议存30章）
- 使用大纲减少卡文
- 固定写作时间（如早起1小时）
- 利用碎片时间构思

#### Q3：签约被拒怎么办？
**A**：
- 检查前三章是否有冲突
- 优化书名和简介
- 研究平台风格调整方向
- 先完本再开新书

#### Q4：数据一直不好要切书吗？
**A**：
- 番茄：20万字看追读，再决定
- 起点：50万字看订阅
- 先写到第一个大高潮再判断
- 切书前确保新书更有把握

#### Q5：如何提升写作水平？
**A**：
- 拆解爆款章节结构
- 学习写作技巧（节奏/爽点/情绪）
- 多写多练，量变到质变
- 接受读者反馈

#### Q6：遇到卡文怎么办？
**A**：
- 回顾大纲和主线目标
- 添加突发事件/冲突
- 让配角制造麻烦
- 先写后面情节再回填

#### Q7：读者差评太多怎么办？
**A**：
- 分辨是客观建议还是主观偏好
- 毒点要记录并避免
- 合理建议要采纳
- 差评多说明有人看，继续写

### 写作技巧进阶

#### 从新手到进阶

**新手阶段（0-50万字）**：
- 学习基础套路
- 保持稳定更新
- 培养写作习惯
- 熟悉平台规则

**进阶阶段（50-200万字）**：
- 优化爽点设计
- 提升人物塑造
- 丰富世界观
- 学会埋伏笔

**高手阶段（200万字+）**：
- 形成个人风格
- 创新套路玩法
- 追求精品质量
- 考虑版权开发

#### 高级写作技巧

**情绪操控**：
```
读者情绪 = 铺垫深度 × 爆发强度
铺垫越深，释放越爽
```

**信息释放节奏**：
- 核心设定：开篇透露
- 关键信息：剧情需要时揭示
- 隐藏线索：中期抛出
- 终极真相：大高潮前揭晓

**角色塑造要点**：
- 主角要有明确目标
- 反派要有合理动机
- 配角要有独特记忆点
- 每个角色推动剧情

#### 推荐学习路径

1. **入门**：《网文写作指南》《爆款小说创作法则》
2. **进阶**：《故事》《剧本》
3. **提升**：拆解100本同题材爆款
4. **突破**：形成自己的写作风格

> **借助AI能力，强化复杂场景处理和逻辑校验**

### 8.1 逻辑推理校验

**适用场景**：
- 复杂杀人诡计/魔法体系的逻辑闭环
- 多线程设定、时间线校验
- 伏笔回收断裂检测
- 人物动机矛盾识别

**使用指令模板**：
```
请按步骤分析以下大纲：
1. 指出第X章中主角的关键决定
2. 与第Y章埋设的证物线索是否存在时间冲突
3. 标注所有伏笔回收断裂处
4. 验证因果闭环完整性

[粘贴大纲内容]
```

### 8.2 复杂世界观构建

**适用题材**：
- 科幻星际、奇幻魔法
- 多势力对抗、力量体系
- 悬疑推理、烧脑剧情

**使用指令模板**：
```
帮我设计一个[题材]小说的世界观设定：
1. 力量体系/规则（需要严密自洽）
2. 势力分布与利益关系
3. 历史背景与冲突根源
4. 逻辑漏洞自查清单

要求：逻辑严密，无明显漏洞
```

### 8.3 最佳组合工作流

**AI辅助 + 小说工坊 组合拳**：

| 阶段 | 工具 | 任务 |
|-----|------|------|
| 1 | AI辅助 | 定骨架：生成逻辑严密的世界观和章节细纲 |
| 2 | 小说工坊 | 填血肉：利用网文节奏模板扩写成正文 |
| 3 | AI辅助 | 做校验：检查逻辑漏洞、伏笔呼应 |
| 4 | 小说工坊 | 精修：情绪渲染、高潮设计 |

**具体操作**：
```
第一步（AI辅助）：
帮我设计一个都市重生小说的10章大纲，主角是普通上班族，
重生回十年前，利用前世记忆逆袭人生。要求每章有明确冲突。

第二步（小说工坊）：
基于以上大纲，用网文节奏扩写成第一章，3000字。
要求：开头有冲突、节奏快、对话自然、结尾有悬念。

第三步（AI辅助）：
请检查第X章的伏笔是否回收、逻辑是否自洽。
```

### 8.4 AI辅助提示词公式

**公式**：场景/情绪 + 核心冲突 + 感官细节 + 限制词

**示例**：
```
❌ 错误：写一场精彩的打斗

✅ 正确：请以第三人称视角，描写主角在雨夜被仇家追杀的场景。
核心冲突是主角断了一只手，只能绝地反击。
加入雨水混合血水的粘稠触感、雷声的听感。
禁止使用"首先、其次"、"旋即"、"不由得"等AI常用词。
```
