# 小说工坊 (Novel Workshop)

> 全能网文写作助手，从卡文到爆更的全程支持

## 功能特点

- 🔥 **卡文急救包** - 6种卡文类型秒出方案
- 🆕 **新手引导** - 30分钟搞定第一章
- 📖 **全流程覆盖** - 灵感→大纲→人设→正文→优化
- 🎯 **分层入口** - 说人话就能用，不用选模块

## 核心模块

| 模块 | 功能 |
|------|------|
| 灵感创意 | 书名生成、简介、热门题材分析 |
| 设定构建 | 人设卡、金手指、世界观、冲突设计 |
| 大纲规划 | 市场型/人物型/分卷大纲 |
| 正文创作 | 开篇、章节、对话、打斗、爽点 |
| 情节优化 | 冲突、反转、节奏、情绪调动 |
| 情感渲染 | 五感锚点、留白艺术、身体记忆库 |
| 卡文急救 | 六种卡文类型极速救援 |
| AI辅助 | 逻辑校验、世界观构建 |

## 适用题材

- 重生都市、信息差碾压
- 穿越古代、知识降维
- 玄幻修仙、升级打怪
- 都市异能、扮猪吃虎
- 甜宠言情、撒糖甜宠
- 脑洞创意、反转设定

## 使用方式

将 `SKILL.md` 放入你的 AI 助手技能目录即可使用。
