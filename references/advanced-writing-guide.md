## 一、世界观设定（教程12）

### 核心原则
世界观是故事的土壤，决定故事能长多高、开什么花。

### 设定步骤
1. **基础架构**：世界规则、物理法则、社会结构
2. **等级体系**：修炼/职业/地位等级（核心爽点来源）
3. **势力分布**：门派/家族/国家等势力格局
4. **历史背景**：历史事件如何影响当前格局
5. **资源体系**：灵石/货币/功法等稀缺资源

### 新人避坑
- ❌ 一上来就写完整的世界设定
- ✅ 从主角视角出发，需要什么写什么
- ❌ 设定与剧情两张皮
- ✅ 设定为剧情服务，剧情检验设定

## 二、金手指设计（教程14）

### 金手指定义
主角独有的、超越常规的特殊能力/资源/机遇。

### 设计原则
1. **独特性**：别人没有，或者别人有但主角更强
2. **可成长性**：能随着剧情升级，保持竞争力
3. **可解释性**：有合理的来源或原因
4. **限制性**：必须有代价或限制，避免无敌

### 常见金手指类型
| 类型 | 示例 | 设计要点 |
|------|------|----------|
| 系统 | 任务系统、签到系统 | 奖励要合理，避免无脑刷 |
| 老爷爷 | 戒指老爷爷、传承记忆 | 老爷爷要有性格，限制解锁 |
| 异能 | 透视、读心、时间暂停 | 设置使用限制和副作用 |
| 知识 | 未来信息、前世记忆 | 信息要有限，依赖主角判断 |
| 空间 | 随身空间、农场空间 | 产出要控制，不能破坏经济 |

### 崩文雷区
- ❌ 一开始就全能
- ✅ 从弱到强，逐步解锁
- ❌ 金手指解决所有问题
- ✅ 金手指+主角智慧+努力

## 三、打斗场景（教程15）

### 核心要素
**画面感 + 节奏感 + 情绪张力**

### 写法要点

#### 1. 动作分解
```
错误示范：主角一剑刺出，对手死了
正确示范：主角身形一晃，长剑化作一道银虹，直刺对手咽喉。对手侧身闪避，反手一掌拍来。掌风呼啸，带着刺骨的寒意。主角借力后跃，脚尖点地，再次扑上。
```

#### 2. 五感描写
- **视觉**：招式、光影、色彩
- **听觉**：兵器交击声、风声、惨叫声
- **触觉**：痛感、气流冲击、力量碰撞
- **嗅觉**：血腥味、烧焦味、药草香

#### 3. 战斗节奏
```
铺垫（紧张）→ 交手（激烈）→ 危机（绝望）→ 转折（金手指）→ 反杀（爽）
```

### 禁忌
- ❌ 纯技能名对轰："我用降龙十八掌！""我挡！"
- ❌ 旁白式解说："此时主角内心OS：必须赢"
- ❌ 打斗与情节脱节

## 四、节奏控制（教程16）

### 节奏过快症状
- 读者还没代入，主角已经无敌了
- 冲突解决太快，缺乏张力
- 章节之间缺乏钩子

### 节奏过慢症状
- 大段环境描写、心理描写
- 无意义的对话和水字数
- 情节推进缓慢，看着想睡觉

### 调整方法

#### 加速技巧
- 删除冗余描写
- 合并类似场景
- 用叙述代替描写："三个月后..."
- 减少内心独白

#### 减速技巧
- 关键场景详细展开
- 重要对话逐字写
- 情绪高潮处留白
- 紧张时刻拉长

### 黄金节奏
```
小高潮（每章）→ 中高潮（每10章）→ 大高潮（每卷/转折点）
```

## 五、伏笔设置（教程17）

### 伏笔类型

#### 1. 人物伏笔
```
早期：主角随手帮助的乞丐
后期：乞丐是隐世高手，关键时刻救主角一命
```

#### 2. 物品伏笔
```
早期：主角捡到一枚玉佩，觉得好看留着
后期：玉佩是打开某个宝藏的钥匙
```

#### 3. 言语伏笔
```
早期："三年后，你会后悔今天的选择"
后期：某人因为当初的选择付出代价
```

#### 4. 性格伏笔
```
早期：主角每次遇到危险都会先观察再行动
后期：正是这个习惯让他发现敌人的破绽
```

### 技巧
- **埋得浅**：让读者能看到一点端倪
- **收得妙**：高潮时揭开奖赏感最强
- **密度适中**：太多伏笔会显得杂乱
- **记录清晰**：作者自己要有伏笔清单

## 六、铺垫艺术（教程18）

### 铺垫与伏笔的区别
| 对比项 | 伏笔 | 铺垫 |
|--------|------|------|
| 目的 | 揭晓时制造惊喜 | 提前建立期待 |
| 呈现 | 隐蔽，不易察觉 | 明显，读者能感受到 |
| 时机 | 揭晓越晚越好 | 揭晓前要有足够暗示 |

### 铺垫类型

#### 1. 势力铺垫
```
早期：提到某个神秘势力的传说
中段：这个势力开始行动
高潮：该势力正式登场
```

#### 2. 能力铺垫
```
早期：主角修炼某种功法，进度很慢
中期：遇到瓶颈，似乎走错了路
高潮：正是这功法让他突破瓶颈
```

#### 3. 情感铺垫
```
早期：男女主互相看不顺眼
中段：患难与共，暗生情愫
高潮：表白，水到渠成
```

### 节奏把控
```
第一次提及 → 强化印象 → 再次出现 → 深入关联 → 爆发时刻
```

## 七、外貌描写（教程19）

### 核心原则
**特征 > 美丑，动态 > 静态，人设 > 描写**

### 写法技巧

#### 1. 选择性描写
```
不要：她有一头乌黑亮丽的长发，大大的眼睛，高挺的鼻梁，樱桃小嘴
要：她一头干练短发，眉眼间带着凌厉，气场强大
```

#### 2. 用特征代替颜值
- "那双眼睛像是会说话" → "眼珠一转，一个主意就冒出来"
- "长得好看" → "走到哪里都有人搭讪"

#### 3. 结合性格
```
外表冷酷的人：语气淡漠，目光冷冽
外表热情的人：笑容满面，眼睛弯弯
反派：笑容不达眼底，目光闪烁
```

#### 4. 动态描写
```
静态：她有一双漂亮的眼睛
动态：她眨眨眼，眼底闪过一丝狡黠
```

## 八、对话描写（教程20）

### 常见问题
- ❌ "你好吗？" "我很好。" （太书面）
- ❌ 大段对话没有动作/神态
- ❌ 对话全是信息，没有性格
- ❌ 每个人说话方式一样

### 提升技巧

#### 1. 生活化
```
错误：公子此举甚是不妥
正确：喂，你脑子进水了？
```

#### 2. 性格化
- 商人：绕弯子，试探
- 武将：直接，粗犷
- 文人：咬文嚼字，引经据典
- 流氓：威胁，带脏

#### 3. 多元素混合
```python
"滚。" 他头也不抬，笔尖在纸上顿了顿。

"可是——"

"再不滚，明天你的店就别开了。" 语气平静，像在说今天天气不错。
```

#### 4. 潜台词
```
表面：这事不急，慢慢来
潜台词：我不支持，但你非要干我也没办法
```

## 九、心理描写（教程21）

### 四种方法

#### 1. 直接描写
```
主角心里一紧：难道被他发现了？
```
适用：情绪转折、内心独白精华

#### 2. 动作暗示
```
他握着茶杯的手微微收紧，指节泛白
```
适用：紧张、愤怒等强烈情绪

#### 3. 环境映射
```
窗外的雨越下越大，如同他此刻的心情
```
适用：烘托情绪、渲染氛围

#### 4. 对比反衬
```
周围人笑得越开心，他心里就越凉
```
适用：孤独、失落等情绪

### 禁忌
- ❌ 大段内心独白流水账
- ❌ 每个想法都要写出来
- ❌ 心理和动作表情割裂

## 十、亲密场景（教程22）

### 核心原则
**服务于剧情，服务于感情线，不是为了写而写**

### 写法要点

#### 1. 暧昧期最甜
```
直接写亲密：100分效果
写暧昧拉扯：120分效果
```

#### 2. 感官描写要克制
- ✅ 氛围、情绪、细节
- ❌ 太详细的身体描写
- 留白比写满更有想象空间

#### 3. 情感比动作重要
```
动作：拉窗帘、关灯...
情感：他的犹豫、她的紧张、两人的心理活动
```

#### 4. 网文尺度把控
- 以签约为目标：点到为止
- 以订阅为目标：可以稍微放开
- 无论如何：不能写违法内容

### 技巧
- 侧面描写：写观众反应，不写当事人
- 时间压缩：跳过过程，写结果
- 留白艺术：读者自己脑补

## 十一、综合应用

### 写作自检清单

| 检查项 | 快节奏 | 慢节奏 |
|--------|--------|--------|
| 每章有钩子 | ✅ | ✅ |
| 冲突有升级 | ✅ | ✅ |
| 情绪有起伏 | ✅ | ✅ |
| 描写详略得当 | ✅ | ✅ |
| 金手指使用有限制 | ✅ | ✅ |
| 伏笔有回收 | ✅ | ✅ |

### 写作顺序建议
1. 先写大纲：确定世界观、核心冲突、感情线
2. 设置伏笔：在写的时候就埋好
3. 控制节奏：按照"铺垫→发展→高潮→收尾"
4. 填充细节：打斗、对话、心理、外貌

## 十二、日常生活中收集写作素材

### 人设填充三要素
| 维度 | 说明 |
|------|------|
| 外貌因素 | 用细节代替形容词 |
| 心理因素 | 性格、动机、欲望 |
| 社会因素 | 身份、关系、背景 |

**错误示范**：
```
内心冷静、理智、御姐，身材娇小，瘦弱，说话快，文静，聪明
```

**正确示范**：
```
穿校服会把拉链拉到最顶上
```

### 让普通人不普通
| 普通职业 | 极致设定 |
|---------|---------|
| 公交车司机 | 时间循环爆炸案（《开端》） |
| 乡村教师 | 传授牛顿力学，让地球通过文明甄别（《乡村教师》） |

**核心**：把普通人放到极致的感情冲突、事件冲突中

### 灵感收集方法

#### 脑袋没东西时
- **填素材**：读书看报、社会热点、电影电视
- **填生活**：七大姑八大姨、邻居八卦
- **填转述**：把爆款故事讲给别人听

#### 脑袋有东西时
1. **先写**：能写多少写多少，掏空自己
2. **后加**：事无巨细，像长舌妇一样记录
3. **再修改**：拎出最重要的，修改是王道

### 爽点设计
> "怨憎会、爱别离、求不得、失荣乐"的反面

| 爽点类型 | 示例 |
|---------|------|
| 发泄不满 | 痛骂无良老板、报复作恶小人 |
| 满足期待 | 美女/帅哥一见钟情、被总裁赏识 |
| 智商角逐 | 亮底牌将对手安排得明明白白 |
| 成长逆袭 | 小人物一步步成长，挑战上层规则 |
| 情感高潮 | 百转千肠的守望离别，苦尽甘来 |

### 技巧：知乎搜索灵感
- **稳定话题**：「婚姻」「情感」「人性」「爽文」等关键词
- **命题作文**：搜「以一句话为开头写作」话题
- **沉淀故事**：提炼身边人故事的人设与冲突点

## 十三、读者爽点速查表

### 核心爽点公式
```
爽 = 不满发泄 + 期待满足 + 被赞叹 + 震撼他人
```

### 常用爽点场景
| 场景 | 爽感 |
|------|------|
| 打脸 | 主角用实力碾压嘲讽者 |
| 逆袭 | 废柴翻身变大佬 |
| 装逼 | 扮猪吃虎，高手低调 |
| 鉴宝 | 发现别人看不出的好东西 |
| 救美/英雄救美 | 关键时刻出现 |
| 奇遇 | 掉崖不死，得传承秘笈 |
| 升级 | 功力突破，等级飙升 |
| 夺宝 | 抢到稀有资源 |
| 复仇 | 君子报仇十年不晚 |
| 打脸反转 | 被看不起→逆袭打脸 |

### 毒点避坑
| ❌ 毒点 | 原因 |
|--------|------|
| 绿帽 | 读者无法接受 |
| 虐主 | 让读者不爽 |
| 圣母 | 没有爽感 |
| 无脑 | 缺乏智商角逐 |
| 太监烂尾 | 前面积累全白费 |
