## 一、杀人手法（推理/武侠通用）

### 1.1 密室杀人手法

| 手法类型 | 描述 | 适用场景 |
|---------|------|---------|
| 假密室 | 密室旁设有密道，藏匿方式五花八门（书柜机关、暗格等） | 武侠仇杀、门派争斗 |
| 心理型密室 | 利用错觉效果（如镜子）造成尸体消失的假象 | 诡计多端的阴谋 |
| 钥匙调包 | 调换密室锁或钥匙，制造不在场证明 | 嫁祸对手 |
| 道具利用 | 利用磁带、绳索等道具移入钥匙形成反锁密室 | 精密谋杀 |
| 共犯协助 | 凶手靠共犯完成密室逃脱 | 帮派作案 |
| 外界杀人 | 不进入密室，通过窗户空隙、门缝等远距离杀人 | 阴险暗杀 |
| 开放型密室 | 雪地/泥地中只留一人脚印，利用时间差作案 | 冬夜谋杀 |

### 1.2 意外伪装与嫁祸

| 手法 | 描述 |
|------|------|
| 车祸 | 对刹车做手脚或直接撞击 |
| 上吊 | 勒死后伪装自杀 |
| 坠落 | 制造错觉让死者自己坠落 |
| 假坠落 | 脚绑绳索，表演跳楼 |
| 苦肉计 | 自伤后嫁祸他人 |

### 1.3 密码暗号类型

- 四方密码（4个5×5矩阵加密）
- 二方密码
- 替换加密法
- 换位加密法

## 八、创作建议

### 8.1 武功设计原则

1. **内外兼修**：内功心法配外家招式
2. **相生相克**：设置克制关系增加戏剧冲突
3. **循序渐进**：主角成长路线
4. **独门绝技**：每个高手有标志性招式

### 8.2 阵法运用技巧

1. **以少胜多**：弱者靠阵法抗衡强者
2. **困敌为主**：阵法不求杀敌，但求困住
3. **配合兵器**：阵法与独特兵器结合
4. **破绽设计**：总有破阵之法

### 8.3 门派冲突设计

1. **正邪对立**：经典武侠主题
2. **门户之见**：不同门派规矩冲突
3. **夺宝争霸**：武林秘籍、神兵利器
4. **门派联姻**：政治婚姻化解恩怨
