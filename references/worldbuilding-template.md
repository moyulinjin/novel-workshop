# 世界观设定模板库

> 适用场景：创建新世界体系、构建力量等级、设计势力分布

---

## 一、世界观设定核心要素

| 要素 | 说明 | 必填程度 |
|------|------|----------|
| 世界基础 | 世界形态、时间线、地理环境 | ⭐⭐⭐ |
| 力量体系 | 修炼等级、能力来源、技能设定 | ⭐⭐⭐ |
| 势力分布 | 主要势力、利益关系、阵营对立 | ⭐⭐⭐ |
| 社会结构 | 阶层划分、规则制度、生活方式 | ⭐⭐ |
| 经济体系 | 货币系统、资源分布、交易方式 | ⭐⭐ |
| 禁忌与规则 | 世界法则、不可逾越的红线 | ⭐⭐ |

## 二、快速世界观框架（简化版）

适用于**都市/现代/近代**题材，30分钟内完成：

### 2.1 基础设定表

```
【世界背景】
- 时代：现代/近代/架空____年
- 地点：____（城市/国家/星球）
- 科技水平：____（与现实的差异）
- 特殊元素：____（超自然/异能/外星等）

【力量体系】（如有）
- 能力类型：____
- 能力来源：____
- 等级划分：____级（Lv.1→Lv.10）
- 觉醒条件：____

【主要势力】
- 势力A：____（立场、目标、代表人物）
- 势力B：____（立场、目标、代表人物）
- 势力C：____（中立/隐藏势力）

【社会规则】
- 普通人的生存状态：____
- 特殊人群与普通人的关系：____
- 禁忌/限制：____
```

### 2.2 设定示例（都市异能）

```
【世界背景】
- 时代：现代都市，2015年
- 地点：龙国、海城
- 科技水平：与现实相近
- 特殊元素：灵气复苏，部分人觉醒异能

【力量体系】
- 能力类型：元素系（风火雷电）、强化系、肉体系、精神系
- 能力来源：灵气觉醒、血脉传承、药物强化、机械改造
- 等级划分：F→E→D→C→B→A→S→SS→SSS（9级）
- 觉醒条件：18岁前首次觉醒，或遭遇生死危机强制觉醒

【主要势力】
- 官方：龙组（政府异能管理机构），负责维护秩序、镇压邪恶
- 民间：四方阁（正道联盟）、暗夜会（中立偏反派）
- 隐藏：古武家族、隐世宗门

【社会规则】
- 异能者需在政府登记，隐藏身份属违法
- 异能者犯罪由特殊法庭审判
- 普通人不知道异能者存在（信息封锁）
```

## 四、世界观设定速查卡

### 4.1 各题材核心设定

| 题材 | 必设要素 | 常见设定 |
|------|----------|----------|
| 都市异能 | 异能来源、等级、官方管理 | 灵气复苏/血脉觉醒/系统绑定 |
| 玄幻修仙 | 修炼等级、宗门势力、天地规则 | 凡→灵→仙→神；资源争夺 |
| 穿越古代 | 历史背景、社会阶层、科考/武侠 | 王朝更替、江湖恩怨 |
| 星际科幻 | 星际版图、种族关系、科技等级 | 联邦/帝国、殖民冲突 |
| 奇幻西幻 | 魔法体系、种族分布、神灵信仰 | 元素魔法、神术、巫术 |

### 4.2 世界观检查清单

- [ ] 世界起源/背景设定清晰
- [ ] 力量体系完整（等级清晰、能力明确）
- [ ] 主要势力≥3个，立场/目标明确
- [ ] 势力关系图谱已绘制
- [ ] 地理分区合理，有代表性场景
- [ ] 社会规则与世界设定一致
- [ ] 至少3个核心冲突点/剧情钩子
- [ ] 主角成长路径与世界设定匹配
- [ ] 力量上限/世界天花板已设定

## 五、剧情钩子生成器

基于世界观快速生成剧情冲突：

```
【冲突类型】→ 【来源】→ 【具体设定】

资源争夺 → 稀缺宝物/秘境 → 主角意外获得引来追杀
势力对立 → 正邪大战/王朝更替 → 主角被迫选边站
身份秘密 → 隐藏血脉/禁忌体质 → 身份暴露引发危机
灭门之仇 → 家族覆灭/师门被灭 → 复仇主线
灭世危机 → 天地大劫/外敌入侵 → 世界线剧情
```
