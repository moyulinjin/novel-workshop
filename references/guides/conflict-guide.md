# 冲突设计指南

## 一、冲突分类

### 1.1 人际冲突
- 主角vs反派
- 主角vs配角
- 配角vs配角

### 1.2 内心冲突
- 欲望vs理性
- 过去vs现在
- 责任vs情感

### 1.3 环境冲突
- 主角vs自然
- 主角vs社会规则
- 主角vs命运

### 1.4 目标冲突
- 不同目标对立
- 资源争夺
- 价值观碰撞

---

## 二、冲突层级设计

### 第一层：日常摩擦
```
强度：低
影响：小
持续：短
目的：铺垫人物关系

示例：
"你挡路了。"
"关你什么事？"
两人对视，火花四溅。
```

### 第二层：利益冲突
```
强度：中
影响：中等
持续：中
目的：推动剧情

示例：
"这个项目，是我的。"
"呵，你以为你是谁？"
商业竞争正式开始。
```

### 第三层：生死大仇
```
强度：高
影响：大
持续：长
目的：制造高潮

示例：
"杀父之仇，不共戴天！"
"那就来吧！"
两大家族，全面开战。
```

### 第四层：终极对决
```
强度：极高
影响：决定命运
持续：故事终点
目的：结局高潮

示例：
"今日，不是你死就是我亡！"
天道崩塌，万物寂灭。
```

## 三、冲突升级阶梯

### 冲突升级公式
```
冲突 = 利益冲突 × 情感冲突 × 身份冲突
```

### 四阶梯设计

**第一阶梯：言语冲突**
- 嘲讽、讥笑
- 口舌之争
- 不会造成实质伤害

**第二阶梯：行动冲突**
- 抢夺资源
- 公开羞辱
- 威胁、恐吓

**第三阶梯：对抗升级**
- 正面冲突
- 伤人/损物
- 势力对抗

**第四阶梯：生死对决**
- 暗杀、围杀
- 灭门、灭族
- 不死不休

## 四、反派设计三要素

### 要素1：嚣张资本
```
反派必须有嚣张的理由

常见设定：
- 实力强大
- 背景深厚
- 人数优势
- 掌握把柄

示例：
赵峰，上京四大家族之一赵家的继承人。
他确实有嚣张的资本。
```

### 要素2：恶行记录
```
反派必须对主角做过恶事

恶行类型：
- 抢夺资源
- 羞辱打压
- 伤害亲友
- 灭门惨案

示例：
三年前，赵峰带人打断了他父亲的腿。
三年后，他父亲郁郁而终。
```

### 要素3：可打脸点
```
反派必须有可被打脸的弱点

弱点类型：
- 实力水分
- 背景造假
- 隐藏秘密
- 心理弱点

示例：
"你以为赵家真的那么强？"
"呵，不过是一个纸老虎罢了。"
```

## 六、冲突场景模板

### 小冲突场景
```markdown
【场景】：日常摩擦

甲方：张三
乙方：林风

【冲突触发】
"让开。"
张三挡住林风的去路。

【冲突展开】
"我为什么要让？"
"因为我说让。"

【冲突升级】
张三推了林风一把。
林风退后一步，但没有还手。

【冲突结束】
"算你识相。"
张三扬长而去。

【埋伏笔】
林风看着张三的背影，微微一笑。
三天后的宗门大比...
```

### 中冲突场景
```markdown
【场景】：利益争夺

甲方：赵家少爷
乙方：林风

【冲突触发】
"这个名额，是我赵家的。"

【冲突展开】
"笑话，"林风冷笑，"这是我自己考来的。"
"你以为，有实力就够了？"

【冲突升级】
赵家动用关系，取消了他的资格。

【反击伏笔】
"好，很好。"

```

### 大冲突场景
```markdown
【场景】：生死大仇

甲方：灭门仇人
乙方：林风

【仇恨起源】

凶手，正是眼前这个人。

【压迫展现】

林风缓缓站起。
眼中，没有恐惧。
只有无尽的杀意。
"今日，我与你，不死不休。"
```

## 七、冲突与爽点结合

### 冲突→爽点公式
```
冲突铺垫（压抑） + 冲突爆发 + 冲突解决（爽点释放）
```

### 设计流程

1. **设计冲突**
   - 确定冲突双方
   - 设计压迫场景
   - 让读者愤怒/期待

2. **设计反转**
   - 主角隐藏实力
   - 关键时刻亮牌
   - 反派措手不及

3. **设计打脸**
   - 实力碾压
   - 身份揭露
   - 众人震惊

4. **设计余韵**
   - 短暂庆祝
   - 新冲突伏笔
   - 读者满足

## 八、冲突雷区

### ❌ 冲突过长
- 冲突拖太久读者会失去耐心
- 建议：每章至少推进一次

### ❌ 反派降智
- 反派为了被打脸变傻
- 建议：反派要聪明/强大，打脸才爽

### ❌ 主角只会挨打
- 主角一直挨打不还手
- 建议：即使还手失败，也要有智慧/骨气

### ❌ 冲突无意义
- 为冲突而冲突
- 建议：每个冲突都要推动剧情/展现人设

### ❌ 冲突后无成长
- 解决了冲突但主角没变强
- 建议：每次冲突后主角都要有收获
