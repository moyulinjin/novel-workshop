# 黄金开篇指南

## 一、开篇核心要素

### 三要素原则
```
第1章必须包含：
1. 穿越/重生设定（建立代入感）
2. 世界观简介（用冲突带出，不大段说明）
3. 身份处境（第3章前出现威胁或机会）
```

### 前300字检查
- [ ] 是否建立主角视角
- [ ] 是否抛出悬念/冲突
- [ ] 是否有记忆点
- [ ] 是否避免大段背景

---

## 二、开篇七技巧

### 技巧1：冲突前置
把冲突放在第一句
```
"砰！"
林风被一巴掌扇倒在地。
```

### 技巧2：悬念开场
先抛出疑问
```
三年前，他亲手把她送进监狱。
三年后，她出狱了。
```

### 技巧3：场景代入
直接进入场景
```
教室里安静得可怕。
林风看着面前的试卷，58分。
```

### 技巧4：身份反差
建立强烈对比
```
三年前，他是世界首富。
三年后，他在工地搬砖。
```

### 技巧5：对话先行
用对话带出信息
```
"废物！"
"我劝你还是滚出林家吧。"
林风抬起头，看着眼前这群人。
```

### 技巧6：情绪感染
直接传递情绪
```
不甘。
愤怒。
绝望。
林风握紧拳头，指甲陷入掌心。
```

### 技巧7：金手指暗示
让读者知道主角有底牌
```
就在他即将被踢出宗门的那一刻，脑海中响起一道声音：
"叮~检测到宿主困境，神级系统已激活..."
```

## 三、开篇类型模板

### 类型A：冲突式开篇
```markdown
【模板】
[激烈冲突/事件]
[主角处境揭示]
[金手指暗示/悬念]

【示例】
"滚！"
林风被一脚踹飞，重重撞在墙上。
他擦了擦嘴角的血，抬起头，看着眼前这群曾经对他点头哈腰的人。
"现在，轮到我了吗？"
```

### 类型B：悬念式开篇
```markdown
【模板】
[异常/谜团]
[主角视角]
[疑问抛出]

【示例】
三年前，他亲手把妻子送进监狱。
三年后，他站在监狱门口，等她出狱。
所有人都说他疯了。
只有他知道，这才是复仇的开始。
```

### 类型C：反差式开篇
```markdown
【模板】
[强烈对比]
[引发好奇]
[简要解释]

【示例】
世界首富躺在出租屋里，吃着泡面。
这是他重生的第三天。
距离他破产，还有三十天。
```

### 类型D：身份揭露式
```markdown
【模板】
[异常待遇]
[众人嘲讽]
[揭露身份]

【示例】
"就凭你？也配进入这个圈子？"
林风没有说话。
他只是掏出一张黑卡，轻轻放在桌上。
全场寂静。
"这位是...林氏集团的林总？"
```

## 四、开篇结构模板

### 前100字
```
建立场景 + 抛出悬念/冲突

示例：
"砰！"
林风被一脚踹飞，撞碎了大门。
"废物东西，还敢肖想我女儿？"
他趴在地上，嘴角溢出鲜血，但眼中的寒意更盛。
```

### 100-300字
```
揭示身份 + 展示困境 + 金手指暗示

示例：
林家，上京市四大家族之一。
而他，林风，是林家的废物赘婿。
三年了，受尽屈辱。
今天，更是要被扫地出门。
"叮~检测到宿主处境，神级系统激活..."
```

### 300-500字
```
冲突升级 + 建立期待 + 钩子结尾

示例：
林风缓缓站起身。
所有人都以为他会跪地求饶。
但他只是淡淡地看了一眼众人。
"从今天起，你们林家的好日子，到头了。"
```

## 五、开篇禁忌

### ❌ 禁忌1：大段背景介绍
```
错误：这是一个修仙的世界，共有九大宗门...
正确：林风看着眼前的宗门石碑，心如死灰。
```

### ❌ 禁忌2：主角一直挨打不还手
```
错误：反派打骂主角300字，主角没有反应
正确：反派打骂，主角隐忍，但在眼神/细节处暗示不会善罢甘休
```

### ❌ 禁忌3：拖沓不进入冲突
```
错误：先写主角起床、吃早饭、逛街...
正确：直接进入冲突场景
```

### ❌ 禁忌4：自言自语式内心独白
```
错误：林风想，我真是太惨了，为什么老天对我这么不公平...
正确：通过动作、神态、环境暗示内心
```

### ❌ 禁忌5：过多配角描写
```
错误：详细描写反派的长相、服装、出身...
正确：反派几句嚣张台词，然后主角反击
```

## 六、开篇质量检查

| 检查项 | 标准 | 是/否 |
|--------|------|-------|
| 第一句是否吸引人 | 是否制造冲突/悬念 |  |
| 前300字是否进冲突 | 是否有冲突展开 |  |
| 主角是否立体 | 是否有性格展现 |  |
| 是否有期待点 | 读者是否想继续看 |  |
| 是否有记忆点 | 是否有独特设定 |  |
| 是否避免大段说明 | 世界观是否用冲突带出 |  |
| 金手指是否暗示 | 是否暗示有底牌 |  |

## 七、开篇示例

### 示例1：都市重生
```markdown
"林风，签了这份离婚协议。"
妻子把协议拍在他面前，眼神冷漠。
林风看着协议，又看着这个女人。
他突然笑了。
上辈子，他签了。
然后被赶出家门，一年后病死街头。
这辈子...
"不签。"
他站起身，第一次直视她的眼睛。
"而且，我还要让你后悔。"
```

### 示例2：玄幻废柴逆袭
```markdown
"林风，滚下台！"
台下的嘲讽声震耳欲聋。
林风趴在地上，嘴角溢血，但眼神却异常平静。
因为只有他知道，今天，才是真正的开始。
三天前，他重生到这具身体。
三天后，他激活了传说中的至尊骨。
"你们想要我的位置？"他缓缓站起来。
"不好意思，我不打算让。"
```

### 示例3：总裁上门女婿
```markdown
"废物，连个碗都洗不干净！"
岳母的巴掌落在林风脸上。
他低着头，一言不发。
没人知道，这个"废物"女婿，是掌控万亿资产的神秘首富。
更没人知道，他正在等一个机会。
等她回来的那一刻。
"林风，"门被推开，妻子走进门，"该吃饭了。"
他抬起头，露出一个温和的笑容。
但没人看到，他眼中一闪而过的锋芒。
```
