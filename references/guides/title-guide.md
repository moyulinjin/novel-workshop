# 书名生成指南

## 一、核心原则

### 1. 一眼看懂
- 读者0.5秒内理解书名含义
- 避免生僻字/专业术语
- 避免过于抽象/文艺

### 2. 有记忆点
- 强烈对比或反差
- 有冲击力的词汇
- 意外的组合
- 数字或具体信息

### 3. 传达核心看点
- 金手指：系统、重生、穿越
- 身份：豪门、天才、总裁
- 爽点：打脸、逆袭、复仇
- 情感：甜宠、虐恋、破镜重圆

---

## 二、命名技巧库

### 技巧1：反差式
```
格式：[身份A] → [身份B]（反差）
示例：《被豪门赶出家门，我转身身家百亿》
```

### 技巧2：悬念式
```
格式：[动作/状态] + [悬念词/疑问】
示例：《重回1990，那一夜我做对了选择》
```

### 技巧3：数字式
```
格式：数字 + 具体信息
示例：《重生1990》《我有九十九个绝色师姐》
```

### 技巧4：身份揭示式
```
格式：[身份/地位] + [逆袭/崛起]
示例：《废物赘婿》《我真的是普通人》
```

### 技巧5：时代标签式
```
格式：时代 + 标签
示例：《回到2008》《八零年代小甜妻》
```

### 技巧6：职业/身份+反差
```
格式：[看似弱势身份] + [隐藏强势】
示例：《外卖员是隐藏大佬》《保安是隐形富豪》
```

### 技巧7：悬念制造式
```
格式：[悬念事件/疑问】+ [结果/状态】
示例：《妻子送我去坐牢后》《那晚我与鬼做了交易》
```

### 技巧8：直击爽点式
```
格式：核心爽点 + 具体化
示例：《打脸后我被团宠了》《修仙三万年无敌》
```

### 技巧9：多元素组合式
```
格式：身份 + 能力 + 目标
示例：《重生都市：开局获得万亿资产》
```

### 技巧10：氛围营造式
```
格式：[氛围词] + [身份/状态】
示例：《狂飙人生》《绝路逆袭》
```

## 五、爆款书名公式

### 都市重生类
```
公式1：被X后，我Y
示例：《被甩后，我成了首富》《被流放后，我靠空间逆袭》

公式2：重生XX，开局/这一世Y
示例：《重生2008，开局年入百万》《重生95，我有千亿农场》

公式3：我是/我乃X
示例：《我是一代神医》《我乃上门狂龙》
```

### 玄幻修仙类
```
公式1：开局获得X
示例：《开局获得神级系统》《开局获得上古传承》

公式2：签到/打卡X
示例：《签到三千年，我已成圣》《打卡十万年，我无敌了》

公式3：玄幻XX：XXX
示例：《玄幻：我的师尊是女帝》
```

### 甜宠言情类
```
公式1：X是我老婆
示例：《傅总，太太是我的》《陆少，夫人她又跑了》

公式2：婚后/先婚后X
示例：《婚后我被大佬宠上天》《先婚后爱：偏执大佬追妻记》

公式3：重生后被X宠
示例：《重生后被偏执大佬宠上天》
```
