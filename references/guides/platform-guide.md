# 平台投稿规范指南

## 一、主流网文平台特点

### 番茄小说
```
用户画像：年轻读者为主，免费阅读
题材偏好：都市、玄幻、穿越、系统流
风格特点：节奏快、爽点多、脑洞大
字数要求：签约2万字，连载中
推荐标签：重生、系统、打脸、甜宠
```

### 起点中文网
```
用户画像：老白读者，有一定阅读品味
题材偏好：都市、玄幻、科幻、游戏
风格特点：文笔要求高，剧情深度
字数要求：签约2000字试读
推荐标签：智商在线、格局大、人物立体
```

### 晋江文学城
```
用户画像：女性读者为主
题材偏好：言情、耽美、无CP
风格特点：感情线细腻，人设出彩
字数要求：签约3万字
推荐标签：甜、虐、苏、爽
```

### 飞卢小说
```
用户画像：追求爽感的读者
题材偏好：同人、玄幻、都市
风格特点：节奏极快，日更万字
字数要求：签约2万字
推荐标签：脑洞、爽文、同人
```

## 二、各平台文案规范

### 番茄文案模板
```markdown
【标题】一句话概括最核心看点

【正文】
前100字：直接冲突/金手指/身份反差
中段：展开主线冲突，说明看点
结尾：留下悬念/期待钩子

【标签】精准标签（2-4个）
```

### 起点文案模板
```markdown
【书名】精准定位
【简介】一段话概括全文

【分章简介】
第1-30章：
第31-60章：
第61-90章：

【卖点】核心爽点1-2个
```

### 晋江文案模板
```markdown
【文案】
一句话简介 + 核心CP + 萌点/虐点

【正文】
背景设定 + 主角介绍 + 相遇/冲突 + 感情发展 + 结局走向

【标签】1-3个核心标签
```

## 五、投稿注意事项

### 书名规范
- 避免过长（10字以内）
- 避免生僻字
- 突出核心看点
- 符合平台风格

### 简介规范
- 前三行决定读者是否继续
- 突出爽点/萌点
- 留下悬念
- 避免剧透太多

### 正文规范
- 前三章决定读者是否追读
- 开篇要快，避免大段铺垫
- 每章结尾留钩子
- 控制节奏，张弛有度

### 更新规范
| 平台 | 推荐日更 | 全勤要求 |
|------|---------|---------|
| 番茄 | 4000+ | 日更4000有全勤 |
| 起点 | 3000+ | 日更3000有全勤 |
| 晋江 | 3000+ | 周榜需稳定更新 |
| 飞卢 | 10000+ | 日更上万 |

## 六、签约与推荐

### 番茄签约
```
签约条件：2万字即可申请签约
签约难度：相对容易
推荐机制：8万字首次推荐，之后智能推荐
关键数据：阅读人数、追读率、完读率
```

### 起点签约
```
签约条件：编辑审核，门槛较高
签约难度：较难
推荐机制：分类推荐、各类榜单
关键数据：收藏、推荐票、打赏
```

### 晋江签约
```
签约条件：编辑审核，女频严格
签约难度：最难
推荐机制：霸王票、营养液、收藏
关键数据：收藏、积分、评论
```

## 七、快速入门指南

### 新手首选：番茄
```
适合：新手作者、快节奏爽文
优势：
- 签约容易
- 推荐算法友好
- 读者包容度高
- 全勤激励

起步策略：
1. 先写10-20万字练手
2. 学习番茄爆款套路
3. 积累读者反馈
4. 提升写作水平
```

### 进阶选择：起点
```
适合：有写作经验、想提升文笔
优势：
- 读者品味高
- 长篇更有发展空间
- 版权开发机会多
- 头部作者收益高

起步策略：
1. 先研究平台爆款
2. 提升文笔和剧情
3. 稳定更新
4. 积累粉丝
```

### 女频首选：晋江
```
适合：女性作者、感情线作者
优势：
- 女性读者集中
- IP改编机会多
- 感情戏有市场
- 小众题材也能生存

起步策略：
1. 研究晋江风格
2. 打磨人设和感情线
3. 稳定更新
4. 积累收藏
```
