## 二、晋江签约指导

### 核心要点

> **重要提醒**：晋江编辑签人不看数据！不看数据！签约看的是**潜力**，看的是**文是否符合晋江市场**！

### 签约流程

| 阶段 | 内容 |
|------|------|
| **新晋期** | 注册笔名后一个月内，唯一一次上新人榜单的机会 |
| **字数要求** | 最少存够1万字再申请签约 |
| **申请时间** | **建议周一申请**，编辑星期二、三较忙 |

### 入V标准

| 文类 | 入V收藏要求 |
|------|-------------|
| 耽美 | 500 |
| 言情 | 300 |
| 百合/无CP | 400 |

| 入V方式 | 说明 |
|---------|------|
| **顺V** | 七八万字达到数据要求，顺利开V |
| **倒V** | 字数无限，从25章开始倒V，入V收藏要求不变 |
| **完结V** | 完结文入V，收藏要求减少50个左右 |

### 签约被拒怎么办？

1. **只用改文案、文名、前三章**——不要大修全文！
2. 保持日更，再次申请
3. 如果题材热门、不是惨不忍睹，反复操作一般会让你过

### 新人生存指南

| 建议 | 说明 |
|------|------|
| **书名** | 不会取就挂热词，如"快穿之"、"重生之"、"系统之" |
| **文案** | 表明攻受/男女主属性，几句话剧情简介加撒糖段子 |
| **更新时间** | 每天3的倍数时间更新，蹭玄学 |
| **封面** | 淘宝店铺/论坛/自己做 |
| **日更** | 保持3000字节奏 |

### 曲线救国策略

> **强烈建议**：同时发一篇古言文，万字即可，**古言最好申签**。如果七天后耽美文没通过，用这篇曲线救国。

### 数据判断指标

| 指标 | 说明 |
|------|------|
| **首点** | 第一章点击量 |
| **末点** | 最后一章发布24小时后点击量 |
| **点收比** | 首点:收藏在10:1以内正常，超过需适当改文 |

| 爽感来源 | 恢复容貌/恢复智慧，反虐回去 |

| 要素 | 说明 |
|------|------|

| 类型 | 特点 |
|------|------|
| 性格反差 | 表面温润实则残忍；表面多情实则纯情；表面邪魅实则冷酷 |

### 写作技巧

#### 节奏公式

```
快（开头2万字）——缓（3万字后铺垫）——分（多条线索）——快（快速推进）——细（情感纠结）——缓（主题升华）——合（结尾）
```

#### 代入感技巧

| 要点 | 说明 |
|------|------|
| 前3万字 | 简单、明朗、快速的故事推进 |
| 拒绝 | 拖泥带水、场景细节描述 |
| 核心 | 一切文字服务于让读者轻松读懂故事 |

#### 情节设定技巧

| 要素 | 说明 |
|------|------|
| **矛盾事件切入** | 开篇即有冲突，快速吸引读者 |
| **男主出场** | 前3万字集中男女主角戏码，不要太慢 |
| **暧昧情节** | 开篇引入暧昧戏份 |
| **节奏** | 开篇要快，不拖沓 |

#### 无线文注意事项

| 要点 | 说明 |
|------|------|
| 章节卡点 | 1-20章1200字+，21章开始3200字+ |
| 付费节点 | **20章是关键**，21章开始计费 |
| 文风 | 小白文风、轻松文风优先 |
| 辞藻 | 不要过于华丽，通俗易懂即可 |
