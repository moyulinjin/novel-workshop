# 网文写作系统课程（15讲完整版）

> 本课程涵盖网文创作从入门到精通的全部核心技能，包括基础知识、签约出版、运营推广等全流程指导。

---

## 第1讲：网文写作基础知识解答

### 一、书名怎样写才吸引人？

书名固然是小说的重要门面，但也有很多书名不起眼的书大红特红。主要避免过于文青，能点出全文的重心即可。比如：一本玄幻小说，主角穿越到了异界，你不能为了名字好听就起个《花开百日红》这种满是文青味的名字，随大流叫个《XX异界逍遥记》都比让人看到名字就一头雾水的书名好。

### 二、简介怎么写才吸引人？

**核心要点：**
1. 每本书都有吸引读者眼球的重点，比如主角的金手指或显赫的背景，又或是现在很流行的高手下山流
2. 简介切忌诗词的格式，一般这种简介扑街成功率是非常高的
3. 简介尽量表露你这本小说的重点

**反面案例：**
- 主角得到某牛逼的东西，但遥远的星系正有一颗流星朝地球逼近，主角要怎样面对XX巨大的威胁……

**正确示范：**
- XX得到透视眼，从此各种风流各种牛逼

**核心原则：** 网文读者很浮躁，他们只想看到主角的牛逼，而不想看到主角倒霉。好不容易主角得到一个金手指，却面临着巨大的威胁……这不是有病是什么？金手指带来的期待感，就这样被破坏的一干二净。

### 三、大纲问题

作者分为**定制型**和**展开型**：
- **定制型作者**：在大纲框架下会游刃有余
- **展开型作者**：一旦有了大纲，就等于限制了自己灵活的脑袋，天马行空的想象力反而会被束缚

**建议：** 大纲是可以有的但不是必须有的，网文本身就是天马行空的产物，一旦作者本身对全文的大局感有了较大的把握，大纲真是个可有可无的东西。

### 四、人物刻画

读者和作者自身的感触是完全不同的，作者写的时候感觉生硬，不代表读者看的时候会生硬。只要情节还在发展，人物刻画的问题只能慢慢练，这是笔力和词汇量的问题。

### 五、竞技类小说写作

每个题材都有其自己的重点。争霸文就写争霸，复仇文就写复仇，竞技文当然要写竞技。理论上说，每个题材的小说都应该保持一种纯粹度，也就是抓住题材的重点然后往死了写。

### 六、场景转换

场景转化，每一次场景转换都尽量做到把上一个故事讲到了一定程度。尽量别东扯一下西扯一下，一旦故事的线断了，自然场景转换就变得生硬了。这个是大局观的问题。

### 七、如何让文更有代入感？

- **开篇代入感不外乎开篇故事的设置**，一开篇就能很快进入到第一个故事的讲述当中
- **很多新人会在开篇浪费大量的笔墨**，去描写一些无关紧要的内容，比如主角以前干嘛了，他是谁的儿子啊，这个世界是分为什么什么等级啦……这些全是败笔
- **开篇讲的不是主角的过去**，不是这个世界的各种等级，也不是这个世界的任何介绍，而是一出场，就进入了故事
- **一个牛逼的金手指**，也是帮助大部分作者最快抓住读者眼球的利器

### 八、如何提高描写能力？

笔力，说到底还是笔力问题，唯一的解决办法就是多看、多练、多学，没什么捷径可走。

### 九、如何把握小说灵魂（主线）？

小说的灵魂，也就是小说的主线。主角要干嘛啊？总得有个大概的发展方向吧？泡妞耍后宫？替父母报仇？要做世界之王？要带领地球舰队远征宇宙？有了目标之后，就要朝着这个目标不断的前进。

### 十、视角问题

谁说一定要男主和女主的视角了？小说的爽点很多，但**第三方装逼也是最常用的手法**。主角干了很牛逼的一件事情，你自己说自己厉害没用啊，得借其他配角的嘴巴，将主角的牛逼惊叹出来……这才够爽吧？

### 十一、如何让词汇更灵活？

文字其实是一种能够熟能生巧的东西。在你需要用到的时候，它们会很自然而然地出现在你的脑海当中。描写要突出重点，介绍的太繁复，反而会让读者觉得你是在注水。

## 第2讲：关于网文习作中实际问题的剖析

### 【案例一】女主角性格区分

**解决方案：做人物卡**

设定三个不同类型的女主角：

| 类型 | 性格特点 | 行为表现 |
|------|----------|----------|
| 小家碧玉型 | 随遇而安，顾小家而不顾大家 | 主角出事时最紧张、最担心、最各种凌乱 |
| 冷艳高傲型 | 眼高于顶，拒人于千里之外 | 默默帮助主角，而不是大喊大叫 |
| 外刚内柔型 | 强势示人，收起软弱 | 担心到极限，却嘴硬的不像话 |

### 【案例二】如何让读者热血沸腾？

你打游戏的时候为什么让你热血沸腾？因为升级，因为刷装备。而小说和游戏的区别是，小说需要**拉仇恨**。

**套路流程：**
1. 主角处于瓶颈期，需要突破
2. 需要宝物，途中被小队拉进团体
3. 七个人，宝物就一样，其他人率先对主角发难
4. 主角顺理成章干掉他们，得到宝物
5. 占据道理的制高点后，干掉心怀不轨的家伙

这就是所谓的让读者热血沸腾。

### 【案例三】如何让文简单明了？

**问题根源：** 对白在为什么情节服务不清楚

**解决方案：** 动笔之前先想好后面的情节怎么写，该怎么去安排，当意识到对白太多影响节奏的时候，先把后面的情节搞定了，有个明确的方向，然后最快收掉这段对白，把故事继续讲下去。

### 【案例四】第一人称写作

第一人称的小说，就是跟着主角的视线走。主角看不见的东西，对情节发展较为有用的，不可以写。

### 【案例五】思绪多却无法组合

想到了若干个情节，又不知道该怎么把他们串联起来？那就选择一个你认为最能打动你自己的写下去。别的记录下来，不管了。后面用得到就用，用不到就不要。

### 【案例六】除了对白还有什么推动情节的方法？

**旁白**也可以加速某个情节的进展，或切入中间，或直接跳过过程，说出结果。比如一个情节到最后，下一章直接变成了五年后。

### 【案例七】断句问题

**一句话三百字，你可以用若干个""号，将这段话分割成五个段落**，每个段落六十字，让读者知道这段内容还是这个角色在讲就可以了。

### 【案例八】前十章进入小高潮的套路

| 章节 | 内容 |
|------|------|
| 第一章 | 切入要讲的故事，结尾处引出某个反角对主角的不满 |
| 第二章 | 主角很难堪，可以是人家欺负你，也可以是旁白讲解主角的处境 |
| 第三章 | 主角的亮点，也就是他的金手指，让读者看到希望 |
| 第四章 | 反角给主角出了个难题 |
| 第五章 | 所有人都在议论这件事情，都认为主角无法办到 |
| 第六章 | 逆袭，高潮结束 |

**一万五千字以内搞定。**

### 【案例九】新人适合大网站还是小网站？

- **大网站**：平台成熟，读者群庞大，但大神多、新人较难被挖掘
- **小平台**：处于发展初期，相对重视培养和挖掘具有潜质的新人

### 【案例十】驾驭不了情节怎么办？

驾驭不了情节，说明大局观不行；东西表达不出来，说明笔力不行，词汇量不够。解决办法：**多看书，多写书，多学习**。

## 第3讲：关于卡文的解决方法

### 第一类：情节卡

**细分两类：**
1. 连主线都没有，全然不知道下面的情节该怎么发展
2. 有主线，但是具体情节该如何安排，依然是一头雾水

**解决方案：**
- 先想清楚主线，写个简易的大纲
- 弄清楚主角的目的，弄清楚主角的对手及可能遇到的困难
- 返回前两章内容，把自己完全放在主角的位置，想想主角在当时的情况会做怎样的反应
- 寻找下一目标和冲突点

**平时积累：**
- 多看小说、电视、动漫、社会现象等
- 随身带个小本子，脑袋里若突然闪现出一些情节，不要管它能不能用，先记下来再说
- 找朋友聊聊天，领域差别越大的朋友，越是有这种作用

### 第二类：言语卡

**问题表现：** 为了想一个合适的词语来描绘主角的相貌，能够卡上半个甚至一个小时

**解决方案：**
- 一部小说并不会因为一个词汇成功或者失败，同样的意思有很多表达方法
- 学会查资料：唐诗宋词历史传记成语字典等
- 不要为了语言华丽而华丽，这样会导致读者阅读困难
- 能身临其境的感受小说角色的心境，自然而然涌出的那些词汇，就是最适合的词汇

### 第三类：码字卡

**情绪懒惰：** 想想自己码字的初衷，想想自己的读者，想想坚持下来下一站自己就可能是大神

**外界干扰：**
- QQ、群消息等 → 关掉QQ静心写文
- 小孩子、客人等 → 先用两三句话记录情节，等处理好现实中的事情后，再快速码字

**关键原则：** 天道酬勤、勤能补拙、民生在勤！

## 第4讲：出版上架的小说目录审核规则

### 一、原创图书的目录审核

#### 1. 卷序规则

**首批完本的书籍：**
- 单卷：可以没有卷序号，直接以书名或"正文"来命名卷名
- 多卷：必须保证卷序号的连贯性以及卷单位的统一

**首批连载的书籍：**
- 单卷：若后续章节无其余卷名，则可以用书名或"正文"来命名卷名
- 若后续章节有其余卷名，需添加卷序号
- 多卷：必须保证卷序号的连贯性以及卷单位的统一

#### 2. 章序规则

- 章节序号"第*章 章节名"这样的形式出现
- 章节目录若为纯数字，需驳回添加章节名或统一改为"第*章"格式
- **章节序号必须保持连贯**
- **章节序号形式必须统一**（阿拉伯数字和汉字不能混用）
- 楔子、尾声、序、外传等可以有它自己的独立性，不必纳入逻辑序号体系

#### 3. 卷章名设置禁忌

1. 不得含有涉及敏感政治信息
2. 不得含有与性相关的关键词
3. 不得含有乱码、垃圾信息等
4. 不得出现正文内容
5. 不得出现作者名
6. 必须显示完整
7. 不得出现繁体字及除中文、英文外的他国文字
8. 不得以标点符号充当章节名
9. 卷章名前后要统一

## 第5讲：小说常见术语详解

### 一、精品文与冲量文

| 类型 | 定义 | 收入模式 |
|------|------|----------|
| **精品文** | 迎合当下热门题材、具有潜在市场价值且在签约前未发表在任何门户网站的精品小说 | 分成+基本定价、分成、一次定价 |
| **冲量文** | 一般质量的网文或失去市场价值的网络小说 | 几百元-数千元/整本 |

### 二、出版物的定价标准

出版物的价值取决于**定价而非字数**，主要看实体书的销售额。cp会买一些定价比较高的书上基地，因为基地关于图书上架的费用是根据图书自身的定价来设定的。

### 三、ISBN/书号

ISBN即书号，是衡量一本出版物是否通过正规途径出版的一个重要指标。

**书号结构：**
- 第一组：978为前缀，我国使用7开头
- 第二组：出版社代码（2-5位数字）
- 第三组：书序码（定长，9减去组号和出版社代码所占的位数）
- 第四组：计算机校验码（0-9）

## 第6讲：编辑眼中的好文章标准

### 1、书名和简介

书名就如同人的眼睛和脸，简介就像是一个女人的身材。读者接触一本书的顺序：书名 → 简介 → 评论区书评 → 目录小标题 → 章节内容

**好的书名简介的作用：**
- 起到浓缩大纲的作用
- 关系到读者对本书的兴趣
- 签约之后，大部份推荐都直接展示这一部分

### 2、开篇

> 一个好的开局，等同于半部好书。

**关键点：** 能在头几章、在二万字以内能发现亮点，一个让读者眼前一亮的情节。

### 3、故事情节

一本上架的书，无非就是为了让读者去订阅付费章节。书只能越来越吸引读者，让他们被篇中的故事迷住，最终订阅章节才算成功。

### 4、偷懒的找书方法

**编辑看书的顺序：**
1. 书名和简介
2. 章节小标题（好的标题对订阅能起到重要的帮助）
3. 一到三节的内容
4. 判断本书主题（三章看完了还不知所云，这本书就不算成功）

### 5、文笔

**判断文笔的方法：** 要看他最近更新过的章节，或者从中随机抓取一些章节，这才能得出真实感受。

**必扑的书：**
- 悲剧类，主角一辈子受欺付的永无出头之日
- 女主被人强奸、突然挂掉、红杏出墙或者半路跟人跑路

### 6、速度

对网文来说，**速度就是生命**。不管他用哪种方式在更新上有了速度，都说明他具备大神的潜质。

## 第7讲：阐述现代小说的开头艺术

### 第一类：人物类

以对故事人物的介绍或人物的自我介绍(如性格、个人经历等）开篇。

**代表作：**《小二黑结婚》《贫嘴张大民的幸福生活》

### 第二类：背景类

以描述故事发生的时间、地点，自然风景及社会环境等作为小说开头，以此来增加故事的真实性。

**代表作：**《红与黑》《亲亲土豆》

**注意：** 连篇累犊的环境描写容易引起读者的厌倦心理。

### 第三类：情节类

小说开篇就进入故事的发生状态，从人物的某种行为开始叙述。

**分类：**
- **顺叙式**：小说的开头与情节的"开端"呈同步发展关系
- **倒叙式**：开头就将故事的结局呈现给读者
- **中间叙式**：从情节发展到关键或精彩的中间环节开始

### 第四类：警句类

以诗句、引言、俗语、比喻或作者自创的简练深刻的话语开篇。

**代表作：**《安娜·卡列尼娜》——"幸福的家庭都是相似的，不幸的家庭各有各的不幸。"

### 第五类：闲谈类

一种漫不经心的、随随便便的谈话风格，像闲话家常，又像促膝谈心。

**代表作：**《沉香屑第一炉香》——"请您寻出家传的霉绿斑斓的铜香炉，点上一炉沉香屑，听我说一个战前香港的故事。"

## 第8讲：完善小说角色的表格绘制法

### 表格第一栏：性格特点

| 类型 | 说明 |
|------|------|
| **表征性格** | 人物在众人面前倾向表现的性格，如傲慢、狂妄、谨慎等 |
| **内在性格** | 人物内心深处的性格，可能在狂怒、无意间或亲近者对话中表现 |

### 表格第二栏：愿景

| 类型 | 说明 |
|------|------|
| **具体愿景** | 努力方向：知识、财富、享乐、权力等 |
| **潜意识愿景** | 在人物的潜意识之下，解释人物的突然行动 |

### 表格第三栏：社会背景

一个人所处的社会地位、教育水平、过往经历，决定了他的语言和行为模式。

### 表格第四栏：外表特点

相貌、衣着等等，虽然是社会背景决定的，但有很大的发挥空间。

### 表格第五栏：特殊性

| 要素 | 说明 |
|------|------|
| **心理障碍** | 强迫症、抑郁症、人格分裂症等 |
| **口癖** | "我是这样认为——""啊啦？"等鲜明口癖 |
| **特定禁忌** | 不吃饺子、不碰小猫、不杀小孩等 |

### 实例：下城区混混约翰的人物卡

**表征性格：** 自大，玩世不恭
**内在性格：** 谨慎，深思熟虑

**愿景：** 首要，自身安全；次要，财富
**潜意识愿景：** 拯救像他妹妹一样不幸堕落的少女

**社会背景：** 小时候在贵族区长大，双亲被谋杀前过着普通贵族生活

**外貌特点：** 金发碧眼的帅哥，脸上有一道淡淡的刀疤

**特殊性：**
- 轻度强迫症
- 口癖："您——"总是试图表现礼貌
- 禁忌：不会伤害任何比他年轻的女性

## 第9讲：如何炒作你的小说

### 一、书名、笔名、简介

**取名字的原则：** 简单、易记、顺口、有含义

| 好的例子 | 不好的例子 |
|----------|------------|
| 齐天大圣孙悟空 | ◎千☆幻¤冰★云⊙ |
| 千幻冰云 | 符号多、复杂 |
| 血红 | 难以记忆 |

**标题和简介的重要性：** 标题和简介使你的文章水准降低了一半的吸引力。

### 二、小说运营的计分标准

#### 一、更新速度（满分30）

| 更新频率 | 得分 |
|----------|------|
| 每章2001字 | 3分 |

## 第10讲：热门网文的标准情节设定

### 核心原理：过山车原理

**原则：** 高低起伏越大的剧情，达到的效果就越惊人。就像游乐园的过山车一样，高低落差越大的段落，乘客尖叫的声音也就越凄惨。

### 剧情节奏

**Y轴：** 高低起伏
**X轴：** 拍子（快的拍子给人紧张的情绪，慢的拍子给人平稳的情绪）

**应用：**
- 平稳剧情 → 慢节奏
- 冲击剧情 → 快节奏

### 吊读者胃口

小说就像作家与读者的斗智，安排的剧情一开始就被读者知道结局的话，读者没有想看下去的欲望。

**让读者上钩的技巧：**
- 透露一点点片段的信息
- 让读者急着想知道，又不能知道
- 慢慢被剧情牵着走

### 制造高潮

**集中所有力量为了高潮：**
- 时间流动
- 节奏
- 吊胃口
- 创意
- 主线
- 人物
- 场景
- 伏笔

**高潮要点：**
1. 多不如好
2. 出人意外
3. 持续的高潮要注意收放

### 人性是故事的心脏

> 任何作品吸引读者目光的地方不是单纯的画技、华丽的语句，而是动人的故事，而动人的故事往往是由人性表现出来的。

## 第11讲：常见小说类型的写作技巧

### 第一类：现代言情

**题材：** 豪门、虐恋、契约、宝宝、灰姑娘、总裁

**情节特点：** 开篇进展迅速，伴有大量暧昧情节，环环相扣

**人物设定：**
- **偏女强系列：** 女主身份背景较新颖，性格独立
- **传统虐恋系列：** 女主偏弱，善良执着却并非圣母

### 第二类：穿越幻想

**题材：** 女强、王爷、王妃、皇上、皇后、公主、将军

**人物设定：**
- **女强系列：** 女主身份特殊，特种兵、杀手、毒医、法医等
- **丑女大翻身系列：** 女主前世弱势，穿越后变得强大

### 第三类：都市暧昧

**读者群体：** 在广州打工的年轻男生

**写作要点：**
- 主角年龄不低于19岁
- 主角性格可以色色的、坏坏的，但不能粗俗、龌龊
- 主角心中正直，可以身边美女无数
- 多而不滥的女主：校花、护士、警花、小蜜、萝莉、御姐、老师

### 第四类：总裁豪门型

**男主角设定：**
- 高帅富是必须的
- 性格：腹黑、阴狠、面瘫、偏执等
- 要体现出一个人的两面性，优缺点兼在

**女主角设定：**
- 灰姑娘或豪门失落的千金
- 善良、坚持、脑子简单
- 越平庸简单，男主越人中龙凤，越给读者强烈的代入感

### 第五类：高干豪门型

- 男主角有高干背景，穿军装比较受欢迎
- 注意涉政问题

### 第六类：古代言情

**男主角：** 皇上、王爷、将军

**女主角：** 纯灰姑娘或豪门庶女

**题材建议：** 走古言小白轻松的风格，古代版的欢喜冤家

### 第七类：古代穿越

**禁止：**
- 暴力和血腥穿越
- 自杀类穿越
- 开青楼、做花魁

**创新方向：** 比较偏门的职业，在现代可能碌碌无为，或者很普通反而在古代大放异彩

### 第八类：玄幻仙侠

**玄幻类：** 男主是妖王、鬼王、龙王等；女主穿越成妖精或落到妖精窝

**仙侠类：** 男主角上古大神、九霄神仙等；女主单纯、善良、小白、圣母

### 第九类：男频都市文

**题材：** 都市暧昧、异能、重生、校园、办公室潜规则、乡土农民风

**主角身份：**
- 学生（兼职黑道老大，需年满18岁）
- 白领屌丝男
- 农民（走村官路线）

### 第十类：乡土小说

**关键要素：**
1. 突出乡土特色，方言质朴
2. 避免过多场景描写，不要抒情
3. 对手设置：村官、村痞、派出所警员逐步升级
4. 男主不要过于主动
5. 避开敏感词，减少直接床戏

### 第十一类：修真文

**核心要素：**
- 修炼方式具有创新性
- 废柴流最佳
- 男主性格要强势，有仇必报
- 金手指出现在前三章

### 第十二类：官场小说

- 草根男+异能
- 司机、秘书等身份受欢迎
- 不涉政

### 第十三类：游戏竞技文

- 屌丝进入游戏开始
- 重生记录数据
- 穿越异界获得升级系统

### 第十四类：青春校园文

**女主：** 平凡家庭出身或公主出身，性格一定要强，或霸道、或痞拽、或冰山、或开朗

**男主：** 非富即贵，性格或霸道、或邪魅、或专横、或冰山

## 第12讲：网络小说的创作技巧

### 一、人物个性的刻画

**人物表现要素：**
1. 独有的表情
2. 习惯的动作
3. 常用的对白
4. 思想与意念
5. 弱点和缺陷
6. 好恶习性

**重要：** 设定条列出来，有表现的部分做出记号，重要个性部分要不断重复表现出来。

### 二、配角的运用和衬托

**正面配角：**
- **导师型：** 给予主角正确知识和观念
- **爱慕型：** 主角心仪的对象
- **协助型：** 时常和主角形影不离

**不确定型：**
- **神秘型：** 对主角有正面或负面影响，不会自己报上名来
- **竞争型：** 实力比主角先占优势，促进主角成长
- **丑角型：** 搞笑而存在的角色

**负面配角：**
- **陷害型：** 和主角作对

### 三、桥段的发挥和设定

| 方法 | 说明 |
|------|------|
| **对比法** | 安排另一个桥段与主角做极度反差对比 |
| **堆栈法** | 把桥段布局安排到高度，再将主角叠到这个高度 |
| **陷阱法** | 将主角慢慢逼向绝境，然后大转变使位置正反颠倒 |

### 四、善用伏笔

**伏笔原则：**
1. 伏笔离不开主题
2. 不要使用过多的伏笔
3. 成为转化的力量与高潮的爆发点
4. 记得收尾

### 五、掌握主题

**多线架构使用原则：**
- 前后呼应
- 主次分明
- 懂得割爱

### 六、同中求异

找出成功作品的骨架、精神及精彩的要素，加入自己新的想法及个人特色重新包装。

### 七、格局的创意

| 类型 | 说明 |
|------|------|
| **点型创意** | 只用于一个小细节，类似一句有趣的对白或搞笑动作 |
| **关键型创意** | 属于剧情中重要桥段或转折点 |
| **架构型创意** | 足以支撑整个长篇故事的主要架构 |

### 八、发想创意的方法

| 方法 | 说明 |
|------|------|
| **市场分析法** | 运用两种以上的流行素材结合成为新的创作题材 |
| **趋势发想法** | 注意未来的趋势方向，越遥远的未来，想象空间越大 |
| **逆向思维法** | 和常人的思考逻辑完全相同，就没有新鲜感 |
| **欲望满足法** | 满足人们对现实环境的不满足 |

## 第13讲：怎样让你的小说引人入胜

### 趣味的两种类型

| 类型 | 读者心理 |
|------|----------|
| **好奇心** | "这究竟是怎么一回事？" |
| **持续注意力** | "以后会发生什么事？"+"小说里的人遇到那件事后会怎么办？" |

### 悬念的构成

**悬念 = 好奇心 + 期待**

### 小说结构三部分

#### 1. 开端

**两部分构成：**
1. **情境/问题：** 主角面临一个需要加以解决的问题
2. **解释性材料：** 展现造成问题的条件

**小说开端的关键：** 
- 要马上引起读者的兴趣
- 在获得读者持续兴趣之前，先激起好奇心

#### 2. 主体

保持读者兴趣，让故事不断发展。

#### 3. 结尾

给读者一个满意的结局。

### 抓住读者兴趣的方法

1. **选择标题：** 引人注目，激发联想，富于刺激性
2. **安排材料：** 让读者意识到作品中的人物在生活中正面临着严重的危机

## 第14讲：如何写好小说中的人物

### 人物表现的三个维度

| 维度 | 说明 |
|------|------|
| **对话** | 人说话总是对一件事说话，要具备"自我（自我意识）""情绪""对他" |
| **动作** | 动作要有气势，要有小说人物的情绪与个性表现其中 |
| **内心** | 内心独白和感情描述 |

### 对话要点

**三个要件：**
1. 自我（人物灵魂）
2. 情绪（表达于外的感觉）
3. 对他（一定有个主题）

**警惕：** 对话只做到情绪与对他，没有人物自我，会产生人格分裂的状况。

### 动作要点

**基础：** 情绪+个性

**示例：**
- 生气的人 → 捏拳头、顿足
- 内向害羞的人 → 手足无措的抓头发或低头

**警惕：** 不要只是单纯的一昧表达坐、站、跑、跳，善用文字修饰，让每个动作都涉及姿势、态度和表情。

### 内心描写

| 类型 | 说明 |
|------|------|
| **内心独白** | 互相的是对话，独白的是自我 |
| **感情描述** | 作者将自己立于人物立场，将人物内心以人物角度阐述 |

### 作家禁忌

**小说最忌讳作者的影子。** 小说写的是人，人是人，不是神。要让读者有思考的空间，思考内中的人物为何，而不是一昧的看作者在小说的旁白赞扬这人是如何。

### 剑宗与气宗

| 宗派 | 说明 |
|------|------|
| **剑宗** | 重外形，注重剧情、文笔优美等外在表现，华而不实 |
| **气宗** | 重内质，重视涵义、意境、考据、铺排等内在隐含 |

**建议：** 气剑双修，以气练剑，以气御剑。

## 第15讲：阻碍你成为网文大神的细节

### 一、书名问题

**禁忌：**
- 太冷僻或深奥
- 长多达十余个字
- 火星文、脑残体
- 又假又大又空

**原则：** 有距离才有美感，字数越多，对事物的定义作用越大，范畴局限也越明显。

### 二、作者名问题

**禁忌：** 英文名字、火星体、脑残体

**原因：** 成熟的、肯支付银子订阅的读者，对此类非主流文化现象相当不感冒。

**影响：** 口碑宣传被浪费，一本很出名的书，就是不知道作者是谁！

### 三、简介问题

**禁忌：** 学人写趣味性或标新立异的简介

**原则：** 除非有相当浓厚的搞笑或搞怪实力，千万别学人用这种拉风的简介。

### 四、人名问题

**男主角：** 注意不能取"苟大柱、熊二毛"之类的大名

**女主角：** 千万别取"马芙蓉、牛小花、毛三妹"之类的雷人芳名

**暗喻类：** 避免"王赤石（王吃屎）、赵银棠（赵淫荡）"之类

### 五、把"经验之谈"奉为圭臬

**原则：**
- 任何人的经验都不应该成为左右你创作的准绳
- 只能将这些当成参考
- 最主要是必须具有很好的心态，以及超强的创作热情

## 附录：签约与出版全攻略

### 一、签约标准（男频）

**编辑审文要素：**
1. 书名、题材
2. 简介
3. 开篇第一印象
4. 文本质量（剧情、故事、文笔流畅度、节奏紧凑度、人物刻画、文字亲和力、原创性、新颖度）
5. 市场潜力（亮点卖点）

**大众网文好签的共同特征：**

| 序号 | 要求 |
|------|------|
| 1 | 第一章人物不超过三个 |
| 2 | 开篇要有矛盾的出现（故事切入点） |
| 3 | 开篇要让读者知道是什么类型的小说 |
| 4 | 三百字内出现主角，最迟不超过五百字 |
| 5 | 围绕主角视角写 |
| 6 | 故事紧凑，发展要快 |
| 7 | 场景设定要小，由小到大 |
| 8 | 分段一般不超过三行 |
| 9 | 玄幻修真仙侠金手指要开的快 |
| 10 | 仙侠修真需要一定的文笔 |
| 11 | 不要出现穿越的过程 |
| 12 | 恐怖惊悚要制造紧张气氛 |

**编辑不签的六种书：**
1. 断更的
2. 文本问题（繁体字、火星文、排版问题）
3. 宣传他站信息
4. 文不搭意、书名内容不知所云
5. 作者预期过高
6. 非小说类（诗歌、话剧等）

### 二、签约倾向

**设定要求：**
- 世界设定和升级体系设定简单明了、等级分明
- 使用已经成熟的升级体系
- 新人最好不要自造升级体系

**主角要求：**
- 最好是**人类**
- 主角在通常状态下最好是**人形**

**主线剧情：**
- 绝大多数作品体现升级流
- 由低到高的升级，升级才有快感

### 三、晋江签约指导

**签约时机：** 字数在**2万-5万**期间去后台申请

**编辑选择：** 站短排名第一个通常是组长，一般不签新人；好攻克的是后面两位及实习编辑

**雷区：**
1. 45°文艺忧伤、矫情不堪的青春伤痛文学
2. 纯对话流却没故事进展
3. 文笔造作
4. 情节逻辑各种漏洞
5. 青春蛋疼文艺忧伤

**好的文名和文案要点：**
- 一目了然，不要不知所云故作悬疑
- 不要用太奇葩的字体
- 不要花花绿绿加太多花哨

### 四、作品安全审核

| 类型 | 禁忌 |
|------|------|
| **涉黄** | 对床戏大肆直白露骨的描写 |
| **涉政** | 辱骂国家领导人或军队，扇动不利国家民族团结 |
| **涉暴** | 大肆添加恶心恐怖情节如吃人、分尸等 |

### 五、版权问题

| 类型 | 说明 |
|------|------|
| **侵权度** | 同人小说容易侵权，尤其是仿动漫剧本 |
| **版权混乱** | 一书多签造成版权混乱 |
| **版权不完整** | 电子版权已签约给别的网站 |
