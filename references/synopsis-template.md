# 小说简介/文案模板库

> 适用场景：书名生成后、投稿发布前，根据平台风格生成吸引读者的文案

---

## 一、模板类型总览

| 类型 | 特点 | 适用题材 | 字数 |
|------|------|----------|------|
| 悬念型 | 开篇抛问题，吸引继续读 | 悬疑/都市/玄幻 | 200-300字 |
| 冲突前置型 | 先展示矛盾冲突 | 爽文/复仇/逆袭 | 150-200字 |
| 情绪引导型 | 先引发共鸣或愤怒 | 甜宠/虐文/治愈 | 200-300字 |
| 金句开场型 | 名言/金句开头 | 文艺/现实/成长 | 150-250字 |
| 身份反差型 | 身份落差引发好奇 | 总裁/马甲/穿越 | 180-250字 |

---

## 二、悬念型模板

**结构**：背景痛点 → 意外转折 → 核心悬念

```
我以为这辈子就这样了。
[描述主角困境：如被背叛/一事无成/绝症等]
直到那天，[意外事件]，一切都不一样了。
[金手指出现或重生]
现在，他们求我回去，
可他们不知道，我已经不是当年那个人了。
```

**示例（都市重生）**：
> 上辈子我是全家的提款机，妻子出轨，还被亲弟算计欠下百万巨债，最后惨死街头。
> 重生回到高考那天，这次我不会再犯同样的错误。
> 只是没想到，这一世的我，竟然还觉醒了一个能看见未来的眼睛……

**示例（玄幻修仙）**：
> 三百年了，那个人终于从镇魔塔里出来了。
> 当年他以一己之力镇压魔域，却被正道联手封印。
> 现在，整个修仙界都在颤抖，因为没人知道，他出来后做的第一件事会是什么……

## 三、冲突前置型模板

**结构**：冲突场景 → 身份揭示 → 逆袭暗示

```
所有人都以为[主角身份/处境]。
[描述被轻视/被打压的场景]
直到那一天，[特殊事件]，
[揭示真实身份/展现隐藏实力]
他们才知道，自己得罪的，究竟是什么人。
```

**示例（总裁马甲）**：
> "废物，连个项目都做不好，滚出去！"
> 这是我这个月第17次被经理当众羞辱。
> 我低头道歉，走出办公室，然后拿出手机发了条消息：
> "把xx项目收购了。"
> 对，我确实是来体验生活的。

**示例（神医下山）**：
> 村口那个治好了癌症的老头，说他是什么神医传承人？
> 还说我命格有问题，必须下山结婚才能破解？
> 我堂堂神医门少主，下山就下山，谁怕谁！
> 只是这未婚妻……怎么看着有点眼熟？

## 四、情绪引导型模板

**结构**：情绪共鸣 → 代入场景 → 转折承诺

```
你有过[情绪痛点]的经历吗？
[描述让人共情的场景]
那种[绝望/心酸/无奈]的感觉，我太懂了。
这一次，[主角]告诉你，
[解决承诺/逆袭宣言]
```

**示例（甜宠）**：
> 凌晨三点，一个人加班到深夜，没人心疼，没人问候。
> 你是不是也这样，在城市里孤独地活着？
> 可是没关系，总有一个人，会穿越人海来爱你。
> 他说：从遇见你的那天起，我的世界才有了光。

**示例（虐渣）**：
> 被绿茶陷害、被闺蜜背叛、被男友当备胎……
> 你是不是也曾在深夜里委屈到哭？
> 重活一世，我终于明白：善良要给对人。
> 这一世，我只为自己而活，欠我的，我会一分不少讨回来。

## 五、金句开场型模板

**结构**：名言/金句 → 解读延伸 → 故事印证

```
"xxx"[名言/金句]
[对金句的解读]
而我的故事，就是这句话最好的诠释。
[简述主角经历，与金句呼应]
```

**示例**：
> "所有的离开，都是蓄谋已久。"
> 我曾以为这句话是心灵鸡汤，直到亲眼看着相爱三年的人，
> 牵着我最好的闺蜜走进酒店。
> 那晚我在雨里站了三个小时，然后重生了。
> 重生后的我只想说：谢谢你的不娶之恩。

## 六、身份反差型模板

**结构**：强烈反差 → 身份揭示 → 认知颠覆

```
在[普通身份]的皮囊下，
藏着[高贵/强大/神秘]的灵魂。
[描述表里不一的对比]
[见证者的震惊反应]
这就是你惹不起的人。
```

**示例（战神归国）**：
> 她是公司里最卑微的实习生，被所有人欺负。
> 他是工地上的搬砖工，被业主当众羞辱。
> 可没人知道，她其实是顶级豪门流落在外的千金，
> 而他，是让整个地下世界闻风丧胆的战神。
> 当身份揭开的那一刻，所有人都后悔了……

## 七、各平台字数规范

| 平台 | 简介字数 | 风格要求 |
|------|----------|----------|
| 起点中文网 | 200-300字 | 偏剧情向、悬念感强 |
| 番茄小说 | 150-200字 | 偏口语化、直白爽 |
| 晋江文学城 | 100-150字 | 偏文艺、情感细腻 |
| 飞卢小说网 | 100-150字 | 偏节奏快、噱头足 |
| 掌阅 | 200-300字 | 偏故事性、有看点 |

## 八、高转化文案公式

**公式1：痛点+爽点+悬念**
```
[痛点描述] → [金手指出现] → [冲突升级] → [留钩子]
```

**公式2：身份反差+实力展示+认知颠覆**
```
[底层身份] → [隐藏实力] → [打脸时刻] → [余韵悠长]
```

**公式3：情绪引爆+情感共鸣+承诺兑现**
```
[共情场景] → [情绪释放] → [治愈承诺] → [期待感]
```

## 九、爆款文案检查清单

- [ ] 前三行必须抓住读者（制造悬念或冲突）
- [ ] 有明确的情绪钩子（愤怒/期待/好奇/心疼）
- [ ] 身份反差或实力对比明显
- [ ] 有具体的场景或细节（避免空泛）
- [ ] 结尾留有悬念或期待感
- [ ] 字数符合目标平台规范
- [ ] 无敏感词/违禁内容
- [ ] 读起来流畅、口语化
