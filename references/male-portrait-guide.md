# 古风男性外貌描写指南

> 本指南整合了古风男性描写词汇、古代服饰词汇、发髻词汇及经典诗词引用，适用于小说创作中的人物外貌描写。

## 一、面容五官描写

### 眉目传情

**眉型描写：**
- 剑眉、星眉、朗眉、蚕眉、清眉、卧蚕眉、远山眉、刀裁眉
- 眉如远山、眉若远黛、眉峰如聚、眉如墨画、眉分八彩
- 双眉斜飞入鬓、眉宇英挺、眉目清朗、眉宇间透着英气

**眼神描写：**
- 星眸、朗目、灼灼目光、目光如炬、目光如电
- 眸光清冷、眸中带笑、眸清似水、眸底秋波流转
- 眼尾微挑、丹凤眼、桃花眼、狭长凤目
- 目似寒星、眼波流转间动人心魄

**面部轮廓：**
- 面如冠玉、颜如舜华、面若中秋之月
- 棱角分明、五官端正、面容俊朗、相貌堂堂
- 剑眉星目、唇红齿白、仪表不凡、气宇轩昂
- 面如傅粉、脸如雕刻、颜如宋玉、貌比潘安

### 神情气质

**静态神情：**
- 神采奕奕、神采飞扬、神清骨秀、气宇不凡
- 温文尔雅、斯文优雅、气质如玉、温润如玉
- 清冷出尘、遗世独立、超然物外、清贵高华

**动态神态：**
- 浅笑盈盈、似笑非笑、嘴角微扬、唇边噙笑
- 眸光微沉、神色淡然、剑眉微蹙、眉宇轻皱
- 嘴角勾起一抹淡笑、眸中闪过一丝锐利

## 五、气质类型描写

### 儒雅书生型
- 面如冠玉、温文尔雅、斯文俊雅、眉清目秀
- 手持书卷、气质如玉、清贵出尘、才情兼备
- 唇角含笑、眸光温润、谈吐不凡、风流蕴藉

### 江湖侠客型
- 剑眉星目、棱角分明、神采飞扬、意气风发
- 身形矫健、步履轻盈、衣袂飘飘、洒脱不羁
- 眸中带笑时玩世不恭、眸光冷厉时杀气凛然

### 王侯将相型
- 气宇轩昂、相貌堂堂、不怒自威、气势不凡
- 龙章凤姿、天日之表、龙凤之姿
- 举手投足尽显贵气、雍容华贵、气度恢弘

### 清冷出尘型
- 面若冰霜、神情淡漠、气质出尘、遗世独立
- 眸光清冷如霜雪、眉宇间透着疏离
- 不食人间烟火、仿若谪仙

### 邪魅狂狷型
- 唇角勾起一抹邪笑、眸光深邃、野心勃勃
- 眉峰微挑、似笑非笑、亦正亦邪
- 举手投足间带着几分玩世不恭的危险气息

## 七、描写句式模板

### 整体出场

```
只见来人【年龄】许，【面容特征】，【身材】，
【衣着描写】，周身【气质描写】，
当真是【总结性评价】。
```

### 五官特写

```
【眼睛】如【比喻】，【眉毛】如【比喻】，
【鼻梁】【特征】，【嘴唇】【特征】，
【面容整体评价】。
```

### 神态动作

```
他【动作】，【眼神】，
【嘴角/表情】，
【内心/气质】尽显。
```

### 综合描写

```
那【人称】——
【面容】、【身材】、【衣着】、【气质】——
只见他【动作神态】，
【细节特写】，
【总结评价】。
```

## 八、创作建议

1. **抓住特征**：根据人物性格设定选择相应的外貌特征
2. **层次分明**：从整体到局部，从远及近描写
3. **动静结合**：静态外貌与动态神态相结合
4. **服饰契合**：衣着风格需与人物身份、性格相匹配
5. **点睛之笔**：最后用一句诗词或比喻升华人物形象
6. **详略得当**：重点描写能体现性格的部位，其他略写
