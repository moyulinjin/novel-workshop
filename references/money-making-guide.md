# 赚钱小说核心法则

> 从读者心理到商业变现的实战指南

---

## 一、什么书能赚钱

### 核心公式

```
赚钱 = 代入感 + YY + 读者爽感
```

### 读者爱看什么？

1. **生动有趣的故事**，曲折离奇的情节
2. **强烈的代入感**，让读者觉得爽（YY）

### 代入感法则

**核心原则**：让读者把主角当成自己

**操作技巧**：
- 避免无关细节，尽量模糊化处理
- 主角设置成年轻男性（十几岁到二十几岁）
- 给主角加一个小缺点增加亲和力（如路痴）

**反面教材**：传统武侠式微的原因就是代入感问题——主角生于古代，思维与现代人脱节

### YY核心要素

| 要素 | 说明 | 示例 |
|------|------|------|
| 奇遇 | 主角得到特别好处 | 掉崖不死得到秘笈 |
| 升级 | 实力不断变强 | 功力又精纯了 |
| 寻宝 | 物质获得 | 捡到宝物送人 |
| 泡妞 | 美人相伴 | 保持暧昧比实际得到更值钱 |
| 发财 | 金钱获得 | 成为首富 |
| 欺人 | 打倒敌人踩脚下 | 把敌人描写强大再打败更爽 |
| 助人 | 帮助他人 | 扔废物给别人，对方感激涕零 |

### 泡妞的正确姿势

**后宫 ≠ 种马**

- **种马**（下乘）：见一个上一个，读者看免费章节就够了
- **后宫**（上乘）：见一个收一个，保持暧昧，让读者期待

**核心技巧**：美人没得到之前最有价值，让读者猜什么时候能搞定

### 毒点规避

- 主角不能是窝囊废
- 读者承受能力差，挫折过大必反弹
- 女主角贞节被夺 → 读者必跑

## 二、新人冲新书榜攻略

### 冲榜基本规则

**入库条件**：
- 入库时间一个月以内
- 公众版字数：5万-15万之间
- 三天内有更新

**积分计算**：
```
积分 = 推荐票×20 + 点击 + 收藏÷5
```

### 冲榜策略

**存稿准备**：最好有7-8万字存稿

**入库时机**：
- 编辑工作时间：周一到周五
- 周末编辑不审书
- 控制字数达到3万的时间在上半周

**更新节奏**：
| 阶段 | 策略 |
|------|------|
| 刚入库 | 多传章节，不用等6小时 |
| 3万字时 | 编辑审核，关注期 |
| 冲榜稳定后 | 放慢更新，延长在榜时间 |
| 快出榜前 | 放慢速度，不要太快消耗字数 |

**禁忌**：
- ❌ 用资料充字数（编辑会无视）
- ❌ 签约后疯狂更新冲榜（每个字都是钱）

### 编辑审核秘密

当新书字数达到3万字时，会出现在编辑小组后台进行交叉审核：
- 质量好 → 直接签约 → 安排推荐
- 质量一般 → 继续观察

**所以**：早点达到3万字，早点让编辑看到

## 三、抓住读者心理

### 核心问题

> 为什么读者会追你的书？

### 答案

**读者追求的是什么**：
- 物质：金钱、权力、力量
- 情感：爱情、友情、认同
- 精神：好奇、刺激、爽感

### 写作中的关键点

**钩子技巧**：
- 开头就要抓住读者
- 每章结尾留悬念
- 让人想知道"然后呢"

**期待感建立**：
- 目标明确：读者想知道能否达成
- 身份期待：想知道揭露什么
- 实力期待：想看爆发

### 读者心理时间表

| 时段 | 更新效果 |
|------|----------|
| 晚9-10点 | 高峰期 |
| 中午12-1点 | 小高峰 |
| 晚7-10点 | 黄金时段 |

## 四、上架前后吸引力提升

### 上架前

**冲新书榜**：见第二章冲榜攻略

**存稿节奏**：
- 存够上架的字数
- 每天固定时间更新
- 保持2-3更

### 上架后

**核心任务**：让读者继续付费

**技巧**：
1. 每章要有实质进展
2. 保持悬念和期待
3. 定期爆发爽点
4. 控制节奏，不要太快写完

**日更策略**：
- 质量优先
- 不要断更
- 保持稳定比数量重要

## 五、万能写作公式

### 题材转换法

**核心思想**：什么题材都能写，只是换了层皮

**操作**：
1. 确定你想写的核心故事
2. 换成不同题材背景
3. 调整人设和设定

**示例**：
- 传统武侠 → 现代都市 → 星际未来 → 魔法异界
- 核心套路不变，包装不同

### 公式总结

```
好书 = 强代入感主角 + 清晰目标 + 层层爽点 + 悬念钩子
```

**自查清单**：
- [ ] 读者能把主角代入成自己吗？
- [ ] 主角的目标明确吗？
- [ ] 每章有爽点吗？
- [ ] 章尾有钩子吗？
- [ ] 读者知道"然后呢"吗？

## 六、上架前后吸引力提升（教程41）

### 核心原则
> 上架作品销售成功的秘诀：让读者看你的书成了一种习惯，要看完你埋设的伏笔才甘心。

### 前三万字法则
1. **必须干净**：去掉作者语、感言、请假条等无关内容
2. **层层递进**：前5000字、前1万字、前3万字都有强吸引力
3. **人物到位**：主角塑造到位，配角不抢戏
4. **别藏宝**：精彩在十几万字？对不起，读者没耐心

### 上架点选择
- 参考评书技巧："预知后事如何，且听下回分解"
- 上架前掐准**高潮点**，让读者为看完情节而付费
- 拙劣做法：脱光衣服后捂三点，拿钱才给看

### 终极大杀器：存稿
| 阶段 | 日更目标 |
|------|----------|
| 稳 | 3000字/天 |
| 好 | 6000字/天 |
| 红 | 10000字/天 |

### 节奏控制
- **不是越爽越好**：一直高潮会让读者"饱"，失去期待感
- 正确节奏：吊胃口→喂饱→消化→再吊胃口
- 爆发策略：稳定更新+定期爆发，效果比纯日更更好

## 七、抓住读者心理（教程6）

### 基本功10条
| 要求 | 标准 |
|------|------|
| 错字率 | < 0.5% |
| 标点 | 规范 |
| 逻辑 | 无错误 |
| 段落 | ≤400字 |
| 括号 | 不用 |
| 旁白 | 不加 |
| 名言 | 不放 |
| 合理 | 可推敲 |
| 拖沓 | 无 |
| 郁闷 | 不写 |

### 读者群分析

#### 上班族
- **心理**：压力大、存在感弱、危机感强
- **需求**：满足感
- **写法**：写主角蹂躏老板，让读者爽

#### 学生
- **心理**：无压力、充满幻想、对社会不满
- **需求**：发泄
- **写法**：制造让读者恨的情节，再让主角踩死

### 代入感培养
- 以单主角成长为主线
- 配角戏份不超过主角
- 不要让主角消失超过5-6章

## 八、题材换皮术（教程2）

### 核心思想
> 网络小说的题材，只是一个"壳"。核心是人物成长体系。

### 套壳原理
```
网游小说 = 网游壳 + 背景壳 + 人物成长
```

去掉网游壳 → 西方奇幻/东方玄幻/都市...

### 真正的核心
**人物成长体系**才是小说的灵魂

| 壳（题材） | 可替换 |
|-----------|--------|
| 都市 | 黑道/职场/校园 |
| 仙侠 | 玄幻/武侠/西幻 |
| 历史 | 穿越/架空 |

### 操作步骤
1. 先想好**人物成长体系**（升级路线）
2. 再选择**背景题材**（流行什么选什么）
3. 调整体系**适配题材**
4. 加入调味料：爱情/恩仇/金钱/权利

### 避坑
- 人物性格要与题材气质匹配
- 仙侠→要飘逸，不能坑蒙拐骗
- 黑道→要狠辣，坑蒙拐骗反而加分
