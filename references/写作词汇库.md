## 一、动作描写的定义与作用

### 1.1 什么是动作描写

动作描写是对人物行为、动作的描写。通过描写人物**做什么**和**怎样做**，来表现人物的性格特点和精神面貌，凸现人物的个性特征——思想的、品格的、性格的、心理的以及习惯的。

### 1.2 动作描写的四大作用

| 作用 | 说明 |
|------|------|
| **表现内心世界** | 通过外在动作揭示人物内心情感 |
| **揭示性格特征** | 动作往往比语言更真实地反映性格 |
| **推动情节发展** | 关键动作推动故事向前发展 |
| **塑造人物形象** | 让读者"如见其人"，使人物"立起来" |

### 1.3 经典范例

> **《背影》朱自清**
> 
> 他用两手攀着上面，两脚再向上缩；他肥胖的身子向左微倾，显出努力的样子。

> **《武松打虎》**
> 
> 那大虫咆哮，性发起来，翻身又只一扑，扑将来，他又只一跳，却退了十步远。那大虫恰好把两只前爪搭在他面前。他将半截棒丢在一边，两只手就势把大虫顶花皮胳答地揪住，一按按将下来。

> **《雪地捕鸟》鲁迅**
> 
> 他扫开一大块雪，露出地面，用一枝短棒支起一面大的竹筛来，下面撒些秕谷，棒上系一条长绳，人远远地牵着，看鸟雀下来啄食，走到竹筛底下的时候，将绳子一拉，便罩住了。

## 二、动作描写的方法与技巧

### 2.1 方法一：细致入微地描摹动作

**核心要点**：把人物动作分解成多个小动作，逐一描写

**示例对比**：
```
❌ 简单版：放学了，他蹦跳着回家去。

✅ 详细版：放学了，他把书本练习簿什么的一古脑儿往书包里一塞，书包扣带只扣了一个，
就把书包往肩上一撂。蹦跳着，还哼着小调。到了校门口，他左右一望，正是车流空隙，
就一窜过了马路。他拐过一条小街，这才慢慢走着。
```

### 2.2 方法二：选用准确的动词

**核心要点**：选择最精准、最有表现力的动词，避免空洞笼统

**动词精细分类**：

| 动作类型 | 初级词汇 | 进阶词汇 | 高阶词汇 |
|----------|----------|----------|----------|
| **经过某处** | 走、跑 | 跳、奔、跃、越 | 踏、蹦、跨、蹿、蹬、蹑 |
| **拿取物体** | 拿、握 | 捏、攥、抽、抠 | 捻、拈、攫、掂、攥、揪 |
| **喝水动作** | 拿、喝 | 伸、挤、拧、揭 | 甩、啜、咕嘟、噙、咂、嘬 |
| **手的动作** | 打、推 | 拍、拨、揉、搓 | 捻、揪、抠、挠、攥、掂 |

### 2.3 方法三：选择典型动作

**核心要点**：抓住最能揭示人物性格或心理的动作进行描写

**示例**：
- 孔乙己"排出九文大钱" → 表现穷困潦倒却要面子的性格
- 葛朗台临终"抓"十字架 → 表现贪婪的本性
- 严监生临死"伸两根指头" → 表现吝啬到极点

### 2.4 方法四：分解动作+感受

**三步法**：
1. **分阶段**：一开始 → 后来 → 最后
2. **分部位**：脸色、眼睛、嘴巴、身体、手、腿等
3. **写感受**：人物心理或作者感受

**示例**：
```
师命难违，只见王明走上讲台，右手拿起板擦，高高扬起右臂...
王明稍稍迟疑了一会儿，身体微微下蹲，用尽全身的力气，猛地向上一跳，
同时右手顺势向右一挥...大家被他逗得哄堂大笑，有的同学竟然笑得直不起腰来。
王明扭过头，眼睛一瞥，吐了吐舌头，脸上洋溢着得意与自豪...
```

## 六、动作描写的五个关键

1. **观察生活**：用心观察人物的一举一动，捕捉细微动作
2. **精选动词**：避免空洞笼统，选择最精准的动作词
3. **分解动作**：将大动作分解为多个小动作
4. **融入感受**：在动作中体现人物心理或作者情感
5. **善用修辞**：运用比喻、夸张等手法增强表现力
