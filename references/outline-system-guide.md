## 一、好大纲的标准

### 核心原则

| 原则 | 说明 |
|------|------|
| **事件驱动** | 大纲以事件为核心，而非以描写为主 |
| **字数≠支撑** | 字数多不代表大纲写得好，关键是事件清晰 |
| **一环扣一环** | 每章设计3个事件，事件之间逻辑衔接 |

### 事件设计要点

```
好事件 = 明确的主角行动 + 具体的后果 + 下一事件的触发点

❌ 不好：主角很生气
✅ 好：主角被嘲讽后愤而离开，却在转角撞见仇人
```

### 文风设定要求

| 体材类型 | 文风要求 |
|----------|----------|
| **豪门总裁** | 语言简洁有力，对话推进剧情，减少冗余描写 |
| **穿越古言** | 语言可略带古风，但不过分咬文嚼字，保持阅读流畅 |
| **青春校园** | 轻快活泼，对话自然，符合年轻人口语习惯 |

## 二、四种构思套路

### 四大类型对比

| 类型 | 定义 | 特点 | 难度 | 适合人群 |
|------|------|------|------|----------|
| **旧瓶装旧酒** | 跟风爆款 | 题材、套路全仿 | ⭐ | 新手入门 |
| **新瓶装旧酒** | 换马甲 | 新题材+老套路 | ⭐⭐ | 进阶写手 |
| **旧瓶装新酒** | 新故事 | 老题材+新创意 | ⭐⭐⭐ | 有经验者 |
| **新瓶装新酒** | 创新 | 新题材+新套路 | ⭐⭐⭐⭐ | 大神级 |

### 构思流程

```
由点到线再到面：

点（一个创意）→ 线（主线拉出）→ 面（分线展开成完整小说）

示例：
点：主角能听见植物说话
↓ 拉线
线：种田流，主角靠植物能力发家
↓ 展开
面：感情线（植物精灵）、势力线（灵植宗门）、复仇线（前世恩怨）
```

## 四、大纲格式与内容要求

### 每章三事件原则

```
第一章（约2000-3000字）

事件1：[开场冲突/危机]
- 主角遭遇的具体事件

事件2：[冲突升级/转折]
- 冲突如何发展

事件3：[留下钩子/悬念]
- 本章结尾的悬念
```

### 大纲内容完整清单

| 项目 | 内容要求 |
|------|----------|
| **书名** | 符合平台风格，30字以内，含核心看点 |
| **体材** | 明确分类（都市/玄幻/古言/悬疑等） |
| **主题** | 一句话概括核心立意 |
| **简介** | 200-500字，涵盖背景、冲突、悬念 |
| **环境设定** | 故事发生的主要场景及其规则 |
| **角色设定** | 主角、配角、反派的核心人设 |
| **技能设定** | 主角能力体系，金手指来源与限制 |
| **情节设定** | 主线、辅线、支线情节走向 |

## 五、大纲的作用与类型

### 大纲的五大作用

```
1. 梳理情节 - 避免前后矛盾，逻辑自洽
2. 梳理线索 - 埋下的伏笔记得回收
3. 梳理悬念 - 控制悬念的揭示节奏
4. 梳理人物关系 - 明确角色间的纠葛
5. 把握节奏 - 控制爽点与压抑点的分布
```

### 类型一：写给自己看的大纲

```
特点：灵活多变，用词明确，便于自己执行
```

### 类型二：写给编辑看的大纲

```
要求：3000字以内，结构清晰，标注作者名/作品名/联系方式
```

## 七、主辅线与伏笔设置

### 故事树法

```
        ┌─────────┐
        │  核心主线  │
        └────┬────┘
     ┌──────┼──────┐
     ▼      ▼      ▼
┌────────┐ ┌────────┐ ┌────────┐
│ 主线1  │ │ 主线2  │ │ 主线3  │
└───┬────┘ └───┬────┘ └───┬────┘
    │          │          │
    ▼          ▼          ▼
┌────────┐ ┌────────┐ ┌────────┐
│ 支线A  │ │ 支线B  │ │ 支线C  │
└────────┘ └────────┘ └────────┘
```

### 金手指设置原则

| 原则 | 说明 |
|------|------|
| **伴随转折** | 金手指在转折点出现，逻辑自洽 |
| **不要一步登天** | 能力要有限制和成长空间 |
| **克制使用** | 金手指要有代价，过度使用会削弱爽感 |
| **独特性** | 最好有独特的功能或限制 |

### 伏笔技巧

**核心原则**：危急时刻揭晓最爽

```
埋伏笔：早期（第1-30章）
↓
铺垫：中期（第31-60章）
↓
揭晓：高潮（第61章+）
```

## 九、情节设定技巧

### 核心：对比冲突是商业化写作的关键

```
冲突 = 对比 = 爽感

没有对比，就没有伤害
没有伤害，就没有爽点
没有爽点，就没有订阅
```

### 塑造强者的方式：先塑造参照物的强大

```
❌ 不好写法：
"丘处机是武林高手，很厉害。"
（读者没有概念）

✅ 好写法：
"丘处机一掌劈出，碗口粗的松树应声而断。
杨康看得目瞪口呆，心想：世上竟有如此人物！"
```

### 主角自身对比

| 类型 | 设定 | 爽点 |
|------|------|------|
| **扮猪吃虎** | 隐藏实力，扮作弱者 | 被轻视→爆发→震惊全场 |
| **转世重生** | 前世记忆，今生逆袭 | 知道未来→抢占先机 |
| **系统加持** | 获得外挂在身 | 别人苦练→我氪金 |

### 七步搞定一个情节设定

| 步骤 | 内容 | 问题 |
|------|------|------|
| **1. 地点环境** | 在哪里发生 | 什么场景适合这个情节？ |
| **2. 主要人物** | 谁在场 | 主角、配角、反派如何登场？ |
| **3. 故事梗概** | 发生了什么 | 这个情节展现什么？ |
| **4. 高潮爽点** | 核心看点 | 奇遇/升级/泡妞/打脸？ |
| **5. 渴望** | 人物想要什么 | 内心最深处的渴望？ |
| **6. 阻力** | 什么阻碍 | 内因？外因？反派？ |
| **7. 行动&冲突** | 如何行动/结果 | 克服阻碍的方法和代价？ |

### 分集情节大纲模板

```markdown
【第X集】

【主要内容】：一句话概括本集核心事件
【环境地点】：场景设定
【涉及人物】：主角、配角、反派
【故事开端/伏笔/悬念】：本集开场的引子
【故事发展/阻力】：冲突如何升级
【解决问题/高潮】：如何解决冲突
【故事结尾/新开端】：本集收尾方式
```

## 十、创作手法与技巧

### 创作流程

```
立意 → 角色设定 → 背景设定 → 开头结尾 → 填充情节
```

### 情绪控制五法

| 情绪 | 效果 | 适用场景 |
|------|------|----------|
| **喜悦** | 满足读者期待 | 爽点爆发、收获奖励 |
| **愤怒** | 激发读者共鸣 | 主角被欺负、打脸时刻 |
| **希望** | 维持阅读动力 | 埋下伏笔、展示潜力 |
| **恐惧** | 增加紧张感 | 危机来临、绝境时刻 |
| **悲伤** | 情感共鸣 | 角色死亡、牺牲情节 |

### 写作规范

| 规范 | 说明 |
|------|------|
| **不要第一人称** | 第三人称更利于展示全貌 |
| **打斗不超过30%** | 控制战斗描写的比重，保持节奏 |
| **对话形式多样化** | 纯对话、动作+对话、旁白+对话结合使用 |

## 十一、创作心得与忠告

### 大纲的本质

```
大纲 = 大树主干
情节 = 肌肉
细节 = 皮肤

没有主干，肌肉再多也是一滩烂肉
```

### 主线控制

> **撒出去收不回来会太监**

**应对策略**：
- 每埋一条线，想好回收点
- 主线之外最多保留2-3条重要辅线
- 写不下去时，宁可砍线不要硬圆

### 修改原则

> **修补不如弥补，没有更好替代前不要改**
