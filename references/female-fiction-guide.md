## 核心精华

### 文案四句话公式

#### 真香流文案

```
第一句：男主对女主不满/不屑/替身
第二句：转折，男主真香了
第三句：再转折，女主其实不在乎
第四句：男主崩溃+路人补刀
```

**示例**：
```
霸道总裁宠却不爱，因为碧水不过是个替身。
直到一天，晋江发现碧水是为了体验角色入戏罢了。
晋江眼泪流下来：求复合！
碧水：大哥你谁？
```

#### 苏爽流文案

```
第一句：主角状态+变故（穿越/重生）
第二句：新状态下面临困境
第三句：金手指逆袭+三个打脸例子
第四句：与CP产生互动（可有可无）
```

### 大纲法汇总

| 方法 | 适用场景 |
|------|----------|
| 百叶窗大纲 | 感情流，层层剥开 |
| 叠纸大纲法 | 狗血感情流 |
| 送信大纲法 | 情感递进 |
| S型曲线大纲 | 紧凑剧情 |
| 逆推大纲法 | 从结局倒推 |

### 感情流核心技巧

**甜文怎么写才不尬**：
1. 暧昧要自然递进
2. 情感主动方的刻画要细腻
3. 留钩子让读者想看后续

**虐点盘点**：
- 误解
- 错过
- 身份差距
- 外部阻碍
- 内心挣扎

### 爽感设计

**打脸套路**：
```
炮灰嘲讽 → 主角打脸 → 他人反应
```

**高潮怎么写才够爽**：
1. 铺垫足够深
2. 爆发足够强
3. 层次要分明

### 人设塑造

**做CP比做人设更重要**：
- 研究CP互动模式
- 正面属性+负面属性结合
- 用细节刻画代替说明

**人设常见问题**：
- 人设和剧情脱节
- 人设前后矛盾
- 配角没有记忆点

### 节奏把控

**起承转合结构**：
```
起：建立世界/引入冲突
承：发展冲突/积累情绪
转：关键转折/高潮爆发
合：收束/新期待建立
```

**营造紧张感**：
1. 时间限制
2. 信息差
3. 悬念叠加

## 军婚宠文创作指南

> 整合自：用户上传/1.军婚宠文创作指导(1)_xxx.doc

### 人设核心：强强联合

#### 女主设定
- **外貌类型**：御姐风范，长相或可爱或妖娆均可
- **核心特质**：心性坚强，有做女强人的信念
- **能力要求**：在各领域发光发热，有吸引各男主男配的闪光点

#### 男主设定
- **身份**：军人身份，军衔职务很高（特种大队大队长/中队长/中将/少将）
- **地位**：号令军政商三界
- **性格矛盾点**（至少具备其一）：
  - 表面温润实则残酷
  - 表面风流实则纯情
  - 表面冷酷实则温柔（温柔只针对女主）

#### 女配设定
- 偏好从一而终，男主曾经无爱恋对象
- 女配多为倒贴型
- **必备**：1-2个贯穿全文、坏到极致的女配，用于离间男女主感情、制造坎坷

#### 男配设定
- 不一定为军人身份，**最好不为军人身份**
- 与女主有较多接触机会（如女主从商，男配在商界）
- 本领设定不能弱势，与男主不相上下
- 是男主最重要的情敌，贯穿全文
- 结局可以是安排好姻缘或默默陪伴女主

### 家世设定

#### 男主家世
- **军政世家，红N代**，国内横着走
- **方向一**：与家里关系好，家里喜欢女主，不构成阻碍，感情受挫时家人会撮合
- **方向二**：与家人关系不好，家里不喜女主，男主宁愿脱离家族也维护女主

#### 女主家世（三种可能）
| 类型 | 描述 | 基调 |
|------|------|------|
| 军政联姻型 | 女主家庭也是军政家庭，履行婚约，男主陪女主打天下 | 温馨 |
| 没落逆袭型 | 女主家庭没落受尽欺辱，女主变强使家族崛起 | 励志女强 |
| 弃子认亲型 | 女主为军政家族抛弃的子嗣，逐步变强后认祖归宗 | 励志女强 |

#### 能力设定
- **男主**：军政商无所不能
- **女主**：商界/军界/娱乐圈/政界某一领域逐步提升（驾驭不了则选其一）

### 情节设定

#### 主线设计
> **两条主线不可分割，要衔接连贯贯穿全文**
1. 男女主角的情感发展
2. 女主逐步变强

#### 开篇要求
- **节奏要快**，不能拖沓慢热
- 没必要的人物情节全部去除
- 必要的背景几句话带过
- **切忌**：太多叙述性内容
- **最好**：用事件直接阐述情感
- **开篇稍带暧昧**，整体尺度不要过大
- **男主开篇前几章就能出场**
- **开篇3万字**以男女主角情感碰撞为主
- 需要代入相关支线伏笔

### 写作技巧

#### 标题文案
- 书名简介章节名迎合市场需求
- 稍带暧昧，能够吸引点击
- 即使标题党，作品也需要让人有阅读的欲望

#### 章节字数
| 阶段 | 字数要求 |
|------|----------|
| 1-20章 | 1200字+ |
| 21章开始 | 3200字+ |

#### 卡点技巧
- 每个章节卡点精准
- 能让人有继续阅读下一章的欲望
- **尤其注意第20章**：无线21章开始计费，若20章读者难以沉迷，21章不可能付费

#### 文风要求
- **不要过于注重华丽辞藻**
- 无线文通俗易懂即可
- **情节比任何都重要**

#### 学习方法
去无线榜单上关于这类型的文**全部浏览阅读一遍**，掌握：
- 此类文的套路
- 基本规律
- 重点学习以上所说的几点

## 古代女强创作指南

> 整合自：用户上传/2.古代女强创作指南_xxx.doc、3.古代女强文设定大纲_xxx.doc

### 类型定位
**主流类型**：弃妇 + 宅斗 + 金手指

### 人设体系

#### 穿越女强
- 女主被渣男休弃，受尽屈辱致死
- 现代女主魂穿后改变屈辱状态，大放异彩
- 渣男回头，男主倾慕

**创新方向**（现在套路已泛滥，需添加新元素）：
| 元素 | 说明 |
|------|------|
| 原身残缺 | 花痴/毁容/面相丑陋，加大羞辱力度，恢复后创造爽感 |
| 特殊技能 | 擅医术/会蛊毒/美食天分/乐理天分等，挖掘支线 |

#### 重生女强
- 女主曾为渣男原配，遭遇背叛，受尽屈辱死去
- 重生至年轻时候，一切悲剧未发生
- 重生后改变历史悲剧，挽回曾经失去的一切

#### 男主性格矛盾（至少具备其一）
- 表面温润实则残忍
- 表面多情实则纯情
- 表面邪魅实则冷酷
- **身有残缺**：腿残/身残/痴傻等，但实质实力强大、只手遮天

#### 女配进阶体系
- 斗的戏码逐步提升：恶奴 → 姨娘 → 庶妹 → 家族 → 男主家族（多为皇宫，涉及皇位纷争/后宫纷争）
- **建议**：主要女配贯穿始终，其余女配与主要女配有细枝末节关系
- 女主在斗的过程中发现女配不能轻易被打倒，需足够强大才能打倒她

### 渣男/男配设定

#### 渣男类型
| 类型 | 特点 |
|------|------|
| 回头是岸型 | 侮辱女主认为配不上，却爱上变强后的女主，成为正面人物 |
| 执迷不悟型 | 爱而不得生恨，设定各种陷害障碍，是情感危机主要破坏者 |

#### 正面男配
- 实力与男主相当
- 男主主要情敌
- 默默守护女主的作用

### 家族体系

#### 守护型
- 女主家族没落，遭受白眼冷遇
- 主要家庭成员对女主很保护
- 女主穿越或重生后以守护家族为己任
- 自立自强，稳住家族地位甚至使家族崛起

#### 破坏型
- 女主家族强大，却不被重视
- 受尽家庭主要成员虐待羞辱，多次陷害暗杀
- 女主不得不奋起反抗
- **注意**：推倒家族后该文基本没有重要爽点，阴谋主线要足够强大能维持长文

### 开篇技巧

#### 矛盾事件切入
- 女主原身被休弃/被人羞辱/不堪屈辱自尽
- 女主穿越后改变被人欺辱的状况
- **反虐回去**，达到第一个爽点

#### 男主出场要求
- **不能太慢**
- 榜单上很多文男主出场慢、多为侧面出场
- **无线读者更倾向于前3万字集中男女主角戏码**
- 最好男主**直接出场**，与女主有交锋接触

#### 暧昧情节
- 很多同类型文没有暧昧或暧昧很少
- **无线读者喜好暧昧戏份**
- 开篇最好能引入男女主角的暧昧戏份

### 情节设计要点

#### 主线设计
> **两条主线不可分割，衔接连贯贯穿全文**
1. 男女主角的情感发展
2. 女主逐步变强

在宅斗宫斗过程中，也要加强女主各方面才能：
- 商业上能赚一大笔财富
- 才华方面惊才绝艳

#### 开篇要求
- 节奏快，不拖沓慢热
- 没必要的人物情节去除
- 背景几句话带过
- 用事件阐述情感
- 开篇稍带暧昧
- 尺度不要过大
- 男主前几章出场
- 3万字以情感碰撞为主
- 代入支线伏笔

### 写作技巧

#### 标题文案
- 迎合市场需求
- 稍带暧昧
- 吸引点击

#### 章节字数
| 阶段 | 字数要求 |

#### 情节重点
- 不要过于注重华丽辞藻

#### 学习方法

### 案例分析：《神医世子妃》

#### 人设
| 角色 | 设定 |
|------|------|
| 女主 | 唐门传人，擅使毒会医术，技能是亮点 |
| 男主 | 夙世子，与女主从冤家到相爱，身世后期有支线伏笔 |
| 女配 | 女主嫡姐，白莲花般，擅长伪装，众星捧月但心机颇深 |
| 男配 | 靖王爷（渣男反派）+ 其他出色人物 |

#### 家族
- **推倒型**：女主受尽羞辱，推倒家族为第一爽点

#### 开篇
- 女主被男主休，大婚之日大门紧闭，一纸休书让原轿返回
- 矛盾集中，女主魂穿后面临家庭成员羞辱
- 绝地反击，逐步变强

#### 不足
- 开篇人物过多，支线复杂
- 反虐主线不够集中
