```
关键词：坚强、隐忍、倔强、不屈
描写要点：眼神要有力量，遭遇困境时不轻易流泪，
          咬牙也要撑过去，有韧性
雷区：不要写成铁人，偶尔的脆弱更动人

【腹黑型】
关键词：精明、算计、笑面虎、绵里藏针
描写要点：表面温温柔柔，说话软软糯糯，
          实则心思深沉，算计精准
雷区：不要写成恶毒女配，要有底线和原则

【飒爽型】
关键词：英气、豪爽、武力值、不矫情
描写要点：说话干脆利落，走路带风，
          打架厉害，但也有柔软的一面
雷区：不要写成男人婆，女性魅力要保留

【御姐型】
关键词：成熟、包容、强势、温柔
描写要点：照顾人有一套，气场强大但不凌厉，
          能屈能伸，懂得进退
雷区：不要写成老妈子，要有自己的魅力和骄傲
```

## 六、肖像描写技巧

### 肖像描写四大原则

```
【原则一】精选特征，删繁就简
❌ 错误：她有一头乌黑亮丽的长头发，弯弯的眉毛，
        大大的眼睛，高高的鼻梁，红红的嘴唇……
✅ 正确：她最引人注目的是那双眼睛，明亮得像会说话

【原则二】特征服务于性格
❌ 错误：她是个温柔的女子，有着高挺的鼻梁
✅ 正确：她眉眼含笑，让人一看就觉得亲切

【原则三】动静结合
❌ 错误：她静静地站在那里
✅ 正确：她静静地站在那里，一阵风吹过，
        撩起她耳边的碎发

【原则四】善用比喻
❌ 错误：她的眼睛很亮
✅ 正确：她的眼睛亮得像是盛了两颗星星
```

### 五官描写技巧

```
【进阶技巧】

   她的眼睛亮了——惊喜
```

### 整体描写技巧

```
【由上到下法】
她有着一头如瀑的长发，眉如远山，一双杏眼清澈动人。
琼鼻挺翘，樱桃小嘴，尖尖的下巴。
一袭白裙，身姿婀娜。

【由下到上法】
她穿着一双绣花鞋，露出纤细的脚踝。
身形修长，腰肢不盈一握。
五官精致得像是上天的宠儿，尤其是那双眼睛，仿佛会说话。

【重点突出法】
她最引人注目的是那双眼睛，明亮得仿佛盛满了星辰。
其次是她高挑的身材，站在人群中鹤立鸡群。
至于五官，反而显得不那么重要了。

【动静结合法】
她静静地站在那里，长发随风飘动，衣袂翩翩。
一颦一笑间，尽显绝世风华。
这画面美得让人窒息。

【对比衬托法】
明明是在喧闹的市井，她却像误入凡间的仙子。
周围的一切在她面前都黯然失色。
她就是那种让人一眼万年、再难忘却的存在。
```
