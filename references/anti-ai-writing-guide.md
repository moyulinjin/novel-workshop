# 去AI味写作指南

> **来源**：The AntiSlop + writing-agent 核心技巧整合
> 
> **目标**：让AI生成的内容更像人类写作

---

## 一、为什么需要"去AI味"

### AI生成的典型特征

**词汇层面**：
```
❌ delve（深入探讨）
❌ game-changer（游戏规则改变者）
❌ revolutionary（革命性的）
❌ leverage（利用）
❌ unlock potential（释放潜力）
❌ cutting-edge（前沿的）
❌ moreover/furthermore（此外/再者）
❌ it's worth noting（值得注意的是）
❌ in today's digital landscape（在当今数字时代）
```

**句式层面**：
```
❌ 配对形容词："comprehensive and thorough"（全面而彻底）
❌ 模板开头："In this post, we'll cover..."（在这篇文章中，我们将讨论...）
❌ 短句堆砌：每个句子都是独立的短句
❌ 句式单一：每句都是10-15词
❌ 比较句型："This isn't X. It's Y."（这不是X，这是Y）
```

## 二、分层检测系统

### Tier 1：立即删除（扣3分/次）

```
必须立即替换的词汇：

| AI词汇 | 替换建议 |
|-------|---------|
| delve | 探索、研究、分析、走进 |
| game-changer | 颠覆、革新、改变局面的 |
| revolutionary | 全新的、开创性的、突破性的 |
| leverage | 借助、利用、凭借 |
| unlock potential | 释放/激发/展现潜力 |
| cutting-edge | 最先进的、领先的、全新的 |
| moreover | 而且、此外、同时 |
| furthermore | 除此之外、与此同时 |
| it's worth noting | 需要注意的是、有一点值得注意 |
| in today's landscape | 当下、如今、在这个时代 |
```

### Tier 2：重复可疑（扣2分/次）

```
同一个场景中不要重复出现：

| 词汇/短语 | 问题 |
|----------|------|
| here's the thing | 显得刻意 |
| at the end of the day | 口语但不自然 |
| the bottom line | 太正式 |
| firstly/secondly/thirdly | 机械罗列 |
| in conclusion | 模板感太强 |
| to sum up | 总结腔 |
| on the other hand | 过于书面 |
```

## 三、结构性AI特征

### 1. Staccato Fragments（短句堆砌）

**AI典型**：
```
门口站着一个男人。
他很高。
穿着黑色风衣。
眼神冰冷。
手中握着一把刀。
```

**人类写法**：
```
门口站着一个男人。
他很高，穿着一件不合身的黑色风衣，眼神冰冷得像淬过毒，手里还攥着把刀——刀刃上有暗红色的痕迹。
```

**修改技巧**：
```
将3-4个连续短句合并成1-2个长句
使用破折号、逗号、分号连接
加入细节描写打破节奏
```

---

### 2. 句式均匀化

**AI典型**：
```
他推开门。（5字）
房间里很暗。（5字）
他看见了一个人。（6字）
那是他的朋友。（5字）
```

**人类写法**：
```
他推开门。
房间里黑得伸手不见五指，只有窗户缝隙透进来的一点光。
他眯起眼睛，隐约看见墙角蜷缩着一个人影。
"老王？"
那是他朋友。

（长、短、长、短、长——有节奏变化）
```

---

### 3. Comparator Sentences（比较句型）

**AI典型**：
```
这不是一个普通的故事。
这是一个关于复仇的故事。
```

**人类写法**：
```
这只是一个普通的复仇故事。
——如果你这么认为，那就错了。
```

## 四、Horoscope Test（星座测试）

### 核心原则

> **"任何人能写给任何人读吗？"**
> 
> 如果是，就是AI味。

### 修改示例

```
❌ AI味：
"在这个快速发展的时代，我们需要不断学习才能不被淘汰。"

✅ 人类味：
"老张四十了，在厂里干了二十年，上个月被通知'优化'。他说想学新东西，但招聘要求写的都是'35岁以下'。"
```

## 五、实操改写技巧

### 技巧1：加入噪音

```
AI总是追求"正确"和"完整"
人类写作会有"不完美"

❌ 完美但假：
"他走进房间，仔细观察了环境，发现桌上有一封信。"

✅ 有缺陷但真：
"他推门进去，没开灯。月光下，桌上有什么白色的东西。走近才看清——是一封信，信封上没写字。"
```

### 技巧2：打破模板

```
AI喜欢用固定模式：
- 首先...其次...最后...
- 一方面...另一方面...
- 总而言之...

人类说话更随意：
- 突然想起来一件事...
- 对了，差点忘了说...
- 不过话又说回来...
- 说起来也怪...
```

### 技巧3：使用口语化

```
❌ 书面语AI：
"他对此表示深切的关注，并认为应当采取相应措施。"

✅ 口语化人类：
"他听了这事儿，半天没说话。
'操。'他最后只蹦出这一个字。"
```

## 六、改写工作流

### 发现问题

```
使用以下检测清单：

□ 是否有Tier 1词汇？（delve, moreover...）
□ 是否有重复的过渡词？
□ 是否短句过多或句式单一？
□ 是否像星座运势？
□ 是否有"这不是X，是Y"的句型？
□ 是否有模板开头/结尾？
□ 段落长度是否过于均匀？

得分：
0-5分：低风险
6-12分：中风险（需要显著修改）
13+分：高风险（可能是未编辑的AI生成）
```

### 改写流程

```
Step 1：清理Tier 1词汇
→ 直接替换或删除

Step 2：处理Tier 2重复
→ 同义替换或删除

Step 3：打破结构模式
→ 合并短句
→ 变化句式
→ 添加留白

Step 4：通过Horoscope测试
→ 问："任何人能写这个吗？"
→ 如果是，增加具体细节

Step 5：加入你的风格
→ 加入个人表达习惯
→ 增加"噪音"和"不完美"
```

## 七、进阶技巧

### 1. 制造"不完美"

```
AI追求完美：
"他走进房间，坐在椅子上，开始工作。"

人类会有各种"状况"：
"他一脚踢开虚掩的门，踩着什么滑了一下。
骂了句脏话，才看清是地上有滩水。
没管，先把空调开了。
然后才想起来——椅子呢？
哦，被隔壁老王搬走了。"
```

### 2. 使用"脏话"和语气词

```
适当使用增加真实感：
"他妈的"
"操"
"我靠"
"妈的"
"去他妈的"

语气词：
"得吧"
"嗯"
"啊"
"嘛"
"呢"

注意：适度使用，不要过度
```
