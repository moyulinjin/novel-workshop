## 一、对白技巧

### 1.1 对话流定义

对话流是指以大量对话来推动情节和故事发展，从文章篇幅来看，超过百分之六十的对话就能称之为对话流。

### 1.2 对话形式的三种基本模式

```
① "……"甲对乙说。
② 甲对乙说："……"
③ "……"甲对乙说，"……"
```

### 1.3 对话流的常见错误

#### 错误一：人物身份不明确

**问题**：三人以上对话时，没有人物身份指示，读者无法区分说话人。

**对比示例**：
```markdown
❌ 错误示范：
"吃饱了？"
"吃饱了！"
"我们一起上学吧"
"好的。"

✅ 正确示范A：
小明对小红说："吃饱了？"
小红拍拍肚子："吃饱了！"
小明点头："我也吃饱了，我们一起上学吧？"
小红点头："好的。"

✅ 正确示范B：
"小红，你吃饱了吗？"小明问。
"吃饱了，小明你呢？"小红拍拍肚子。
"吃饱了，我们一起上学吧。"小明拉住小红的手。
"好的。"
```

#### 错误二：标点格式出错

表情动作后的对话没有加冒号等标点问题。这是新手作者常见的小毛病，建议加强标点符号使用规范。

#### 错误三：引号开头的段落过多

引号开头的段落太多会导致读者阅读时有不舒适感。建议多种对话形式交替使用。

### 1.4 对话形式多样化技巧

**第一**：灵活运用上述三种对话形式变化。

**第二**：在对话中穿插表情、小动作、场景、细节描写，增强人物心理活动。

**第三**：控制对话数量，不要完全以对话推动情节发展。

**第四**：对话的细节描写是塑造人物形象和适度"灌水"的最好时机。

## 二、人物塑造

### 2.1 主角设定的两大必要条件

#### 条件一：移情作用
主角必须让读者产生认同感，感觉人物和"我"或身边的人很像，这样读者才会将感情"转移"到主角身上。

**移情作用刻画技巧**：
- 刻画人物的某种缺点
- 强调人物的某种习惯或嗜好

#### 条件二：强烈的欲望
主角的"欲望"直接决定故事主线走向：
- 励志类：欲望是达到目标、实现梦想
- 言情类：欲望是和所爱的人在一起
- 潜在欲望：表面欲望与真实欲望不同（如表面想扳倒对方，实际是相爱）

### 2.2 人物设定的三个维度

#### 总体设定

**背景设定**（三个大类）：
1. 心灵创伤型
2. 温情关怀型
3. 寻求存在感

**现实生活设定**（三种状态）：
- 职业生活：与主线情节直接交叉
- 个人生活：与朋友、家人的关系
- 私生活：内心独白，最体现深层个性

#### 性格设定

**冲突原则**：人物应拥有和内心相反的表面性格。
- 例如：内心寂寞沉静 → 表面阳光开朗 → 独处时痛哭才有张力
- 注意：校园文主角可不用矛盾性格；偏情感故事则必须要有

**男主角塑造**：更强调性格本身的冲突，双重个性体现神秘感
- 悲惨过去 → 开朗活泼外表
- 冷漠不近人情 → 内心细腻敏感

**性格设定技巧**：
- 突出角色明显缺点（增强亲和力）
- 设定口头禅、特殊爱好、习惯动作（但不宜过多）

#### 目标设定

**欲望的两种类型**：
1. 达成具体事件的欲望（事件贯穿，易写但情感线容易脱离）
2. 情感上的欲望（改变、寻找依靠、不再孤独）

### 2.3 人物描写的五种方法

#### 肖像描写
用精确、生动、形象的语言描绘人物外形特点（容貌、服饰、姿态、神情、风度等）。关键在于以形传神、形神兼备。

**示例**（《老人与海》）：
> 老人消瘦憔悴，脖颈上有些很深的皱纹。腮帮上有些褐斑...他身上的一切都显得古老，除了那双眼睛，它们像海水一般蓝，显得喜洋洋而不服输。

#### 动作描写
描写人物在特定情境下的动作（举手投足、跑跳坐卧），要抓住表现人物个性的富有特征性的动作。

**示例**（《水浒》武松）：
- 到酒店三碗又三碗地喝酒
- 看告示
- 上岗
- 打老虎

#### 语言描写
通过对话表现人物内心世界，突出性格特征。不同语言风格体现不同人物身份和性格。

#### 心理描写
揭示人物在特定环境中的内心活动（感觉、知觉、情感、体验、愿望等）。

**示例**（《阿Q正传》）：
> 阿Q站了一刻，心里想，"我总算被儿子打了，现在的世界真不像样……"于是心满意足的得胜的走了。

#### 环境描写
交代人物活动的物质空间，烘托人物心理，渲染气氛。包括自然环境和社会环境。

### 2.4 人物塑造的四大忌讳

1. **忌变**：性格变化太过厉害容易引起读者反感（由坏变好除外）
2. **忌乱**：性格混乱，正人君子vs对待女人变猥琐
3. **忌纯**：纯恶纯善的主角（前者黑暗，后者不真实）
4. **忌没有聚焦点**：没有让读者一下记住的特征

**聚焦点的力量**：
- 李寻欢 → 修长洁白的手，用小刀刻木头
- 楚留香 → 揉鼻子
- 张无忌 → 懦弱无能
- 韦小宝 → 好色有智
- 洪七公 → 九个手指头 + 贪吃好酒

### 2.5 配角塑造要点

- 主角光环太强难塑造 → 配角反而容易出彩
- 反派要有深度：大忠大义的反派比纯邪恶反派更打动人
- 反派也要有情感逻辑：为何敌视主角？是否值得同情？

## 三、文笔修炼

### 3.1 文采的六个表现

1. **整齐美**：句式整齐匀称，多用整句
2. **错落美**：灵活变换句子长短和结构
3. **丰富美**：词汇丰富多变，绝无呆板雷同
4. **韵律美**：琅琅上口，有音乐节奏
5. **意象美**：融合主观情感的客观物象
6. **意境美**：主观感情与客观环境有机融合

### 3.2 网络文学文笔的核心定义

> 文笔，就是作者能够让读者用最方便最快捷、最容易理解的方式，让读者能够心情愉悦地读出作者想要给读者看的东西。

**简单概括**：
- 读者看起来很爽 = 文笔好
- 这个"爽"不是情节的爽，而是**文字整洁、逻辑顺畅、一目了然**

### 3.3 网络文学文笔的基础要求

#### 第一要素：基础扎实

1. **错字率**：控制在万分之五以下
2. **段落长度**：不超过300字（最好五行以内）
3. **句子长度**：一个句号前不超过5个逗号
4. **标点使用**：规范准确

#### 第二要素：流畅通顺

写完一段后，**默读三遍**，顺了没错字再继续。

#### 第三要素：考虑读者感受

- 人类有**密集物恐惧症**
- 看电脑屏幕超过3行会微疲劳，超过五行会用扫的扫过去
- 三行左右最符合生理和心理

### 3.4 细腻文笔的四个修炼方向

#### 一、从容细赏的心态

- 用心感受，不只是用眼睛看
- 多感官调动：视觉、嗅觉、听觉、触觉
- 例：写衣橱开合的瞬间感受

#### 二、捕捉瞬间思绪

- 对心头掠过的思绪或直觉进行捕捉和放大
- 文笔细腻实为心灵的细腻
- 瞬间内心闪现外化为永恒文字

#### 三、开掘精神意蕴

- 不只聚焦于"形"，而是开掘内在的"神"
- 赋之以生命、品格、精神
- 细腻 = 传神的笔法

#### 四、敏锐善感的心灵

- 敏感 → 领略丰富
- 善感 → 体味深细
- 内心保留一点孤独寂寞

## 四、语境运用

### 4.1 语境的定义

语境就是运用语言的具体环境，包括：
- **狭义语境**：使用语言的时间、地点、场合、对象、上下文
- **广义语境**：使用语言的时代、社会特点、使用者职业/性格/习惯等

### 4.2 语境对语言运用的两大作用

#### 限制作用

**表现一：对词语的理解和选用**
- 同一个词语在不同语境中表达不同意思
- 例："骄傲"——在"英雄"语境下=自豪；在"谦虚vs骄傲"语境下=自以为是

**表现二：对句子的理解和组织**
- 例："一份'北京'，一份'中国'"——在报摊前=两份电视报
- 主动句vs被动句的选择也由语境决定

**表现三：对段落和整篇的安排**
- 时代背景、作者简况影响理解

#### 补充作用

**表现一：对深层含义的理解**
- 例："今天可真冷啊"——在开着窗的冬天家里 = 暗示关窗

**表现二：对言外之意的理解**
- 例："还要让它'留芳'百世吗？"——实际是反语讽刺

## 五、情节设计

### 5.1 小说情节的画面感

#### 什么是画面感？

如同看人"第一眼"的感觉——顺不顺眼。只有感觉顺眼了，读者才会继续从细节看。

#### 破坏画面感的四个问题

1. **段落太长**：2000字一章只有三四个段落
2. **句子太长**：超过二十甚至三四十个字
3. **错字连篇**：前言不搭后语，形容词堆砌
4. **对话单调**：全部引号开头，一点变化都没有

#### 画面感的基本要求

| 要素 | 要求 |
|------|------|
| 单句 | 没有标点的句子不要太长 |
| 复句 | 以句号/感叹号/冒号/问号结尾 |
| 段落 | 超过300字即为长段 |
| 形式 | 描述方式多变，不要太单一 |
| 开头 | 不用每段开头都用主角名字 |

### 5.2 悬念构成的条件

1. 人物命运中潜伏着危机
2. 生与死、成功与失败均有可能
3. 发生势均力敌而又必须有结果的冲突
4. 观众对人物性格、行动产生爱憎
5. 观众对未来发展有期待

### 5.3 悬念的两大类型

#### 期望式悬念
建立在观众不保密的基础上，观众有预感但仍期待。

#### 突发式悬念
依靠对观众保密，出乎意料之外又在情理之中。

### 5.4 延宕手法

在尖锐冲突中，利用副线情节或穿插场面暂时缓和，实际上更加强化紧张性。

**节奏控制要点**：
- 书不能老是高潮迭起 → 读者快感阈值提高
- 正确节奏：吊胃口→喂饱→消化→再吊胃口
- 如同打耳光：前几下最痛，停下来抹药水再打更痛

### 5.5 突转与发现

- **突转**（陡转/突变）：向相反方面的突然变化，逆境↔顺境
- **发现**：从不知到知的转变

两者常联用，通过突转写出人物剧烈心理变化。

### 5.6 时空交错手法

根据梦境、幻觉、遐想、回忆等心理活动组织时空，把过去、现在、未来相互穿插交织。

### 5.7 巧合的运用

**无巧不成书**，但巧必须：
- 合理
- 自然
- 有意义

## 六、爽点创作

### 6.1 爽点的本质

> 网络小说的真谛：让读者爽，只有让读者爽了，读者才会投票支持、花钱订阅。

读者群体特征：
- 现实中太多顾忌和束缚
- 迫切需要在虚拟世界中得到满足和发泄
- 主角必须说出读者想说却不敢说的话
- 主角必须做出读者想做又不敢做的事
- 主角必须获得读者根本无法企及的成就

### 6.2 五大爽点类型

#### 第一种：情感爽点（先抑后扬）

**核心手法**：先压抑，再给予超出预期的小奖励

**示例**：冰山美人/小龙女型角色
1. 前期铺垫让读者认为主角根本不可能追到
2. 突然轻轻一个吻
3. 比大量性描写更让读者疯狂

**要点**：压抑不能太厉害，否则读者会完全失去兴趣

#### 第二种：物品获得（强化存在感）

**核心手法**：让读者觉得这个东西必须得到

**三步法**：
1. 告诉读者这个东西多好多好
2. 说明没有这个东西剧情接不下去
3. 说明获得这个东西极其困难

**适用作品**：《仙葫》《阳神》《永生》

#### 第三种：养成爽点

**核心手法**：主角在逆境中不断成长

**禁忌**：
- 绝不能一开始就让主角天下无敌
- 不能每次升级都写成高潮
- 主角升级只是突破瓶颈时小爆发

**反例**：
> 主角引起入体，进入练气期！主角好不容易筑基了！！主角历经万苦炼成金丹了！！！主角修成元婴！！！！

（每次都是高潮 = 没有高潮）

#### 第四种：优越感

**核心手法**：让主角给读者优越感

**示例**：
- 开劳斯莱斯、叼雪茄、拿美钞拍欺负过你的上司脸
- 穿200块衣服买房子，被鄙视后掏出一袋现金

**要点**：先抑后扬，被鄙视后反转

#### 第五种：送菜流

**核心手法**：制造一个比主角更牛的"送菜"角色

**送菜角色特征**：
- 比主角牛逼
- 运气更好
- 实力更强
- 老有奇遇

**结果**：他的好运→主角好运，他的宝物→主角宝物，他的女人→主角女人

**适用作家**：梦入神机

### 6.3 爽的源泉分析

#### 奖励要让人爽的关键是"需要"

**对比**：
- 穷学生突然中1000万 → 爽但一般
- 打工攒5万差5万，中彩票刚好凑够 → 超爽

**关键**：奖励必须是主角**迫切需要的**

#### 三种人为制造"需要"的方法

1. **缺少的东西**：缺少A被大Boss完虐，完成任务得到A才能力挽狂澜
2. **解除压力**：中了诅咒必须听命于人，完成任务解除诅咒+获得更好发展
3. **大家都想要的**：宝物虽价值10但只有主角知道 vs 宝物价值1但众人争抢

### 6.4 爽的本位：代入主角

#### 主角要有特色

让读者记住主角而非泯然于众。

#### 主角效用：一锤定音

- 第一种：实力坚挺，总能保持重要地位
- 第二种：平时不出力，关键时刻扭转乾坤

**第二种更耀眼**。

#### 主角不能有弱点

- **不擅长** = 能力上的无可奈何（可以接受）
- **弱点** = 心理上的漏洞（主角不能有）

主角只能输给比自己强的人，不能输给比自己弱的人。

## 七、书评应对

### 7.1 正确对待书评的态度

**核心原则**：有用的拿来用，没用的不要管，骂我的不在意

### 7.2 书评的作用

- 指出作品问题
- 促进创作提升
- 书评区静悄悄反而要反思

### 7.3 处理书评的具体建议

#### 针对"死不改"的作者
- 只改**基础性错误**和**开头三万字**
- 后面爱改不改

#### 针对"拼命改"的作者
- 改完基础和三万字后，哪怕大神叫你改也不改
- **大部分作者太监，死于不断修改导致失去灵气**

### 7.4 写评论的方法

- 看别人的小说要写书评和评论
- 只有写了评论才能真正剖析一部作品
- 互相给别的作者写评论，从基础、情节、人物、布局等角度评

### 7.5 作者心态建设

> 作者是作品的上帝，作品里的世界应由作者来主宰。

- 读者可以指手画脚、嬉笑谩骂、夸奖
- 但最后的主宰权还是作者的
- 这种超然心态需要资本：坚实的基础 + 足够的实力 + 强大的自信

## 八、审核规避

### 8.1 淫秽及色情出版物的认定标准

**淫秽出版物**（整体上宣扬淫秽行为）：
1. 淫亵性描写性行为、性交及其心理感受
2. 公然宣扬淫秽色情形象
3. 淫亵性描述或传授性技巧
4. 具体描写性犯罪手段、过程、细节
5. 具体描写少年儿童的性行为
6. 淫亵性描写同性恋性行为或性变态
7. 其他令普通人不能容忍的性行为淫亵性描写

**色情出版物**：
- 整体上不是淫秽，但部分有上述内容
- 对普通人特别是未成年人有毒害
- 缺乏艺术或科学价值

### 8.2 不属于淫秽色情的范围

1. 夹杂淫秽色情内容但**具有艺术价值**的文艺作品
2. 表现**人体美**的美术作品
3. **自然科学和社会科学**作品（解剖生理知识、生育知识、疾病防治、性知识、性道德、性社会学等）

### 8.3 网络低俗内容十三项整治重点

1. 表现或隐晦表现性行为、令人产生性联想、具有挑逗性或污辱性的内容
2. 对人体性部位的直接暴露和描写
3. 对性行为、性过程、性方式的描述或带有性暗示、性挑逗的语言
4. 对性部位描述、暴露，或只用很小遮盖物的内容
5. 全身或隐私部位未着衣物，仅用肢体掩盖隐私部位的内容
6. 带有侵犯个人隐私性质的走光、偷拍、漏点等内容
7. 以挑逗性标题吸引点击的
8. 禁止传播的色情、低俗小说，音视频内容
9. 一夜情、色情、SM等不正当交友信息
10. 情色动漫
11. 宣扬血腥暴力、恶意谩骂、侮辱他人等内容

13. 未经允许或利用"人肉搜索"恶意传播他人隐私信息

## 九、上架策略

### 9.1 前三万字决定成败

- 前5000字、前1万字、前3万字都要有强吸引力
- **不要在正文前加作者语、感言、请假、作者介绍**
- 与正文无关的东西会破坏读者阅读心情

### 9.2 开头的要求

1. 没有乱七八糟的无关内容
2. 主角人物塑造到位
3. 没有多余配角抢戏

### 9.3 选择上架点

**核心原则**：掐高潮点

- 评书技巧：**"预知后事如何，且听下回分解"** 前一定是最紧张刺激的情节
- 抓住读者**好奇和求知八卦心理**

**错误做法**：
- 脱光衣服后捂住三点，拿钱才给看
- 让人看了快爽的时候掐住要害，说"加入VIP"

**正确做法**：
- 在免费章节末尾布置高潮
- 读者为了看完高潮情节而付费

### 9.4 爆发的力量

**存稿是终极大杀器**：
- 大神上架后三五万字一起爆发
- 有存稿才能让读者支持更给力

**爆发策略**：
- 隔段时间爆发一次比每天稳定更新更有吸引力
- 这是一种促销心理

### 9.5 节奏控制

- **文喜看山不喜平**
- 不能一直高潮 → 会提高读者快感阈值
- 正确节奏：吊胃口→喂饱→消化→再吊胃口

### 9.6 上架后保持吸引力的方法

1. 不断保持读者阅读快感
2. 不断调动读者情绪
3. 让读者持续追看埋设的伏笔
4. 让读者看你的书成一种**习惯**

## 结语

网文写作的核心在于：
- **让读者爽**是一切的前提
- **基础扎实**是成功的保障
- **代入感**是吸引读者的关键
- **节奏控制**决定阅读体验

掌握以上九大要素，祝各位作者创作愉快，成神有望！
