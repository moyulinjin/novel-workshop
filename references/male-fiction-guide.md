## 核心精华摘录

### 七字真言

> **简单、粗暴、爽！**

### 天榜密码

1. **开篇钩子**：冲突前置，3章内必有爽点
2. **期待感建立**：让读者想知道"然后呢"
3. **情绪波动**：压抑→爆发→满足循环
4. **节奏把控**：精准到每章结尾留钩子

### 核心卖点公式

```
核心卖点 = 金手指 + 冲突 + 爽点
```

### 七大书名公式

1. 身份反转式：《我，废物女婿，被丈母娘发现身份》
2. 实力碾压式：《都市：开局9999级战力》
3. 逆袭打脸式：《妻子嫌我穷？离婚后我继承千亿》
4. 系统流：《最强系统：开局绑定校花》
5. 穿越流：《穿越大唐：我有军火库》
6. 战神回归式：《战神归来，发现女儿住狗窝》
7. 医术流：《开局救美，被女主播找上门》

### 爽点三翻四震

**三翻**：
- 第一翻：压制
- 第二翻：更压制
- 第三翻：极致压制

**四震**：
- 震惊：主角展现能力
- 震动：配角群体反应
- 震撼：大场面出现
- 震怒：反派被碾压

### 市场定位型大纲模板

```
【开篇钩子】
主角现状 + 金手指觉醒 + 第一个小冲突

【困境设定】
外部压力 + 内部矛盾 + 目标明确

【金手指使用】
初次尝甜头 + 发现规则 + 建立优势

【第一次打脸】
打脸对象 + 打脸方式 + 观众反应

【成长积累】
实力提升 + 势力扩张 + 资源积累

【中期高潮】
更大冲突 + 更大舞台 + 更大爽点

【最终决战】
终极敌人 + 终极对决 + 终极爽点

【圆满结局】
所有伏笔回收 + 圆满收尾 + 新期待建立
```

### 期待感建立方法

| 方法 | 说明 |
|------|------|
| 悬念钩子 | 章尾留悬念，让读者想看下一章 |
| 目标期待 | 主角目标明确，读者想知道能否达成 |
| 身份期待 | 身份隐藏/反转，读者想看揭露 |
| 实力期待 | 实力提升可见，读者想看爆发 |
| 情感期待 | 感情线发展，读者想看在一起 |
