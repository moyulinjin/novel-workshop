# 金手指类型库

## 一、系统流

### 通用系统
```
名称：神豪系统/无敌系统/万能系统
来源：天降系统/穿越绑定/意外激活

核心功能：
  - 任务系统（完成任务获奖励）
  - 商城系统（购买道具/技能）
  - 属性系统（查看/提升属性）
  - 分析系统（洞察/推演/预测）

系统互动：
  - 心灵感应（脑海直接交流）
  - 视野界面（系统UI显示）
  - 实时提示（警告/建议/分析）

升级机制：
  - 完成任务获得经验值
  - 解锁更高级功能
  - 系统等级提升

副作用/限制：
  - 任务有失败惩罚
  - 部分功能需要点数/货币
  - 系统有冷却时间
```

### 系统互动示例（战斗中）
```
系统：检测到敌方左肋灵力流动异常，疑似旧伤。
宿主：有多严重？
系统：建议攻击该位置，命中率提升50%，伤害加成30%。
宿主：好，给我详细攻击路线。
系统：已规划三条路线，推荐路线一：从右侧切入，速度最快，耗时2秒。
宿主：就按路线一，准备。
系统：蓄力中...蓄力完成，随时可以发动。
宿主：动手！
```

### 系统互动示例（日常）
```
系统：宿主，检测到前方有宝物反应，距离300米。
宿主：什么宝物？
系统：初步判断为千年灵草，价值500点数。
宿主：500点...还不错。有没有风险？
系统：扫描周边...发现一名守护者，实力筑基中期，建议谨慎行事。
宿主：筑基中期...有点麻烦。系统能帮忙吗？
系统：可以提供对方弱点分析，命中率提升20%。
宿主：足够了，我去看看。
```

## 二、重生流

### 经典重生
```
核心优势：
  - 先知未来走势
  - 知道彩票/股票/事件
  - 记忆人脉和资源
  - 避开前世坑

限制设计：
  - 重生代价（身体/寿命）
  - 前世遗憾/心结
  - 信息差有时效性
  - 蝴蝶效应改变未来

经典场景：
  - 重生1990年摆地摊
  - 重生高考前逆袭
  - 重生到危机前囤物资
  - 重生弥补前世遗憾
```

### 信息差变现
```
变现路径：
  - 彩票号码（精准记忆）
  - 房价走势（炒房）
  - 股票涨跌（抄底逃顶）
  - 商机发现（抢占先机）

限制：
  - 需要启动资金
  - 不能说信息来源
  - 时机把握要准
  - 避免引起怀疑
```

## 三、祖传宝物

### 老爷爷传承型
```
名称：祖传玉佩/上古戒子/神秘吊坠
外观：古朴/神秘/有符文

核心能力：
  - 智慧传承（老爷爷意识）
  - 功法传承（直接灌输）
  - 推演指导（分析局势）

升级路径：
  - 吸收能量解锁
  - 完成试炼升级
  - 积累功德进化

使用限制：
  - 每日沟通有时间限制
  - 需要灵力驱动
  - 老爷爷记忆不完整
```

### 空间储物型
```
名称：储物戒指/空间手镯/随身空间
外观：古朴玉镯/特殊戒指

核心能力：
  - 储物空间（保鲜/无限大）
  - 种植空间（可种植灵药）
  - 独立空间（闭关修炼）

升级路径：
  - 吸收能量扩大
  - 滴血认主解锁
  - 融合其他空间

使用限制：
  - 有容量上限
  - 进出会消耗时间
  - 可能有认主限制
```

## 四、血脉天赋

### 特殊血脉
```
血脉类型：
  - 龙族血脉（力量/防御）
  - 凤凰血脉（重生/火焰）
  - 神族血脉（天赋/悟性）
  - 妖族血脉（速度/野性）

觉醒条件：
  - 濒死觉醒
  - 特定刺激觉醒
  - 血脉浓度达标
  - 特定物品激活

代价/限制：
  - 血脉反噬（失控）
  - 使用有冷却
  - 消耗生命力
  - 觉醒不完全
```

### 天生异能
```
异能类型：
  - 读心术（读取思想）
  - 预知未来（短暂预见）
  - 空间异能（瞬移/切割）
  - 精神异能（控制/幻象）

使用限制：
  - 消耗精神力
  - 有距离限制
  - 可被屏蔽
  - 使用有副作用
```

## 五、技能异能

### 医术/鉴宝
```
核心优势：
  - 医术碾压（治愈绝症）
  - 鉴宝能力（价值连城）
  - 透视/感知（发现隐藏）

限制：
  - 能力有极限
  - 消耗精力/灵力
  - 需要积累经验
  - 可能被针对
```

### 召唤/召唤师
```
核心优势：
  - 召唤异兽/灵宠
  - 召唤亡灵/傀儡
  - 召唤军团
  - 多重召唤

限制：
  - 召唤物有忠诚度
  - 消耗大量资源
  - 召唤位有限制
  - 召唤物可死亡
```

## 六、综合模板

### 金手指设计检查
- [ ] 核心优势明确？
- [ ] 升级路径清晰？
- [ ] 使用有限制？
- [ ] 有副作用/代价？
- [ ] 与剧情结合紧密？
- [ ] 能制造冲突？
- [ ] 能释放爽点？

### 平衡性原则
1. 不能太强（否则没挑战）
2. 不能太弱（否则没意义）
3. 限制要有（制造紧张感）
4. 升级要快（满足读者期待）
5. 限制要合理（不能太刻意）
