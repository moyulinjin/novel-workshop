# 故事结构模板

## 一、三幕式结构

```
第一幕（25%）：建置
├── 主角正常世界
├── 召唤/事件发生
└── 拒绝/犹豫

第二幕（50%）：对抗
├── 新世界适应
├── 障碍与升级
├── 中点转折
├── 最低点
└── 转折点

第三幕（25%）：解决
├── 高潮准备
├── 最终对决
├── 新平衡建立
└── 结局
```

---

## 二、起承转合结构

### 起点（10%）
- 主角初始状态
- 世界观建立
- 核心矛盾引入

### 承接（20%）
- 冲突展开
- 主角行动
- 遭遇挫折

### 转折（40%）
- 主角成长
- 矛盾升级
- 关键抉择

### 高潮（20%）
- 最终对决
- 矛盾解决
- 情绪爆发

### 结尾（10%）
- 收束
- 新的开始
- 悬念/彩蛋

## 四、长篇分卷结构

### 单卷结构（20-30万字）

```
卷首（5%）：冲突引入
├── 新格局建立
├── 核心矛盾出现
└── 主角目标明确

卷中（75%）：冲突升级
├── 小高潮（20%）
├── 中转折（40%）
├── 中高潮（60%）
└── 大转折（80%）

卷尾（20%）：高潮与收束
├── 终极对决
├── 矛盾解决
└── 新悬念/新卷伏笔
```

### 百万字长篇结构

```
第一阶段：开篇（1-50万字）
├── 核心冲突：生存/立足
├── 主线：成长与崛起
├── 爽点：扮猪吃虎、以弱胜强
└── 副线：势力建立

第二阶段：发展（50-150万字）
├── 核心冲突：地位争夺
├── 主线：势力扩张
├── 爽点：资源垄断、众人归附
└── 副线：情感发展

第三阶段：高潮（150-250万字）
├── 核心冲突：终极对抗
├── 主线：登顶之路
├── 爽点：身份揭秘、终极复仇
└── 副线：势力对决

第四阶段：结局（250-300万字）
├── 核心冲突：最终决战
├── 主线：价值实现
├── 爽点：圆满闭环
└── 副线：余韵收尾
```

## 五、节奏模型

### 波浪式节奏
```
低潮 → 小高潮 → 中潮 → 中高潮 → 高潮 → 短暂低潮 → 更高高潮...
```

### 阶梯式节奏
```
小爽 → 中爽 → 大爽 → 短暂压抑 → 更大爽...
```

### 螺旋式节奏
```
核心冲突逐层升级
每层内部也有起承转合
整体呈螺旋上升
```

## 七、伏笔与呼应

### 伏笔类型

**显性伏笔**
- 明确提出某个信息
- 后续某个时刻呼应
- 读者能记住

**隐性伏笔**
- 细节暗示
- 后续不经意间揭示
- 读者恍然大悟

**结构性伏笔**
- 人物名字暗示性格
- 物品出现预示用途
- 环境描写暗示氛围

### 呼应技巧

**即时呼应**
- 1-3章内呼应
- 保持紧凑感

**中期呼应**
- 10-30章呼应
- 保持悬念

**终局呼应**
- 100章以上呼应
- 制造高潮
