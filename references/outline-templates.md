# 大纲模板与示例

## 分卷大纲模板

### 基本信息区
```
【小说名称】XXX
【题材类型】XXX
【核心爽点】XXX
【字数规划】XXX万字（预计XXX卷）
【更新节奏】日更X章，每章XXX字
```

### 分卷结构模板
```
【第一卷：卷名】（第1-XX章，约XX万字）
【核心冲突】
本卷核心矛盾是什么？主角面临什么困境？

【主要爽点】
1. 开局爽点：XXX
2. 中段爽点：XXX
3. 结尾爽点：XXX

【关键事件】
1. XXX（第X-X章）
2. XXX（第X-X章）
3. XXX（第X-X章）

【人物出场】
- 主角：XXX（身份、能力、目标）
- 主要配角：XXX（与主角关系、功能定位）
- 主要反派：XXX（冲突原因、结局）

【本卷目标】
主角在本卷达成什么成就？
```

## 章节大纲模板

### 单章大纲模板
```
第X章：章节名
【核心事件】
本章发生的核心事件是什么？（一句话概括）

【场景设定】
- 地点：XXX
- 时间：XXX
- 出场人物：XXX

【剧情发展】
1. 开场（约500字）：XXX
   - 钩子：如何吸引读者？
   - 承接：与上一章如何衔接？

2. 中段（约1500字）：XXX
   - 冲突：本章的主要矛盾
   - 爽点：本章的爽点设计

3. 结尾（约500字）：XXX
   - 悬念：如何吸引读者看下一章？

【爽点设计】
- 爽点类型：打脸/收获/突破/反转/其他
- 具体表现：XXX

【情绪曲线】
平静 → 紧张 → 爆发 → 满足

【字数规划】XXX字
```

## 完整示例

### 示例：都市重生商战文

#### 分卷大纲

```
【小说名称】重生2008：从摆地摊到首富
【题材类型】都市重生+商战致富
【核心爽点】信息差变现、逆袭打脸、财富积累
【字数规划】200万字（预计8卷）
【更新节奏】日更2章，每章2500字

【第一卷：重生归来】（第1-30章，约7.5万字）
【核心冲突】
主角重生回高三，家庭负债，急需赚钱

【主要爽点】
1. 开局爽点：重生第一桶金（摆地摊赚几千块）
2. 中段爽点：股票第一战，十万变五十万
3. 结尾爽点：高考成绩曝光，打脸所有质疑者

【关键事件】
1. 重生觉醒，发现商机（第1-3章）
2. 摆地摊赚第一桶金（第4-8章）
3. 进入股市，抓住第一波行情（第9-20章）
4. 高考前夕，与富二代的矛盾（第21-25章）
5. 高考结束，成绩曝光打脸（第26-30章）

【人物出场】
- 主角：张伟（重生者，知晓未来10年走势）
- 主要配角：王强（死党，后期合作伙伴）
- 主要反派：李明（富二代，看不上主角）

【本卷目标】
主角积累第一桶金50万，考入理想大学

---

【第二卷：商海起航】（第31-60章，约7.5万字）
【核心冲突】
大学创业，与本地商人的竞争

【主要爽点】
1. 开局爽点：大学开学，豪车接送引发轰动
2. 中段爽点：生意扩张，月入百万
3. 结尾爽点：收购第一家公司，成为老板

【关键事件】
1. 大学开学，展现财力（第31-35章）
2. 开设实体店，与同行竞争（第36-45章）
3. 抓住2009年机遇，快速扩张（第46-55章）
4. 收购本地公司，正式成为企业家（第56-60章）

【人物出场】
- 主角：张伟（大学生创业者）
- 新增配角：林雪（校花，后期女主）
- 主要反派：本地商人王老板

【本卷目标】
主角资产突破千万，建立第一家公司
```

#### 章节大纲示例（第1-10章）

```
===== 第1-10章大纲 =====

【本阶段主题】重生觉醒，第一桶金
【核心目标】主角从负债家庭到赚到第一笔钱

---

第1章：重生
【核心事件】张伟重生回到十年前的高中教室
【爽点设计】重生设定揭晓
【情绪曲线】困惑 → 震惊 → 兴奋
【字数规划】2500字

第2章：困境
【核心事件】张伟面对家庭的负债困境
【爽点设计】前世记忆对比
【情绪曲线】压抑 → 思考
【字数规划】2500字

第3章：商机
【核心事件】张伟发现重生后的第一个商机
【爽点设计】先知优势展现
【情绪曲线】期待 → 决定
【字数规划】2500字

第4章：第一桶金
【核心事件】张伟开始摆地摊赚钱
【爽点设计】第一笔收入
【情绪曲线】艰辛 → 小满足
【字数规划】2500字

第5章：冲突
【核心事件】遇到地头蛇收保护费
【爽点设计】小冲突建立
【情绪曲线】压抑 → 隐忍
【字数规划】2500字

第6章：反击
【核心事件】张伟用智慧化解危机
【爽点设计】小爽点：智取地头蛇
【情绪曲线】紧张 → 释放
【字数规划】2500字

第7章：扩大
【核心事件】生意越做越大
【爽点设计】收入增加
【情绪曲线】满足 → 野心
【字数规划】2500字

第8章：股市
【核心事件】张伟进入股市
【爽点设计】先知优势在股市
【情绪曲线】期待 → 紧张
【字数规划】2500字

第9章：翻倍
【核心事件】股票赚了第一笔大钱
【爽点设计】中爽点：十万变五十万
【情绪曲线】紧张 → 狂喜
【字数规划】2500字

第10章：高考
【核心事件】高考来临，张伟信心满满
【爽点设计】期待打脸
【情绪曲线】期待 → 决心
【字数规划】2500字
```

## 大纲检查清单

- [ ] 主线是否清晰贯穿？
- [ ] 结构是否完整合理？
- [ ] 爽点分布是否合理？
- [ ] 节奏是否张弛有度？
- [ ] 人物成长线是否清晰？
- [ ] 冲突是否层层递进？
- [ ] 每卷是否有明确目标？
