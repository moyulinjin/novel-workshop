## 1. 《故事》- 罗伯特·麦基

### 核心原理

**故事三角**
- **大情节**：经典设计，线性时间，正能量结局，外部冲突
- **小情节**：最小主义，片段化，省略因果，情绪化结局，内部冲突
- **反情节**：反结构，非线性，因果关系弱化，偶发事件

**冲突类型**
- **内心冲突**：人物自我对抗
- **个人冲突**：人与人之间的对抗
- **外部冲突**：人与自然/社会的对抗

**故事价值与张力**
- 故事是关于永恒、普遍形式的创造
- 价值是故事讲述的货币：正义/邪恶、真理/谎言、爱/恨、生/死
- 张力来自价值观的对立和碰撞

### 结构原理

**场景设计**
- 每一个场景就是一个缩微故事，有其内在的戏剧性动作
- 场景 = 变化，变化需要冲突
- 场景结束的转折点产生情感变化

**节拍与动作**
- 节拍是动作/反应中最小单位
- 场景由节拍构成
- 场景构建序列，序列构建幕，幕构建故事

**场景三要素**
1. 什么人处于什么情势中？
2. 什么愿望/需求？
3. 什么障碍？

## 2. 《冲突与悬念》- 詹姆斯·斯科特·贝尔

### 核心公式

> **冲突是引擎，悬念是变速箱**

### LOCK系统

| 要素 | 说明 |
|------|------|
| **L - Lead** | 引人注目的主人公 |
| **O - Objective** | 明确的目标 |
| **C - Conflict** | 激烈的冲突 |
| **K - Knockout** | 精彩的结尾 |

### 三种死亡类型

| 类型 | 说明 | 强度 |
|------|------|------|
| **肉体死亡** | 生命威胁，最直接 | ★★★★★ |
| **职场死亡** | 事业/地位威胁 | ★★★ |
| **心理死亡** | 内心世界崩塌，身份认同危机 | ★★★★★ |

### 冲突构建技巧

**让主人公主动出击**
- 被动的主人公让读者失去兴趣
- 主动行动即使失败也更能吸引读者

**层层加码**
- 建立冲突 → 升级冲突 → 再次升级 → 最终决战
- 每次升级要超出读者预期

**秘密与揭示**
- 隐藏信息制造张力
- 适时揭示真相

**让人物彼此对立**
- 让每个角色都有自己的议程
- 不同议程的碰撞产生冲突

## 3. 《故事策略》- 埃里克·埃德森

### 英雄目标序列（23个段落）

| 段落 | 功能 | 位置 |
|------|------|------|
| 1 | 人物/世界观 | 开篇 |
| 2 | 日常世界 | 第1幕 |
| 3 | 主题呈现 | 铺垫 |
| 4 | 催化剂/意外事件 | 催化剂 |
| 5 | 辩论/内心挣扎 | 辩论 |
| 6 | 接受任务 | 第1幕结尾 |
| 7 | 制定计划 | 第2幕开始 |
| 8 | 盟友出现 | 新世界 |
| 9 | 第一考验 | 玩伴游戏 |
| 10 | 承诺升级 | 进展 |
| 11 | 伪失败 | 娱乐游戏 |
| 12 | 重大转折 | 中点 |
| 13 | 倒数计时 | 坏事发生 |
| 14 | 彻底失败 | 坏事蔓延 |
| 15 | 灵魂考验 | 第2幕结尾 |
| 16 | 觉醒/新计划 | 第3幕开始 |
| 17 | 荣耀时刻 | 高潮前 |
| 18 | 计划验证 | 准备 |
| 19 | 最终考验 | 高潮 |
| 20 | 严酷考验 | 后果 |
| 21 | 代价 | 代价 |
| 22 | 决定性行动 | 结局 |
| 23 | 新平衡 | 结局 |

### 主题寻找方法

- 故事开始前先问：你想通过故事表达什么？
- 主题不是道德说教，而是对生活的某种洞察
- 通过人物选择来展示主题

### 情节设计要点

- 每个情节点都应该引向下一个情节点
- 因果链条要紧密
- 转折要出人意料又在情理之中

## 4. 《情节！情节！》- 诺亚·卢克曼

### 核心观点

> **人物是情节的基石**

### 人物两面性

| 表面生活 | 内心生活 |
|----------|----------|
| 职业、外表、社会角色 | 欲望、恐惧、秘密、创伤 |
| 他人眼中的形象 | 真实的内心世界 |
| 推动情节的外部行动 | 驱动人物的内在动机 |

### 人物特征调查表

**核心身份**
- [ ] 年龄、性别、外貌特征
- [ ] 职业和社会地位
- [ ] 教育背景
- [ ] 家庭背景和关系
- [ ] 宗教信仰（如果有）
- [ ] 政治立场（如果有）

**性格特质**
- [ ] 优点（5个以上）
- [ ] 缺点（5个以上）
- [ ] 怪癖或独特习惯
- [ ] 口头禅或特殊表达
- [ ] 处理压力的方式

**内心世界**
- [ ] 最深的恐惧
- [ ] 最强烈的欲望
- [ ] 最隐秘的秘密
- [ ] 内心创伤/心结
- [ ] 核心信念/价值观
- [ ] 性格矛盾点

**关系网络**
- [ ] 亲密关系（家庭/爱人/挚友）
- [ ] 敌人/对手
- [ ] 盟友/伙伴
- [ ] 导师/引路人
- [ ] 仰慕者/追随者

**背景故事**
- [ ] 童年重要事件
- [ ] 人生转折点
- [ ] 遗憾/未完成的事
- [ ] 对未来的期望

### 情节构建原则

1. **因果律**：事件之间要有因果联系
2. **张力曲线**：保持张力的起承转合
3. **意外与必然**：转折要出人意料，但事后回想又必然发生
4. **主题统一**：所有情节都应服务于主题

## 5. 《怎样写故事》- 莉萨·克龙

### 核心原理

> **故事是大脑的语言**
> 
> **小说的秘密在于第三条轨道：主人公内心斗争**

### 6W原则

| 要素 | 问题 | 作用 |
|------|------|------|
| **Who** | 主人公是谁？ | 建立情感连接 |
| **Want** | 主人公想要什么？ | 驱动故事发展 |
| **When** | 故事发生在什么时候？ | 时代背景 |
| **What** | 如果失败了会怎样？ | 建立赌注 |
| **Where** | 故事发生在哪里？ | 增强沉浸感 |
| **Why** | 主人公为什么想要？ | 深层动机 |

### 故事的三条轨道

| 轨道 | 内容 | 说明 |
|------|------|------|
| **第一条** | 外部情节 | 发生的事件、采取的行动 |
| **第二条** | 人物关系 | 与他人的互动、情感变化 |
| **第三条** | 内心斗争 | 主人公的内心成长与转变 |

**重点**：第三条轨道（内心斗争）是故事的灵魂

### 内在缺陷与恐惧

- 每个主人公都有内在缺陷
- 缺陷源于过去的创伤或恐惧
- 故事过程就是缺陷暴露和克服的过程
- 缺陷必须在高潮处被征服

### 场景写作要点

1. **情感意义**：每个场景都要有情感目的
2. **内心行动**：场景中人物内心的变化
3. **外在行动**：场景中发生的具体事件
4. **因果链条**：一个场景必须引向下一个场景

## 6. 《畅销书写作技巧》- 德怀特·V·斯温

### 畅销书公式

**好故事 = 冲突 + 行动 + 结局 + 情感 + 展示**

### 五要素详解

| 要素 | 说明 | 示例 |
|------|------|------|
| **冲突** | 人与人、人与自然、人与命运的对抗 | 主角面临困境/威胁 |
| **行动** | 人物为解决冲突所做的努力 | 主角采取的行动 |
| **结局** | 冲突的解决 | 胜利/失败/和解 |
| **情感** | 故事引发的读者情感共鸣 | 感动、紧张、兴奋 |
| **展示** | 通过具体场景而非叙述展示 | 场景化描写 |

### 三幕结构应用

| 幕 | 内容 | 比例 |
|----|------|------|
| **第一幕** | 建置：引入人物、世界、冲突 | 25% |
| **第二幕** | 复杂化：冲突升级、阻碍重重 | 50% |
| **第三幕** | 解决：高潮对决、结局 | 25% |

### 开篇技巧

- 第一句话就要抓住读者
- 尽快引入冲突或悬念
- 建立"故事承诺"：让读者知道这是一个值得读的故事
- 从中间开始：in medias res（从中间开始）

## 核心框架总结

### 📌 人物塑造核心

| 来源 | 核心要点 |

| 麦基 | 内在冲突是人物深度的基础 |
| 贝尔 | 人物要有明确目标和强烈欲望 |
| 卢克曼 | 表面生活与内心生活双重构建 |
| 克龙 | 内在缺陷是人物转变的关键 |
| 斯温 | 人物行动展现性格 |

### 📌 冲突构建核心

| 来源 | 核心要点 |
|------|----------|
| 麦基 | 冲突类型：内心/个人/外部 |
| 贝尔 | LOCK系统：引人注目+目标+冲突+结尾 |
| 贝尔 | 三种死亡：肉体/职场/心理 |
| 卢克曼 | 因果链条是情节基础 |
| 斯温 | 冲突是故事的第一要素 |

### 📌 故事结构核心

| 来源 | 核心要点 |
|------|----------|
| 麦基 | 故事三角：情节/反情节/小情节 |
| 贝尔 | 从头到尾保持张力 |

| 克龙 | 三条轨道：情节/关系/内心 |

### 📌 场景写作核心

| 来源 | 核心要点 |
|------|----------|
| 麦基 | 场景=变化，变化需要冲突 |
| 克龙 | 场景必须有情感意义 |
| 克龙 | 内在行动+外在行动双线 |
| 斯温 | 展示而非叙述 |

### 📌 实用检查清单

**人物检查**
- [ ] 主人公有明确的目标吗？
- [ ] 主人公有内在缺陷/恐惧吗？
- [ ] 表面生活和内心生活都构建了吗？
- [ ] 人物关系有张力吗？

**冲突检查**
- [ ] 冲突是否贯穿始终？
- [ ] 冲突是否层层升级？
- [ ] 有三种死亡的威胁吗？
- [ ] 每个场景都有冲突吗？

**结构检查**
- [ ] 第一幕是否建立了冲突？
- [ ] 中点是否有重大转折？
- [ ] 高潮是否解决了所有核心冲突？
- [ ] 结局是否令人满意？

**场景检查**
- [ ] 每个场景都有变化吗？
- [ ] 每个场景都有情感目的吗？
- [ ] 使用了展示而非叙述吗？
- [ ] 场景之间因果链条紧密吗？
