## 二、武侠小说三要素

### 武（武功）
- 不渲染暴力血腥场面
- 通过武功探讨人体极限
- 武功高强者越不注重兵刃
- 顶级高手用掌而非刀

### 侠（侠义精神）
- 满足读者的侠客梦
- 带来美的享受
- 传承文化内涵

### 情（情感线索）
- 江湖情谊
- 儿女情长
- 家国情怀

## 三、武侠小说经典结构

### 开头模式
1. 灭门惨案，引出主角
2. 偶遇高人，获得传承
3. 身世之谜，逐步揭开
4. 江湖历练，层层递进

### 中段发展
1. 正邪对峙
2. 秘籍争夺
3. 门派恩怨
4. 势力洗牌

### 结局模式
1. 正胜邪败
2. 功成身退
3. 归隐山林
4. 新老交替

## 四、武功设计指南

### 取名原则
- 有文学性：落英神剑、九阴真经
- 有动物性：降龙十八掌、打狗棒法
- 有意境：凌波微步、独孤九剑
- 有气势：一剑破万法、天地同寿

### 武功层次
- 外功：拳脚刀剑
- 内功：心法修为
- 轻功：腾挪闪避
- 暗器：防不胜防

## 六、经典武侠公式

```
主角 = 身世坎坷 + 奇遇不断 + 升级飞快
反派 = 阴谋算计 + 武功高强 + 最后必败
女主 = 美貌与实力并存 + 情有独钟
```

## 七、注意事项

1. **武打描写要有美感**，不要纯暴力血腥
2. **侠义精神要贯穿**，不能主角也滥杀无辜
3. **武功设定要合理**，不能前后矛盾
4. **江湖规矩要自洽**，要建立自己的规则体系
5. **情感线要自然**，不能太狗血
