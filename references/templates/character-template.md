## 五、角色类型库

### 主角类型

| 类型 | 特点 | 爽点来源 |
|------|------|---------|
| 废柴逆袭型 | 初始弱小，逐步变强 | 成长快感 |
| 天才横推型 | 开局就强，一路碾压 | 碾压快感 |
| 穿越者型 | 拥有现代知识/见识 | 认知差快感 |
| 重生者型 | 知道未来，避免悲剧 | 先知快感 |
| 系统流型 | 拥有任务系统 | 升级快感 |
| 传承型 | 获得前辈衣钵 | 继承快感 |

### 女主类型

| 类型 | 特点 | 情感爽点 |
|------|------|---------|
| 傻白甜型 | 天真可爱，依赖主角 | 保护欲 |
| 女强人型 | 独立自主，与主角并肩 | 势均力敌 |
| 傲娇型 | 嘴硬心软，口是心非 | 反差萌 |
| 病娇型 | 偏执深情，不顾一切 | 独占欲 |
| 御姐型 | 成熟稳重，照顾主角 | 被照顾感 |
| 清冷型 | 气质出尘，不近人情 | 融化冰山 |

### 反派类型

| 类型 | 特点 | 打脸方式 |
|------|------|---------|
| 嚣张跋扈型 | 到处惹事，自以为是 | 当众打脸 |
| 阴险狡诈型 | 暗中使坏，表面无辜 | 揭露真相 |
| 嫉妒怨恨型 | 因嫉妒而针对主角 | 以德报怨 |
| 立场对立型 | 无对错，只是立场不同 | 尊重对手 |
| 疯狗型 | 无脑针对，见就打 | 教训恶犬 |
| 幕后黑手型 | 操纵他人，自己隐藏 | 揪出真凶 |

## 六、人物关系设计

### 关系矩阵

```
主角
├── 正面关系
│   ├── 导师（给予帮助）
│   ├── 爱人（情感支撑）
│   ├── 挚友（并肩作战）
│   ├── 小弟（追随崇拜）
│   └── 贵人（关键时刻帮助）
│
├── 中立关系
│   ├── 陌生人（擦肩而过）
│   ├── 竞争者（良性竞争）
│   └── 旁观者（见证主角成长）
│
└── 负面关系
    ├── 宿敌（生死大仇）
    ├── 小人（恶心骚扰）
    ├── 误会者（被蒙蔽的敌人）
    └── 替罪羊（无冤无仇但被利用）
```

### 关系演变设计

**友→敌**
- 背叛
- 误解加深
- 利益冲突

**敌→友**
- 共同敌人
- 打服了
- 救命之恩

**中立→友**
- 帮助过主角
- 有共同目标
- 欣赏主角

## 七、人设一致性检查

### 检查清单

1. **性格一致性**
   - [ ] 行为是否符合性格设定
   - [ ] 说话方式是否一致
   - [ ] 关键时刻反应是否合理

2. **能力一致性**
   - [ ] 实力表现是否稳定
   - [ ] 金手指使用是否有限制
   - [ ] 成长是否合理

3. **关系一致性**
   - [ ] 关系发展是否自然
   - [ ] 互动是否合理
   - [ ] 变化是否有铺垫

4. **动机一致性**
   - [ ] 行为动机是否清晰
   - [ ] 目标是否合理
   - [ ] 选择是否合理

## 八、高光场景设计

### 设计要素

**情节要素**
- 什么情况
- 主角怎么做
- 结果如何

**情绪要素**
- 读者感受到什么
- 爽点在哪里
- 如何超出预期

**记忆要素**
- 有没有标志性动作
- 有没有经典台词
- 有没有独特设定
