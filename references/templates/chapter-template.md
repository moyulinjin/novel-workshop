# 章节写作模板

## 一、单章四段式结构

```markdown
=== 单章写作框架 ===

章节：第X章 [章节名]
类型：[日常/战斗/情感/过渡]
字数：1500-2000字

【四段式结构】

1. 引入（10-20%，150-400字）
   - 承接上文（1-2句话）
   - 抛出问题/冲突
   - 建立期待

2. 发展（40-50%，600-1000字）
   - 冲突展开
   - 人物互动
   - 信息传递
   - 伏笔埋设

3. 高潮（30-40%，450-800字）
   - 核心爆发点
   - 情绪顶点
   - 爽点释放（如有）

4. 结尾（10%，150-200字）
   - 收束本章
   - 引向下章
   - 留下钩子
```

## 二、黄金开篇模板

```markdown
=== 黄金开篇模板 ===

【开篇三要素】
1. 穿越/重生设定（快速建立代入）
2. 世界观（用冲突带出，不大段说明）
3. 身份处境（第3章前出现威胁或机会）

【开篇类型】

类型A：冲突式开篇
"砰！"
[冲突发生]
[主角处境揭示]
[金手指暗示]

类型B：悬念式开篇
[异常现象/事件]
[主角视角]
[疑问抛出]

类型C：反差式开篇
[强烈对比/反差]
[引发好奇]
[解释原因]
```

### 开篇公式

```markdown
【前100字】
建立场景 + 抛出悬念/冲突

【100-300字】
揭示身份 + 展示困境 + 金手指暗示

【300-500字】
冲突升级 + 建立期待 + 钩子结尾
```

## 三、打斗场景模板

```markdown
=== 打斗场景五层结构 ===

【第一层：对峙】
- 双方登场
- 气势交锋
- 心理博弈
- 试探开始

【第二层：试探】
- 招式试探
- 实力摸底
- 发现弱点
- 调整策略

【第三层：僵持】
- 正式交锋
- 互有攻防
- 形势胶着
- 观众紧张

【第四层：突破】
- 找到机会
- 爆发底牌
- 形势逆转
- 反派震惊

【第五层：决胜】
- 致命一击
- 结果揭晓
- 后续影响
- 打脸收尾

=== 打斗示例 ===

反派：举起长剑，剑尖直指林风，"就凭你，也配与我动手？"
（对峙层 - 嚣张）

林风没有回答，只是微微侧身，躲过第一剑。
（试探层 - 示弱）

剑光如雨，众人只见林风不断后退，看似毫无还手之力。
（僵持层 - 蓄势）

"差不多了。"林风轻声说道。
（突破层 - 转折）

下一秒，反派的长剑停在半空。他的眼中满是难以置信。
（决胜层 - 碾压）
```

## 四、对话场景模板

```markdown
=== 对话场景创作指南 ===

【对话核心原则】
1. 口语化 - 对话要像真实说话
2. 独立成段 - 每句对话单独成段
3. 动作代替"说" - 用动作描写代替"某某说"
4. 潜台词 - 有言外之意

【对话格式】
"第一句对话。"角色动作/表情。
"第二句对话。"另一角色回应。

【对话与叙述比例】
- 冲突场景：对话70%，叙述30%
- 日常场景：对话50%，叙述50%
- 心理场景：对话30%，叙述70%

【潜台词设计】
表面说的 = 字面意思
实际想的 = 潜台词
读者理解的 = 深层含义

=== 对话示例 ===

"你真的要这样做？"她看着林风。
（表层：疑问）

林风没说话，只是笑了笑。
（潜层：他心意已决）

"你会后悔的。"她转身离开。
（表层：威胁）
（潜层：她在表达关心）
```

## 五、爽点场景模板

```markdown
=== 爽点场景创作模板 ===

【铺垫阶段】
1. 反派出场，轻视主角
2. 压迫升级，主角隐忍
3. 众人嘲讽，读者愤怒
4. 暗示即将反转

【爆发阶段】
1. 主角抓住机会
2. 亮出底牌/实力
3. 反派措手不及
4. 读者期待实现

【打脸阶段】
1. 反派被打
2. 众人震惊
3. 配角惊叹
4. 余味延续

=== 爽点场景示例 ===

反派：林风？就凭你这个废物，也配站在这里？
（铺垫 - 嚣张）

周围响起一片嘲笑声。
林风低着头，看不清表情。
（铺垫 - 压抑）

"我劝你还是自己滚下去吧。"反派走近一步。
（铺垫 - 升级）

林风缓缓抬起头，眼中闪过一丝笑意。
（转折 - 暗示）

下一秒，反派发现自己动不了了。
（爆发 - 开始）

"你...你做了什么？！"
（打脸 - 反派震惊）

"没什么。"林风轻轻抬手，"只是让你安静一下。"

"砰！"
反派被一股无形的力量击飞出去，重重摔在十丈之外。
（打脸 - 结果）

全场死寂。
（打脸 - 众人反应）
```

## 六、情感场景模板

```markdown
=== 情感场景创作指南 ===

【情感渲染要素】
1. 情感坐标系 - 喜/怒/哀/惧/惊
2. 五感锚点 - 视觉/听觉/嗅觉/触觉/味觉
3. 留白艺术 - 不说破，让读者自己感受
4. 身体记忆 - 用身体反应表现情感

【悲伤场景】
避免：直接写"他很悲伤"
正确：通过细节展现悲伤

示例：
老张站在门口，手里攥着儿子的照片。
他没有哭。
只是站了很久很久。
直到肩膀开始颤抖，他才转身进屋。

【甜蜜场景】
避免：直接写"他们很开心"
正确：通过互动和细节展现甜蜜

示例：
"小心，烫。"
她接过碗，指尖不小心碰到他的手。
两人同时缩回。
她低头喝汤，耳根发红。

【紧张场景】
避免：直接写"他很紧张"
正确：通过身体反应和环境描写表现紧张

示例：
林风的掌心沁出汗珠。
周围的声音仿佛变得很远。
裁判的哨声响起。
他深吸一口气，起跳。
```

## 七、章节结尾模板

```markdown
=== 章节结尾类型 ===

【悬念型结尾】
抛出新问题，留下未解之谜
示例：
就在这时，门被推开了。
站在门口的人，让林风的瞳孔骤然收缩。
"怎么...是你？"

【冲突型结尾】
正要动手被打断，或危机降临
示例：
林风正要出手。
"住手！"
一个威严的声音从身后传来。

【期待型结尾】
主角即将爆发，反派即将遭殃
示例：
林风看着反派，嘴角勾起一丝冷笑。
"现在，轮到我了。"

【反转型结尾】
出乎意料的转折
示例：
"你说的那个林风，"那人笑了笑，"就是我。"

=== 结尾检查清单 ===
- [ ] 是否留下钩子
- [ ] 是否引向下一章
- [ ] 是否强化读者期待
- [ ] 字数是否达标
```
