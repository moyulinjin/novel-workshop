# 都市剧情爽点设计大全

> 都市网文读者爽感套路汇总

## 一、扮猪吃虎类

1. **宝物唾手可得** - 对别人珍贵的宝物对主角轻而易举
2. **撞见冒牌货** - 小有名气时撞到假装自己的人
3. **小弟挑衅** - 隐藏身份时被小弟挑衅，老大出来道歉
4. **轻描淡写** - 做出惊天动地的事后跟围观群众谈笑风生
5. **不惜代价救人** - 哪怕旁人眼中不值当也要救小弟/亲人/朋友
6. **被动出名** - 别人寄望主角出手救治绝境，但主角并不知晓
7. **废变宝** - 在别人手中名不见经传的技能，在主角手里派上大用
8. **被轻视被为难** - 明明是贵宾却被看门的/其他客人刁难
9. **高人即本人** - 其他人眼里的神秘高手就是被忽视/轻视/嘲笑的主角
10. **以物换物** - 主角有没用但可换宝物的物品

## 二、打脸逆袭类

11. **聚会震惊** - 威风八面的人物见到主角后纷纷放下颜面请教/称赞
12. **训练小弟** - 主角用严格方法训练，读者期待出师后的表现
13. **冷门取胜** - 拿不被看好的武器/宠物/功法打败强敌
14. **危险变福利** - 别人眼里的危险致命之物，主角轻松化解
15. **帮助回报** - 被帮助的人带来意想不到的丰厚回报
16. **绝境获益** - 别人看来是危机，主角不但没事还能获取好处
17. **兄弟情深** - 跟强敌战斗只为一救兄弟/女人
18. **蚍蜉撼树** - 被轻易击倒但顽强站起，哄笑声变小
19. **涌泉相报** - 滴水之恩以涌泉报，回报让人吃惊
20. **不知真相** - 不知情者用小弟贬低主角，却不知小弟力量是主角赐予

## 三、装逼打脸类

21. **被迫离开又不愿** - 被派来打探情报的小弟/美女，主角让其离开，前者却不愿
22. **轻视震惊** - 一直轻视主角的人被主角小弟/朋友震惊
23. **看戏心态** - 只有主角知道真相，一直在看一场好戏
24. **难题变简单** - 别人感觉无法完成的谜题，主角轻松完成
25. **势力联合** - 相互敌对的势力因主角影响联合
26. **完善传承** - 神秘世家美女获残缺传承，主角可帮她完善
27. **绝望变希望** - 短时间内完成让别人绝望的事情
28. **阴险对敌** - 主角对敌人阴险狡诈、不择手段，对朋友诚挚慷慨
29. **化解必杀** - 将别人视为不可抵挡的致命一击轻松化解
30. **认真搞笑** - 主角去严肃认真地搞笑

## 四、成长逆袭类

31. **特长派上用场** - 主角的某个特长在关键时刻派上大用
32. **努力回报** - 通过远超常人的努力和汗水得到回报
33. **平凡不平凡** - 每次考试都一分不差考到60分
34. **微弱优势** - 每次只赢别人一点点，但别人怎么也追不上
35. **小弟反被教训** - 想让老大教训主角的小弟，反被知情的老大教训
36. **聪明人明智** - 聪明人让手下不招惹主角
37. **幕后大佬** - 谁都不知道明面上的掌控者其实是听命于主角的傀儡
38. **高岭之花心动** - 无人能及的高岭之花因一件小事对主角产生好感
39. **升级期待** - 刻苦努力升级后的第一件事最让读者期待
40. **小聪明+大智慧** - 主角有小聪明也有大智慧

## 五、势力扩张类

41. **被抛弃后崛起** - 被强大势力抛弃，隐瞒身份从零发展
42. **雪中送炭** - 大势力有难时主角施以援手
43. **废物变宝** - 从别人手中获取破烂转手变宝物高价卖出
44. **能力用在正道** - 不单用在正途，也用在歪门邪道
45. **意外宠物** - 获得宠物以为是普通小动物，没想到另有秘密
46. **为友出头** - 同伴被强者欺压，主角为其出头，出乎意料影响巨大
47. **隐藏身份** - 不知情者当他是小角色，知道真相的人期待他站出来
48. **劫恶济己** - 劫掠恶人物品不但不劳而获，还能收获好评
49. **护短护友** - 善待小弟为小弟谋福利，让其态度由不服变为敬畏
50. **蚍蜉撼树有好处** - 在别人看来是蚍蜉撼树，对主角其实有好处

## 六、忠犬与反派类

51. **等待的忠犬** - 被欺骗的忠犬至死都在等待将它抛弃的人归来
52. **小弟变强** - 主角训练的小弟让人期待训练出师后的实力
53. **不知者无畏** - 不知情者贬低主角，却不知主角是赐予小弟力量的人
54. **知情者感动** - 周围哄笑声越来越小，知情者/被救者暗处大为感动
55. **兄弟/女人非被绑架** - 主角以为兄弟或女人被绑架去救，却非被绑架
56. **不屈精神** - 即使面对强敌困境也坚守底线和信念，让读者钦佩

## 七、爽点设计原则

### 核心公式
```
期待感 + 打压 + 打脸 = 爽
```

### 节奏把控
1. **设置目标** - 让读者知道主角要什么
2. **制造困难** - 让读者担心能不能成功
3. **绝地反击** - 主角证明自己
4. **收获回报** - 升级/装逼/抱得美人

### 情绪调动
1. **让读者站队** - 先让读者喜欢主角
2. **制造对立** - 让反派嚣张
3. **调动情绪** - 让读者愤怒/期待
4. **满足期待** - 让主角打脸

## 八、爽文字数节奏

| 阶段 | 字数 | 爽点密度 |
|------|------|----------|
| 开篇 | 1-3章 | 密集爽点 |
| 前期 | 4-20章 | 每章1-2个小爽点 |
| 中期 | 20-50章 | 每10章1个大爽点 |
| 高潮 | 50-100章 | 连续爽点轰炸 |
| 结尾 | 最后10章 | 终极爽点+收尾 |

## 九、注意事项

1. **不要无脑爽** - 要有合理的情节支撑
2. **不要太过分** - 打脸要有度
3. **不要忽略感情** - 爽文也要有感情线
4. **不要忘记成长** - 主角要有成长线
5. **不要忘记配角** - 配角也要有性格
