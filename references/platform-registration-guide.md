# 平台签约与投稿指南

> 晋江、起点等主流平台签约流程与投稿技巧

---

## 一、晋江签约SOP流程

### 1.1 了解平台基础规则

**官方入口**：晋江官网 → 作者导航

可查阅：
- 全勤奖规则
- 上榜规则
- 作者专栏设置
- 封面上传规范

### 1.2 扫榜方法论

扫榜是统计学概率问题，核心是找到热点趋势：

**操作步骤**：
1. 打开晋江言情新晋榜/千字收益榜
2. 罗列榜单前10名，统计信息：
   - 书名
   - 题材类型
   - 文风（轻松/正剧）
   - 字数
   - 收藏数
   - 是否签约
   - 作者情况

3. 关键词拆分归类：
   ```
   现代文标签：团宠、锦鲤、年代文、后妈、欧皇、娱乐圈、暗恋...
   古言标签：穿书、清穿、不好惹...
   幻想标签：星际、美食、退学...
   ```

4. 分析要点：
   - 是否包含热点关键词？
   - 是否蹭热点文？
   - 文名是否一眼看出主线？
   - 热点是否有变种/重合？
   - 热点关键词在第一章如何体现？

**总结**：汇总数据 → 了解热点趋势 → 蹭热点 or 差异化竞争

### 1.3 开文准备流程

1. **了解雷区**：晋江有详细雷区列表（师生恋、骨科等）

2. **扒纲学习**：
   - 找同题材爆款文前1万字
   - 复制粘贴成独立文档
   - 分析：描写、人物塑造、节奏、剧情穿插、场景设计

3. **素材库搭建**：
   - 推荐工具：口袋写作APP
   - 核心：按自己需求改良
   - 重点：素材库要自己整理才有用

4. **先定设定后写正文**：
   > ⚠️ 不要先写正文一万字再来套设定，这样后期必崩！

5. **核心梗设计**：
   - 晋江言情"梗"非常重要
   - 核心梗是重点
   - 文案抛梗三流派：打脸流、真香流、沙雕流
   - 会拆一个就能写千千万万个

## 二、晋江审签核心要素

### 2.1 过签关键点

| 要素 | 说明 |
|------|------|
| 书名 | 一眼看出主线剧情 |
| 文案 | 有梗、有趣、有看点 |
| 核心梗 | 必须清晰、有创意 |
| 前三章 | 冲突前置、节奏快 |

### 2.2 晋江雷区规避

- 师生恋（禁）
- 骨科（禁）
- 敏感政治话题（禁）
- 抄袭判定（同质化严重）

### 2.3 扒纲技巧

**工具**：TXT文档

**步骤**：
1. 复制目标文前1万字
2. 按场景拆分大纲
3. 分析每段的：起承转合
4. 提炼：打脸点、爽点、节奏点

## 三、起点/男频投稿

### 3.1 男频主流题材

| 题材 | 核心爽点 | 代表类型 |
|------|----------|----------|
| 都市 | 装逼打脸、赚钱逆袭 | 重生、神医、战神 |
| 玄幻 | 升级打怪、宗门崛起 | 废柴逆袭、系统流 |
| 仙侠 | 修仙长生、红尘历练 | 凡人流、杀伐果断 |
| 悬疑 | 推理破案、惊悚刺激 | 探险流、刑侦流 |

### 3.2 起点投稿要点

**字数要求**：
- 签约门槛：3万字
- 上架门槛：30万字
- 全勤：日更4000+

**核心套路**：
- 黄金三章必须有冲突
- 开篇最好有打脸
- 金手指必须明确

## 五、素材库搭建

### 5.1 为什么要建素材库

- **职业发展必做**：系统化积累
- **避免卡文**：有东西可查
- **提升质量**：持续学习优秀作品

### 5.2 素材库分类

```
素材库/
├── 爆款分析/
│   ├── 书名分析/
│   ├── 文案分析/
│   └── 扒纲存档/
├── 人物模板/
│   ├── 主角人设/
│   ├── 反派人设/
│   └── 配角人设/
├── 场景描写/
│   ├── 都市场景/
│   ├── 古风场景/
│   └── 幻想场景/
├── 爽点记录/
│   ├── 打脸爽点/
│   └── 升级爽点/
└── 词汇积累/
    ├── 动作词/
    ├── 环境词/
    └── 心理词/
```

### 5.3 写作提高书单

**必读**：《起点总编"抄袭"技巧》- 杨晨
> "抄"是每个新手必经阶段，但不是常规意义的抄袭

## 六、核心提醒

1. **扫榜比埋头写更重要** - 了解市场趋势
2. **先定设定后写正文** - 避免后期崩盘
3. **核心梗是关键** - 晋江尤其重要
4. **扒纲是学习捷径** - 分析爆款结构
5. **素材库要自己建** - 别人的没意义
