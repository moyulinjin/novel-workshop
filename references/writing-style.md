# 快节奏网文写作规范

## 核心原则

### 本质
- **目的**：让读者快速进入剧情，持续获得爽感
- **特点**：简洁、直接、高效
- **禁忌**：文青式表达、复杂描写、过度铺垫

### 五大核心要求
1. ✅ 第三人称叙事
2. ✅ 语言直白口语化
3. ✅ 短句碎段
4. ✅ 无复杂修饰词
5. ✅ 以主角情绪推动剧情

---

## 叙事视角

### 第三人称（必须）
```
正确示例：
张伟看着眼前的银行卡，愣住了。

一千万。

他这辈子都没见过这么多钱。

上辈子，他累死累活打工，月薪不过五千。

这辈子，重生回来才三天，就有了这笔巨款。
```

```
错误示例（第一人称）：
我看着眼前的银行卡，内心翻涌着难以言表的激动...
```

### 视角聚焦
- 始终聚焦于主角视角
- 避免频繁切换视角
- 配角视角仅用于打脸反转

---

## 语言风格

### 直白口语化（必须）
```
正确示例：
"卧槽，这小子哪来的钱？"

"不会是借的高利贷吧？"

周围人议论纷纷。

张伟没理会，直接刷卡。

"滴——"

支付成功。
```

```
错误示例（文青腔）：
周围的议论声此起彼伏，充满了质疑与不解...
```

### 禁止的文青式表达

| 禁止类型 | 错误示例 | 正确改写 |
|---------|---------|---------|
| 心理独白 | 他的内心翻涌着复杂的情绪 | 他愣住了 |
| 环境描写 | 夕阳的余晖洒在破旧的街道上 | 天快黑了 |
| 情感渲染 | 那一刻，他感受到了命运的无常 | 他妈的，真倒霉 |
| 隐喻比喻 | 他的眼神如同深潭，藏着无尽的故事 | 他盯着那个人 |
| 诗意表达 | 时光荏苒，岁月如梭 | 三年过去了 |

## 段落结构

### 短句碎段（必须）
```
正确示例：
重生了。

张伟睁开眼，盯着天花板看了半天。

这是十年前的老房子。

墙皮脱落，露出里面的红砖。

屋顶还漏着雨。

他爬起来，走到窗边。

外面的街道很破。

路边摆着地摊，卖菜的，卖早点的。

一切都那么熟悉。
```

```
错误示例（一大段）：
重生了的他睁开眼睛...
```

### 段落长度规范
- **每段**：不超过3行（约50字）
- **每句**：不超过20字
- **对话**：每句独立成段
- **动作**：每个动作独立成段

### 换行时机
- 动作结束 → 换行
- 对话结束 → 换行
- 情绪变化 → 换行
- 时间跳跃 → 换行

---

## 开篇规范

### 开篇三要素（必须）
1. **交代穿越/重生**：第一句或第一段必须说明
2. **交代世界观**：简洁说明时间、地点、背景
3. **交代身份**：主角是谁，处于什么处境

```
正确示例：
第一章 重生

张伟重生了。

上一秒，他还躺在ICU里，呼吸机滴滴作响。

下一秒，他就回到了十年前的高中教室。

讲台上，班主任正在骂人。

"张伟！你能不能认真点？"

张伟愣了一下。

这是……2008年？

他低头看了看自己。

校服。课桌。课本。

真的是高中。

他重生回到了十八岁。
```

### 黄金三章
- **第一章**：重生/穿越 + 身份处境 + 第一个爽点期待
- **第二章**：展现金手指/能力 + 第一次小爽点
- **第三章**：第一次打脸/反转 + 建立主线目标

---

## 情绪推动

### 以情绪驱动剧情
**核心方法**：
```
主角处境 → 主角情绪 → 主角行动 → 结果反馈
```

```
示例：
张伟看着银行卡余额。

三千块。

就这点钱，连房租都不够交。

他有点烦。

上辈子混了一辈子，到死都没攒下钱。

这辈子，他不想再这样了。

他得想办法赚钱。

而且要快。
```

## 避坑指南

### 1. 禁止模板化词汇
- ❌ 充满了、坚定、不甘、难以置信、震惊、愤怒、绝望、悲伤
- ❌ 冷冷地说、愤怒地说、嘲讽地说
- ✅ 用动作/场景代替解释性叙述

### 2. 禁止过度解释
```
❌ 错误：林风眼中充满了愤怒，紧紧握住拳头，不甘地看着赵峰。

✅ 正确：林风盯着赵峰，拳头慢慢收紧，指节发白。他呼吸变得急促，但没有说话。
```

### 3. 禁止口号式台词
```
❌ 错误："我一定要变强！我要让你们付出代价！"

✅ 正确：他没说话，转身离开了。房间里只剩下沉重的呼吸声。
```

### 4. 禁止主角过度"正确"
- 偶尔的自私、小聪明、坏心思是真实人性的体现
- 完美主角缺乏代入感

## 示例对照

### 情绪表达对比
```
❌ 错误：
苏晴很悲伤，眼泪从脸上滑落。

✅ 正确：
苏晴站在窗边，看着楼下的车流。
她的眼睛红了，眼角有一道水痕。
她没有擦，任由它滑落到下巴。
```

### 愤怒表达对比
```
❌ 错误：
林风愤怒地说："我要杀了你！"他狠狠地握紧拳头，眼神凶狠。

✅ 正确：
林风看着赵峰，没有说话。
他慢慢走到赵峰面前，蹲下。
"赵峰。"
"你...你不敢..."
"死吧。"
```

### 震惊表达对比
```
❌ 错误：
赵峰难以置信地看着林风，震惊得说不出话来。

✅ 正确：
赵峰瞪着林风，张了张嘴，没发出声音。
他后退一步，撞到了身后的桌子。
```
