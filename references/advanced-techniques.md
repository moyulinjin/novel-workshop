# 高级写作技巧

## 一、避免模板化词汇

### 常见模板词汇

**情绪类模板词**：
- 充满了、坚定、不甘、难以置信、震惊、愤怒、绝望、悲伤

**状态类模板词**：
- 冷冷地说、愤怒地说、嘲讽地说、轻蔑地说、温柔地说、平静地说

**动作类模板词**：
- 紧紧握住、狠狠地、猛地、傻傻地、默默地

### 替代方案

**替代原则**：
1. 用具体动作代替抽象词汇
2. 用场景细节代替情绪描述
3. 用对话代替旁白
4. 用对比代替直接描述

---

## 二、动作/场景代替解释性叙述

### Show, Don't Tell

**核心要求**：
- 用动作展示情绪，而不是直接告诉读者
- 用场景营造氛围，而不是解释说明
- 用细节暗示状态，而不是直接描述

### 方法1：用动作展示情绪

```
❌ 错误：林风很愤怒。

✅ 正确：
林风一脚踹翻了面前的椅子。
椅子撞在墙上，发出巨响。
他喘着粗气，盯着赵峰。
```

### 方法2：用场景营造氛围

```
❌ 错误：这是一个悲伤的夜晚。

✅ 正确：
窗外下着雨，雨水拍打在窗户上。
房间里很安静，只有抽泣声。
灯关着，黑暗中只有手机屏幕的微光。
```

### 方法3：用细节暗示状态

```
❌ 错误：苏晴很紧张。

✅ 正确：
苏晴坐在沙发上，手指不停地搓着衣角。
她看了一眼钟，又看了一眼门。
来回看了三次。
```

### 方法4：用对比代替描述

```
❌ 错误：赵峰很傲慢。

✅ 正确：
赵峰靠在沙发上，翘着二郎腿。
"又来了一个？"
他连眼皮都没抬一下。
"排队去吧，前面还有十个。"
```

## 三、口语化对话技巧

### 基本要求
- 对话要口语化，带停顿
- 每句对话独立成段
- 用动作/反应代替"说"

### 对话范例

```
❌ 错误：
"我不不会放过你的。"林风愤怒地说。

✅ 正确：
林风看着赵峰，没有说话。
"我不会，放过你的。"

❌ 错误：
"这是我的地盘，识相的话就滚。"他傲慢地说。

✅ 正确：
他靠在墙上，点了一根烟。
"这是我的地盘。"
吐出一口烟圈。
"识相的话，滚。"
```

### 口语化技巧
- 短句、碎片化
- 有停顿、有省略
- 符合角色性格
- 带语气词（可选）

## 四、细节表现情绪

### 情绪外化法

```
悲伤 → 用动作/生理反应表现
- 眼泪滑落
- 肩膀颤抖
- 沉默不语
- 声音沙哑

愤怒 → 用破坏性动作表现
- 砸东西
- 握拳
- 呼吸急促
- 声音变大

紧张 → 用重复性动作表现
- 搓手
- 踱步
- 喝水
- 看向别处

震惊 → 用失态动作表现
- 后退
- 撞到东西
- 张嘴
- 东西掉落
```

### 正确示范

```
❌ 错误：
苏晴很悲伤，眼泪流了下来。

✅ 正确：
苏晴坐在床边，看着手里的照片。
她的肩膀微微颤抖，但没有发出声音。
一滴眼泪掉在照片上，晕开了她的笑容。

❌ 错误：
王强很震惊，难以置信地看着这一切。

✅ 正确：
王强瞪大眼睛，看着眼前的场景。
他后退一步，脚下一滑，摔倒在地。
"这...这怎么可能？"
```

## 五、平静的狠劲

### 高级写法

主角的狠劲"藏在平静里"，比直接爆发更有冲击力。

### 正确示范

```
❌ 错误：
林风愤怒地说："我要杀了你！"
他狠狠地握紧拳头，眼神凶狠。
他一掌拍出，将赵峰震飞出去。

✅ 正确：
林风看着赵峰，没有说话。

他慢慢走过去。

蹲下。

"赵峰。"

"你...你不敢..."

"死吧。"

剑光一闪。

人头落地。

林风收剑，转身。

袍子上没有一滴血。
```

### 关键要点

1. **用沉默代替嘶吼**
2. **用慢动作代替快爆发**
3. **用反差对比增强冲击**
4. **让结果说话，而不是情绪**

## 六、写作质量检查

### 自检清单

- [ ] 是否避免了模板化词汇？
- [ ] 是否用动作/场景代替解释？
- [ ] 对话是否口语化？
- [ ] 情绪是否通过细节表现？
- [ ] 主角狠劲是否藏在平静里？
- [ ] 段落是否短句碎段？
- [ ] 是否避免了文青式表达？

### 常见错误

| 错误 | 问题 | 修正 |
|-----|------|------|
| 充满了愤怒 | 抽象描述 | 用动作展示 |
| 冷冷地说 | 标签化 | 用场景表现 |
| 他很紧张 | 直接描述 | 用细节暗示 |
| 我要杀你了 | 口号式 | 用行动说话 |
| 全是长句 | 阅读疲劳 | 短句碎段 |
