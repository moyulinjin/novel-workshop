## 第一部分：写作工具与环境

### 01. 用 Markdown 编辑器来写作

**推荐工具：Typora**
- 下载地址：https://typora.io
- 支持实时预览，所见即所得

**Markdown 基础语法**

| 标记 | 用法 | 示例 |
|------|------|------|
| 标题 | `#` + 空格 | `# 一级标题`、`## 二级标题` |
| 引用 | `>` + 空格 | `> 这是一段引用文字` |
| 强调 | `\*` 包裹 | `**粗体**`、`*斜体*` |
| 列表 | `-` 或 `1.` | `- 列表项`、`1. 有序列表` |
| 代码 | `` ` `` 包裹 | `` `代码片段` `` |

**进阶语法**
- 链接：`[文字](URL)`
- 图片：`![alt](图片URL)`
- 分割线：`---`
- 表格：使用 `|` 和 `-` 构建

**印象笔记 Markdown 语法参考**
- 官网文档：https://help.yinxiang.com/hc/articles/69276

## 第二部分：信息搜索与获取

### 02. 搜索：高级搜索技能

**核心价值**
- 快速定位高质量素材
- 提升信息获取效率
- 构建知识搜索体系

**印象笔记高级搜索语法**

| 语法 | 功能 | 示例 |
|------|------|------|
| `intitle:关键词` | 搜索标题 | `intitle:写作` |
| `tag:标签名` | 按标签搜索 | `tag:读书笔记` |
| `notebook:笔记本` | 指定笔记本 | `notebook:写作素材` |
| `created:日期` | 按创建日期 | `created:2024-01` |
| `-` | 排除关键词 | `写作 -小说` |

**搜索语法参考文档**
- 官网：https://help.yinxiang.com/hc/articles/62969

**实用技巧**
1. 组合使用多个语法：`intitle:写作 tag:素材`
2. 手机端同样支持高级搜索语法
3. 建立搜索习惯，定期用关键词检索素材库

## 第三部分：笔记系统方法论

### 03. 写出赚钱的读书笔记

**读书笔记写作框架**

```
【见】- 看见：摘录原文金句、核心观点
【感】- 感受：写下自己的感受、触动点
【思】- 思考：联想相关知识、提出问题
【行】- 行动：制定实践计划、行动步骤
```

**框架详解**

| 环节 | 核心问题 | 示例 |
|------|----------|------|
| 见 | 看到了什么？ | 原文："写作是思考的提炼" |
| 感 | 有什么触动？ | 感悟：之前总觉得自己想清楚了才能写 |
| 思 | 联想到什么？ | 思考：这说明写作本身就是思考过程 |
| 行 | 如何行动？ | 行动：边写边思考，不要等想清楚再写 |

**赚钱读书笔记的关键**
- 选择实用型书籍：方法论、干货类
- 提炼可操作的方法步骤
- 加入个人实践案例
- 提供行动清单

**参考范文**
- 《写作的风格意识》读书笔记

### 04. 知识银行：「见感思行」完整版创作

**核心要点：串联知识的能力**

「见感思行」完整版的关键在于**「思」**——将不同知识点串联起来，形成知识网络。

**深化版框架**

```
【见】- 看到了什么？
      - 原文摘录/核心观点
      - 数据、案例、金句

【感】- 有什么感受？
      - 触动了哪个点？
      - 与自己经验的共鸣

【思】- 深度思考（核心）
      - 这个知识与哪些知识相关？
      - 为什么这个知识是对的？
      - 它背后的底层逻辑是什么？
      - 能举出哪些反例？

【行】- 如何行动？
      - 具体的实践步骤
      - 可量化的目标
      - 定期复盘计划
```

**串联知识的技巧**
1. 建立知识点之间的连接词
2. 追问"为什么"至少三次
3. 用类比解释复杂概念
4. 画知识图谱，理清关系

### 05. 独具一格的标签组块分类法

**标签系统设计原则**
- 标签数量适中（建议 20-50 个）
- 层级清晰，避免重复
- 按使用频率调整

**推荐分类维度**

```
📁 按内容类型
   ├── 素材（故事、案例、金句、数据）
   ├── 主题（职场、情感、成长、赚钱）
   └── 形式（清单、故事、方法论、观点）

📁 按来源
   ├── 读书笔记
   ├── 课程笔记
   ├── 电影/纪录片
   └── 日常观察

📁 按状态
   ├── 待整理
   ├── 已加工
   └── 待使用
```

**标题组块命名法**
- `人物+名字+关键词`：如"人物-李子柒-乡村生活"
- `故事+关键词`：如"故事-创业-坚持"
- `时间+关键词`：如"2024-03-读书清单"

### 06. 卡片大法：一流作家的写作秘籍

**为什么卡片写作是顶级作家的秘密？**

| 好处 | 说明 |
|------|------|
| 降低大脑负荷 | 大脑只需思考，不用记忆 |
| 提高创意密度 | 一个卡片一个想法，聚焦深入 |
| 形成知识块 | 积累可复用的素材单元 |
| 产生复利效应 | 卡片越多，写作越轻松 |

**卡片写作核心原则**
> **写清楚一件事，而不是什么都写**

**卡片标准格式**

```
标题：[卡片主题，一句话概括]

来源：[书名/课程/电影/生活]

核心内容：
- 关键观点1
- 关键观点2

个人感悟：
[用自己的话复述]

应用场景：
[什么时候可以用这张卡片？]
```

**使用建议**
- 每张卡片控制在 300-500 字
- 定期翻阅、迭代更新
- 建立卡片之间的链接

## 第四部分：范例研究与模仿

### 07. 范例银行：研习一流作品，形成范例

**为什么范例如此重要？**
- 模仿是学习的起点
- 范例提供可参考的坐标系
- 高手都在研究标杆作品

**研习一流作品的方法**

```
1️⃣ 选择榜样
   - 找到你欣赏的作家/作品
   - 明确你想学习的方面

2️⃣ 拆解分析
   - 结构：开头、中间、结尾如何组织？
   - 语言：用了哪些表达技巧？
   - 节奏：段落如何衔接？
   - 风格：有什么独特印记？

3️⃣ 模仿实践
   - 选取一段进行仿写
   - 对比原文找差距
   - 逐步形成自己的风格

4️⃣ 积累范例
   - 建立范例素材库
   - 标注值得借鉴的点
   - 定期回顾更新
```

**推荐研习对象**
- 阿城《遍地风流》- 短句与留白
- 汪曾祺散文 - 语言质感
- 李盆文章 - 观点锐利

**练习方法**
- 每周精读一篇文章
- 写一篇 200 字的仿写
- 对比总结改进点

## 第五部分：写作风格与表达

### 08. 如何进行口语化写作

**为什么要口语化？**

罗振宇观点：
> "为什么现在有这些学渣？因为书面语言。好多学生有阅读困难，不懂书面语言。"

**口语化写作的优势**
- 降低阅读门槛
- 增加亲和力与信任感
- 符合碎片化阅读习惯
- 更容易被转发传播

**口语化写作技巧**

| 技巧 | 示例 |
|------|------|
| 用"我"代替"我们" | ❌ 我们要努力 → ✅ 你要努力 |
| 用短句代替长句 | ❌ 因此，考虑到多方面因素 → ✅ 所以 |
| 用具体代替抽象 | ❌ 提升认知水平 → ✅ 让脑子更灵光 |
| 加入语气词 | 可以用"啊""嘛""呢" |
| 像在说话 | 想象对面有个人，写给他看 |

**检验口语化的方法**
- 朗读出来是否顺畅
- 朋友能否听懂
- 是否像"人话"

### 09. 如何优化一条标题，实现搜索无死角

**标题组块的三个原则**

```
原则一：可检索
   - 包含目标读者会搜索的关键词
   - 避免生僻字、谐音梗

原则二：可理解
   - 标题结构清晰
   - 让读者一眼看懂价值

原则三：可传播
   - 有吸引力，引发好奇
   - 符合平台推荐算法
```

**标题组块公式**

| 类型 | 公式 | 示例 |
|------|------|------|
| 人物型 | 人物+名字+关键词 | "李子柒：乡村生活的另一种可能" |
| 故事型 | 故事+关键词 | "我从月薪3000到副业过万的经历" |
| 时间型 | 时间+关键词 | "2024年最值得读的10本书" |
| 数字型 | 数字+主题词 | "3个方法让你早起不痛苦" |

**关键词选择策略**
1. 目标读者会搜什么词？
2. 行业内的核心关键词？
3. 长尾词：更精准，竞争小

**实操检查清单**
- [ ] 标题是否包含核心关键词？
- [ ] 是否能通过标题判断内容？
- [ ] 长度是否合适（10-30字）？
- [ ] 是否有吸引力？
