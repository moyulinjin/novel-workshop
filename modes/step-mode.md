# 渐进询问创作模式

## 一、模式说明

### 适用场景
- 有具体的创作想法
- 想要精细控制每个设定
- 已经有明确的故事构思
- 不喜欢AI自动生成

### 核心优势
- 灵活性高，可精细控制
- 保留创作自主权
- 适合有明确规划的用户
- 每一步都可以调整

## 三、分阶段提问

### 第一阶段：核心基础设定

#### 问题1：主人公设定
```
请提供以下信息：
- 主角姓名：（或由AI生成）
- 初始身份：（废柴少爷/底层打工人/被背叛门徒）
- 性格特质：（隐忍腹黑/桀骜不驯/扮猪吃虎）
- 核心目标：（复仇/登顶/守护）

如果不完整，可选择「由AI自动生成剩余设定」
```

#### 问题2：金手指设定
```
请提供以下信息：
- 核心依仗：（祖传玉佩/系统/上古功法/特殊血脉）
- 限制/代价：（需积累声望/解锁条件/副作用）
- 独特能力：（看破弱点/加速修炼/推演功法）

如果不完整，可选择「由AI自动生成剩余设定」
```

#### 问题3：世界观设定
```
请提供以下信息：
- 故事背景：（修仙界/都市商圈/武道世家/星际联邦）
- 核心规则：（修炼等级/权力体系/商业法则）
- 主要势力：（宗门/家族/财团/联盟）

如果不完整，可选择「由AI自动生成剩余设定」
```

### 第二阶段：冲突与反派设定

#### 问题4：核心反派设定
```
请提供以下信息：
- 主要反派姓名：（或由AI生成）
- 反派身份：（天才师兄/敌对家族族长/职场竞品高管）
- 恶行/动机：（抢夺宝物/陷害主角/打压发展）
- 实力/优势：（需略高于初期主角）

如果不完整，可选择「由AI自动生成剩余设定」
```

#### 问题5：次要角色设定
```
请提供以下信息：
- 助攻配角：（忠仆/导师/红颜知己）
- 轻视角色：（家族长老/同学/同事）
- 中立角色：（神秘大佬/规则制定者）

如果不完整，可选择「由AI自动生成剩余设定」
```

#### 问题6：核心冲突设定
```
请提供以下信息：
- 主角与反派的核心矛盾：（身世仇恨/资源争夺/地位竞争）
- 冲突阶段升级：（初期羞辱→中期对抗→后期决战）
- 支线冲突：（家族内斗/宗门考核/商业博弈）

如果不完整，可选择「由AI自动生成剩余设定」
```

### 第三阶段：爽点与节奏设定

#### 问题7：核心爽点类型
```
请提供以下信息：
- 重点爽点类型：（可多选）
  □ 扮猪吃虎
  □ 越级碾压
  □ 身份揭秘
  □ 反派打脸
  □ 资源垄断
  □ 荣耀加身
  □ 复仇闭环
  □ 其他_____
- 优先级排序：
```

#### 问题8：爽点爆发节点
```
请提供以下信息：
- 初期第一个爽点场景：（宗门小比/家族会议/职场汇报）
- 中期关键爽点事件：（解锁金手指/夺取反派资源）
- 后期终极爽点：（终极决战/登顶时刻）

如果不完整，可选择「由AI自动生成剩余设定」
```

#### 问题9：节奏把控偏好
```
请提供以下信息：
- 节奏偏好：
  □ 密集爽点（短篇幅高频打脸）
  □ 层次爽点（铺垫后爆发大高潮）
  □ 适中节奏
- 情感线需求：
  □ 需要（羁绊守护/双向奔赴）
  □ 不需要
  □ 可以有但不是重点
```

### 第四阶段：结局与细节设定

#### 问题10：结局走向设定
```
请提供以下信息：
- 结局类型：
  □ 圆满闭环（主角登顶、反派覆灭）
  □ 留白延伸（达成核心目标，预留后续）
- 主题升华：（守护正义/打破规则/成长蜕变）
```

#### 问题11：高光场景设定
```
请提供以下信息：
- 最想刻画的高光场景：（决战时刻/名场面/跪地求饶）
- 特定元素融入：
  □ 专属台词
  □ 标志性动作
  □ 场景氛围
```

#### 问题12：特殊设定补充
```
请提供以下信息：
- 专属道具：（本命武器/传承信物）
- 口头禅/暗号：
- 禁忌规则：（可被主角打破）
- 其他补充：
```

### 第五阶段：章节规划

#### 问题13：章节规划确认
```
请确认：
- 总章节数量：（50章/100章/200章）
- 每章字数：（建议1500-2000字）
- 每次生成章节数：（建议1-3章）
- 优先生成章节：（开篇/高潮/结局）
```

## 六、模式切换

### 随时切换到自动模式
```
如果觉得逐个回答太麻烦，可以：
1. 输入「切换自动模式」
2. AI自动生成所有剩余设定
3. 确认或微调后继续
```

### 组合使用
```
也可以组合使用：
1. 回答关键问题（如：题材、核心冲突）
2. 让AI补充其他设定
3. 根据需要调整
```
