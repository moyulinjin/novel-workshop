# AI全自动创作模式

## 一、模式说明

### 适用场景
- 没有具体创作想法
- 想要快速开始
- 第一次使用，希望快速上手
- 只有一个大致的创作方向

### 核心优势
- 快速上手，降低门槛
- AI自动填充所有细节
- 适合探索式创作
- 30分钟即可获得完整设定

---

## 二、工作流程

### 步骤1：用户输入需求
```
一句话需求，例如：
- "写一个修仙爽文"
- "都市职场逆袭"
- "末世生存爽文"
- "甜宠古代言情"
```

### 步骤2：AI自动生成设定
```
AI基于用户需求，自动生成完整的五阶段设定：

第一阶段：核心基础设定
├── 主人公设定
├── 金手指/核心能力
└── 世界观与规则

第二阶段：冲突与反派设定
├── 核心反派设定
├── 次要角色设定
└── 核心冲突设定

第三阶段：爽点与节奏设定
├── 核心爽点类型
├── 爽点爆发节点
└── 节奏把控偏好

第四阶段：结局与细节
├── 结局走向
├── 高光场景
└── 特殊设定

第五阶段：章节规划
├── 章节数量
├── 每章字数
└── 关键节点
```

### 步骤3：用户确认与微调
```
确认选项：
□ 确认设定，直接使用
□ 微调部分设定
□ 切换到渐进询问模式详细设定
□ 开始内容生成
```

### 步骤4：进入内容生成
```
确认后直接进入第五阶段：
- 章节规划确认
- 详细内容生成
- 每章1500-2000字
```

## 四、自动生成示例

### 用户需求
```
都市重生逆袭爽文
```

### AI自动生成
```markdown
=== 小说设定方案 ===

【基本信息】
书名候选：《重生2008，从摆摊到首富》
题材：都市重生
风格：逆袭爽文
预期字数：100万字

【第一阶段：核心设定】

主人公：
- 姓名：林风
- 性别：男
- 初始身份：被净身出户的废物女婿
- 性格特点：表面隐忍，内心狠辣
- 核心目标：登顶首富，报仇雪恨

金手指：
- 类型：重生预知+商业头脑
- 名称：先知优势
- 核心能力：记得未来十年的商业机遇
- 限制/代价：每次改变历史都有未知代价

世界观：
- 故事背景：2008年金融危机后的都市商战
- 核心规则：资本为王，人脉为皇
- 主要势力：四大家族、新兴资本、灰色势力

【第二阶段：冲突设定】

核心反派：
- 姓名：赵峰
- 身份：赵家继承人
- 恶行：三年前陷害主角，夺其家产
- 与主角冲突：商业竞争+私人恩怨

配角：
- 助攻角色：神秘老头（隐藏大佬）
- 见证角色：前妻（见证主角崛起）

核心冲突：
- 冲突类型：商业复仇+身份逆袭
- 升级路径：商业打压→家族对抗→终极对决
- 关键节点：首次反击、中点翻盘、最终决战

【第三阶段：爽点设计】

核心爽点：
- 主要爽点类型：扮猪吃虎、身份揭露、商业打脸
- 次要爽点类型：红颜倾心、收服小弟、资源垄断

爽点分布：
- 初期爽点：摆摊小赚、打脸前妻亲戚
- 中期爽点：商业帝国崛起、身份揭露
- 后期爽点：赵家覆灭、登顶首富

节奏偏好：
- 快节奏
- 爽点密度：高

【第四阶段：结局设定】

结局走向：圆满闭环

高光场景：
- 场景1：赵峰跪地求饶
- 场景2：前妻后悔求复合

特殊设定：
- 专属道具：祖传玉佩（隐藏身份）
- 经典台词："三十年河东，三十年河西"
- 独特元素：商业博弈

【第五阶段：章节规划】

总章节：300章
每章字数：2000字
生成顺序：1-10章→11-30章→...
```

## 五、模式切换

### 切换到渐进询问模式
```
如果自动生成的设定不够满意，可以：

1. 提出需要修改的部分
2. AI根据反馈进行调整
3. 或切换到渐进询问模式
4. 逐一详细设定
```

### 混合模式
```
也可以使用混合模式：

1. AI生成基础设定
2. 用户补充/修改关键设定
3. AI基于修改后的设定继续生成
4. 循环直到满意
```

## 六、常见问题

### Q：自动生成的设定太套路化？
A：可以补充具体的创意需求，AI会据此调整

### Q：设定中有不喜欢的部分？
A：直接指出，AI会重新生成或调整

### Q：想要更多细节？
A：可以指定详细程度，或切换到渐进模式

### Q：不确定自己的需求？
A：可以输入模糊需求，AI会通过追问澄清
