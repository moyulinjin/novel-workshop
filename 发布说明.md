# 小说工坊 - 开源发布包

## 📦 版本信息

- **名称**：小说工坊 (Novel Workshop)
- **版本**：v1.0.0
- **发布日期**：2026-04-18
- **协议**：MIT License

---

## 🔥 核心亮点

1. **卡文急救包** - 写到这里不知道怎么往下写了？3秒出方向，5秒出方案
2. **分层入口** - 说"卡文了"直接跳转，不用选模块
3. **新手引导** - 30分钟搞定第一章，从零开始
4. **情感渲染** - 五感锚点、留白艺术、身体记忆库
5. **全流程覆盖** - 灵感→大纲→人设→正文→优化

## 📊 功能项清单

| 序号 | 功能模块 | 功能描述 |
|:---:|---------|---------|
| 1 | 灵感创意 | 书名生成、简介生成、创意归类、热门题材分析 |
| 2 | 设定构建 | 人设卡构建、金手指设计、世界观设定、冲突设计 |
| 3 | 大纲规划 | 市场型大纲、人物型大纲、分卷大纲 |
| 4 | 正文创作 | 开篇生成、章节写作、对话场景、打斗描写 |
| 5 | 情节优化 | 冲突设计、反转生成、节奏诊断 |
| 6 | 情感渲染 | 情感坐标系、五感锚点、身体记忆、留白艺术 |
| 7 | 知识库 | 书籍榨取、知识点提取、费曼检验 |
| 8 | 辅助工具 | 封面生成、提示词管理、写作风格指南 |
| 9 | 帮助中心 | 新手指南、平台攻略、常见问题 |
| 10 | 卡文急救包 | 六种卡文类型急救、极速模式、卡文预防 |
| 11 | 新手引导 | 30分钟启动计划、开篇五步法 |

## 🎯 快速开始

输入以下关键词，直接跳转对应模块：

| 关键词 | 跳转 |
|--------|------|
| 卡文/卡住了 | → 卡文急救包 |
| 书名/标题 | → 爆款书名生成 |
| 大纲/章纲 | → 大纲规划 |
| 人设/主角 | → 设定构建 |
| 开篇/第一章 | → 新手引导 |
| 打斗/对话 | → 正文创作 |
| 爽点/高潮 | → 爽点设计 |
